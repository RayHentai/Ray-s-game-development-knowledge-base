## 核心理解

链表是一种**通过指针串联在一起的线性结构**，每一个节点由两部分组成，
- 一个是数据域
- 一个是指针域（存放指向下一个节点的指针）

最后一个节点的指针域指向null（空指针的意思）。
链表的入口节点称为链表的头结点也就是head。
![[链表 图示.png|364]]

```csharp title:定义
public class Node<T>
{
    // 节点存储的数据
    public T Data { get; set; }
    // 指向下一个节点的引用
    public Node<T> Next { get; set; }
    // 节点的构造函数，用于初始化节点
    public Node(T data)
    {
        Data = data;
        Next = null; // 初始时没有下一个节点，因此设为 null
    }
}
```

## 链表的类型

**1. 单链表**
单链表中的指针域只能指向节点的下一个节点
![[链表 图示.png|331]]

**2. 双链表**
双链表每一个节点有两个指针域，一个指向下一个节点，一个指向上一个节点。
既可以向前查询也可以向后查询
![[双链表 图示.png|422]]

**3. 循环链表**
链表首尾相连

## 关键特性

**存储方式：在内存中不是连续分布的**

链表是通过指针域的指针链接在内存中各个节点。
所以链表中的节点在内存中不是连续分布的 ，而是**散乱分布在内存中的某地址上**，分配机制取决于操作系统的内存管理。
