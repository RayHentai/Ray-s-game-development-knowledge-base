## 解法二：双指针（最优）

**核心思路：**

**时间复杂度：** O(n)
**空间复杂度：** O(1)

```csharp
public class Solution {
    public ListNode ReverseList(ListNode head) {
        ListNode pre = null;
        ListNode cur = head;
        while (cur != null){
            ListNode temp = cur.next;
            cur.next = pre;
            pre = cur;
            cur = temp;
        }
        return pre;
    }
}
```
