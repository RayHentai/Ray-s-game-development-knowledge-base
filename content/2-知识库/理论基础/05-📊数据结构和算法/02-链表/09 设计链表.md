## 题目描述

你可以选择使用单链表或者双链表，设计并实现自己的链表。

单链表中的节点应该具备两个属性：`val` 和 `next` 。`val` 是当前节点的值，`next` 是指向下一个节点的指针/引用。

如果是双向链表，则还需要属性 `prev` 以指示链表中的上一个节点。假设链表中的所有节点下标从 **0** 开始。

实现 `MyLinkedList` 类：

- `MyLinkedList()` 初始化 `MyLinkedList` 对象。
- `int get(int index)` 获取链表中下标为 `index` 的节点的值。如果下标无效，则返回 `-1` 。
- `void addAtHead(int val)` 将一个值为 `val` 的节点插入到链表中第一个元素之前。在插入完成后，新节点会成为链表的第一个节点。
- `void addAtTail(int val)` 将一个值为 `val` 的节点追加到链表中作为链表的最后一个元素。
- `void addAtIndex(int index, int val)` 将一个值为 `val` 的节点插入到链表中下标为 `index` 的节点之前。如果 `index` 等于链表的长度，那么该节点会被追加到链表的末尾。如果 `index` 比长度更大，该节点将 **不会插入** 到链表中。
- `void deleteAtIndex(int index)` 如果下标有效，则删除链表中下标为 `index` 的节点。

**示例：**

```c
//输入
["MyLinkedList", "addAtHead", "addAtTail", "addAtIndex", "get", "deleteAtIndex", "get"]
[[], [1], [3], [1, 2], [1], [1], [1]]
//输出
[null, null, null, null, 2, null, 3]

//解释
MyLinkedList myLinkedList = new MyLinkedList();
myLinkedList.addAtHead(1);
myLinkedList.addAtTail(3);
myLinkedList.addAtIndex(1, 2);    // 链表变为 1->2->3
myLinkedList.get(1);              // 返回 2
myLinkedList.deleteAtIndex(1);    // 现在，链表变为 1->3
myLinkedList.get(1);              // 返回 3
```

## 第一反应

用List记录节点，然后在链表中实现要求的方法
这种方法有点投机取巧，因为完全是依靠数组实现的
设计链表主要考察的是对链表`next`指针的管理能力，不能依赖任何其他数据结构

## 解法一：用List\<T>实现

**思路：**

**时间复杂度：** O()
**空间复杂度：** O()

```csharp
public class MyLinkedList {
    List<MyLinked> linkedList;
    public MyLinkedList() {
        linkedList = new List<MyLinked>();
    }
    
    public int Get(int index) {
        if (index > linkedList.Count - 1 || index < 0)
            return -1;
        return linkedList[index].val;
    }
    
    public void AddAtHead(int val) {
        MyLinked next = linkedList.Count == 0 ? null : linkedList[0];
        MyLinked head = new MyLinked(val, next);
        linkedList.Insert(0, head);
    }
    
    public void AddAtTail(int val) {
        MyLinked last = new MyLinked(val);
        if (linkedList.Count > 0)
            linkedList[linkedList.Count - 1].next = last;
        linkedList.Add(last);
    }
    
    public void AddAtIndex(int index, int val) {
        if (index > linkedList.Count || index < 0)
            return;
        if (index == linkedList.Count)
            AddAtTail(val);
        else if (index == 0) 
            AddAtHead(val);
        else{
            MyLinked node = new MyLinked(val, linkedList[index]);
            linkedList.Insert(index, node);
        }
    }
    
    public void DeleteAtIndex(int index) {
        if (index <= linkedList.Count - 1 && index >= 0){
            linkedList.RemoveAt(index);
        }
    }
}
public class MyLinked {
    public int val;
    public MyLinked next;
    public MyLinked(int val = 0, MyLinked next = null){
        this.val = val;
        this.next = next;
    }
}

/**
 * Your MyLinkedList object will be instantiated and called as such:
 * MyLinkedList obj = new MyLinkedList();
 * int param_1 = obj.Get(index);
 * obj.AddAtHead(val);
 * obj.AddAtTail(val);
 * obj.AddAtIndex(index,val);
 * obj.DeleteAtIndex(index);
 */
```

## 解法二：[解法名称]（最优）

**核心思路：**

**时间复杂度：** O()
**空间复杂度：** O()

```csharp
public class MyLinkedList {
    private MyLinked dummy;
    private int size;
    public MyLinkedList(){
        dummy = new MyLinked();
        size = 0;
    }
    
    public int Get(int index) {
        if (index > size - 1 || index < 0)
            return -1;
        MyLinked cur = dummy;
        for (int i = 0; i < index + 1; i++){
            cur = cur.next;
        }
        return cur.val;
    }
    
    public void AddAtHead(int val) {
        MyLinked head = new MyLinked(val);
        head.next = dummy.next;
        dummy.next = head;
        ++size;
    }
    
    public void AddAtTail(int val) {
        MyLinked tail = new MyLinked(val);
        MyLinked cur = dummy;
        for (int i = 0; i < size; i++)
            cur = cur.next;
        cur.next = tail;
        ++size;
    }
    
    public void AddAtIndex(int index, int val) {
        if (index > size || index < 0)
            return;
        if (index == 0)
            AddAtHead(val);
        else if(index == size)
            AddAtTail(val);
        else{
            MyLinked tail = new MyLinked(val);
            MyLinked cur = dummy;
            for (int i = 0; i < index; i++)
                cur = cur.next;
            MyLinked nowNext = cur.next;
            cur.next = tail;
            tail.next = nowNext;
            ++size;
        }
    }
    
    public void DeleteAtIndex(int index) {
        if (index > size - 1 || index < 0)
            return;
        MyLinked cur = dummy;
        for (int i = 0; i < index; i++)
            cur = cur.next;
        cur.next = cur.next.next;
        --size;
    }
}
public class MyLinked{
    public int val;
    public MyLinked next;
    public MyLinked(int val = 0, MyLinked next = null){
        this.val = val;
        this.next = next;
    }
}
/**
 * Your MyLinkedList object will be instantiated and called as such:
 * MyLinkedList obj = new MyLinkedList();
 * int param_1 = obj.Get(index);
 * obj.AddAtHead(val);
 * obj.AddAtTail(val);
 * obj.AddAtIndex(index,val);
 * obj.DeleteAtIndex(index);
 */
```

**注意：**
- `AddAtHead` 要先把原来的头接到新节点上，再让dummy指向新节点，顺序不能反
- 所有需要走到前驱节点的操作，从dummy开始走index步，不是index-1步
- `AddAtIndex` 的边界是 `index > size`，不是 `size - 1`，因为等于size时是插到末尾
