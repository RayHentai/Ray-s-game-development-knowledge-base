## 核心理解

**数组是存放在连续内存空间上的相同类型数据的集合。**
数组可以方便的通过下标索引的方式获取到下标对应的数据。

![[数组 图示.png]]

---

## 关键特性

- **数组下标都是从0开始的**
- **数组内存空间的地址是连续的（包括二维数组）**
- **数组的元素是不能删的，只能覆盖。**

---

## 注意事项

- 正是**因为数组在内存空间的地址是连续的，所以在删除或者增添元素的时候，就难免要移动其他元素的地址。**

### 二分法

使用前提条件数：**数组中无重复元素** 

核心思路：**循环不变量规则：在循环中坚持根据查找区间的定义来做边界处理**

![[二分法 图示.png|312]]

**不变量也就是区间**，在二分法中有两种定义区间的方式，**对区间的定义也影响了循环中该如何判断边界**。
左闭右闭 `[left, right]`  `middle = (left + right) / 2`
左闭右开 `[left, right)`  `middle = (left + right) / 2`

**循环则是不断重新定义区间，直到区间不合法时退出** 

如果 `target`>`nums[middle]`则说明 `target` 在区间的右端，反之则说明在左端
如果等于直接 `return middle

**1. 以左闭右闭为例** `while (left <= right)`
接着重新定义区间即可
\> 时 `left =  middle + 1`，当前区间 `[middle + 1, right]`
< 时  `right =  middle - 1`，当前区间 `[left, middle - 1]`

**2. 以左闭右开为例** `while ( left < right)`
\> 时 `left =  middle + 1`，当前区间 `[middle + 1, right]`
< 时  `right =  middle`，当前区间 `[left, middle]`

直到 `不满足while循环的条件` 时，如果还没找到target，则在循环外部`return -1`

**可能存在的问题**
1. `middle = (left + right) / 2` 这样计算middle会存在**溢出问题**，假设 `left + right > int的最大值`，那么就会数据溢出，可以换一种方式：`middle = left + (right - left) / 2`，这样算等同前面的算法，并且也规避了一出问题

```
//左闭右闭
left = 0, right = nums.Length - 1
while left <= right:
    middle = left + (right - left) / 2
    if nums[middle] == target:
        return middle
    else if nums[middle] < target:
        left = middle + 1
    else:
        right = middle - 1
return -1

//左闭右开：
left = 0, right = nums.Length
while left < right:
    middle = left + (right - left) / 2
    if nums[middle] == target:
        return middle
    else if nums[middle] < target:
        left = middle + 1
    else:
        right = middle
return -1
```

### 双指针

**双指针法：用两个指针在一次遍历中完成原本需要两层循环才能做的事**

**几种常见形式：**
**快慢指针：两个指针同向移动，速度不同**
  典型应用：移除元素（快指针找有效元素，慢指针维护新数组边界）
![[快慢指针 图示.gif|266]]

**对撞指针（左右指针）：两个指针从两端向中间移动**
  典型应用：有序数组平方（比较两端平方值，从末尾往前填）

![[对撞指针 图示.gif|294]]

**滑动窗口：两个指针同向移动，维护一个区间**
  典型应用：长度最小的子数组（右边界扩大窗口累加sum，sum >= target时缩小左边界，找最短区间）
![[滑动窗口 图示.gif|273]]

---

#### 快慢指针

分别声明快慢指针 `slowIndex = 0`, 快指针直接声明在for循环中，因为每次开始循环都要++

进入for循环 `for (int fastIndex = 0; fastIndex < num.Length; fastIndex++)`

如果 `nums[fastIndex] != var` 快指针找到有效元素 慢指针维护边界 
-> 慢指针指向的元素等于快指针指向的元素`nums[slowIndex] = nums[fastIndex]` 
-> 慢指针前移 `++slowIndex`
可以合并一下 先用再自增 `nums[slowIndex++] = nums[fastIndex]` 

如果 `nums[fastIndex] == var`  快指针找到无效元素 慢指针维护边界
-> 无需进行任何处理

在循环外`return slowIndex` 即为新数组长度

```
slowIndex = 0;
for fastIndex = 0; fastIndex < num.Length; fastIndex++
	if nums[fastIndex] != var
		//nums[slowIndex] = nums[fastIndex]
		//++slowIndex;
		//优化成一句代码
		nums[slowIndex++] = nums[fastIndex] 
return slowIndex
```

#### 对撞指针

分别声明左右两端的指针 `leftIndex = 0`、`rightIndex = nums.Length - 1`
声明新数组 记录元素 `int[] newArray = new int[nums.Length]`

进入for循环 `for (int i = 0; i < num.Length; i++)`

如果 `nums[leftIndex] * nums[leftIndex] > nums[rightIndex] * nums[rightIndex]`
-> 记录元素 放到新数组最末端 `newArray[nums.Length - 1 - i] = nums[leftIndex] * nums[leftIndex]`
-> 右移左指针 `++leftIndex`

如果 `nums[leftIndex] * nums[leftIndex] < nums[rightIndex] * nums[rightIndex]`
-> 记录元素 放到新数组最末端`newArray[nums.Length - 1 - i] = nums[rightIndex] * nums[rightIndex]`
-> 并左移右指针 `--rightIndex`

循环结束，在循环外 `return newArray`

```
leftIndex = 0
rightIndex = nums.Length - 1
array = new int[nums.Length]
for int i = 0; i < nums.Length; i++
	if nums[leftIndex] * nums[leftIndex] > nums[rightIndex] * nums[rightIndex]
		newArray[nums.Length - 1 - i] = nums[leftIndex] * nums[leftIndex]
		++leftIndex;
	else
		newArray[nums.Length - 1 - i] = nums[rightIndex] * nums[rightIndex]
		--rightIndex;
return newArray
```

**注意事项**
1. 新数组在循环中的索引一定是 `nums.Length - 1 - i` 不减这个1就会超出索引边界

#### 滑动窗口

分别声明代表边界的左右指针 `leftIndex = 0` 右指针可以在for循环中声明

声明代表最小长度的变量，给一个默认值 `minLength = int.MaxValue` 返回结果时用三目运算符判断该值是否变化即可

声明一个存储元素和的变量 `sum`

进入for循环`for (int rightIndex = 0; rightIndex < nums.Length; rightIndex++)`每次循环扩张右指针

在for循环内部计算窗口所有元素和 `sum += nums[rightIndex]`

在for循环内部加入一个while循环，判断当前总和是否 >= 目标值 `while (sum >= target)`

在while循环内，记录当前最小长度，缩紧左指针
先判断当前窗口内部的元素是否小于最大值
如果是则记录 `minLength = Math.Min(minLength, rightIndex - leftIndex + 1)`
窗口元素和减去左指针元素，左指针右移一格 `sum -= nums[leftIndex++]`

在循环外返回结果 return `minLength == int.MaxValue ? 0 : minLength`

```
leftIndex = 0
sum = 0;
minLength = int.MaxValue
for rightIndex = 0; rightIndex < nums.Length; rightIndex++
	sum += nums[rightIndex]
	while sum >= target
		minLength = Math.Min(minLength, rightIndex - leftIndex + 1)
		sum -= nums[leftIndex++]
return minLength == int.MaxValue ? 0 : minLength
```

### 模拟

没有特别的算法技巧，按照题目描述的过程一步一步用代码还原出来，这种方法统称为模拟

模拟的难点不在算法，在于**边界条件的处理**，比如螺旋矩阵II中的循环不变量（统一用左闭右开还是左闭右闭），奇数n的中心点处理，边界收缩的时机等等，稍微想不清楚就会乱。
