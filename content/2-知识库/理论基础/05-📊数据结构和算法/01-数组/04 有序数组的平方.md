## 题目描述

给你一个按 **非递减顺序** 排序的整数数组 `nums`，返回 **每个数字的平方** 组成的新数组，要求也按 **非递减顺序** 排序。

**示例 1：**

**输入：** nums = \[-4,-1,0,3,10]
**输出：** \[0,1,9,16,100]
**解释：** 平方后，数组变为 \[16,1,0,9,100]
排序后，数组变为 \[0,1,9,16,100]

**示例 2：**

**输入：** nums = \[-7,-3,2,3,11]
**输出：** \[4,9,9,49,121]

**提示：**

- `1 <= nums.length <= 104`
- `-104 <= nums[i] <= 104`
- `nums` 已按 **非递减顺序** 排序

**进阶：**

- 请你设计时间复杂度为 `O(n)` 的算法解决本问题

## 解法三：双指针（自己做出来的）

**核心思路：**

**时间复杂度：** O(n)
**空间复杂度：** O(1)

```csharp
public class Solution {
    public int[] SortedSquares(int[] nums) {
        int[] newArrey = new int[nums.Length];
        int leftIndex = 0;
        int rightIndex = newArrey.Length - 1;
        for (int i = 0; i < newArrey.Length; i++){
            if (nums[leftIndex] * nums[leftIndex] > nums[rightIndex] * nums[rightIndex]){
                newArrey[newArrey.Length - i - 1] = nums[leftIndex] * nums[leftIndex];
                ++leftIndex;
            }
            else
            {
                newArrey[newArrey.Length - i - 1] = nums[rightIndex] * nums[rightIndex];
                --rightIndex;
            }
        }
        return newArrey;
    }
}
```
