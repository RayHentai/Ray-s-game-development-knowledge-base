## 解法二：模拟（跟着Claude写出来的最优解）

**核心思路：** 循环不变量原则

**时间复杂度：** O(n²)
**空间复杂度：** O(1)

```csharp
public class Solution {
    public int[][] GenerateMatrix(int n) {
        int[][] matrix = new int[n][];
        for (int i = 0; i < n; i++)
            matrix[i] = new int[n];
        int top = 0;//上边界
        int bottom = n - 1;//下边界
        int left = 0;//左边界
        int right = n - 1;//右边界
        int num = 1;
        while (top <= bottom && left <= right){
            for(int i = left; i <= right; i++)
                    matrix[top][i] = num++;
            ++top;
            for(int i = top; i <= bottom; i++)
                matrix[i][right] = num++;
            --right;
            for(int i = right; i >= left; i--)
                matrix[bottom][i] = num++;
            --bottom;
            for(int i = bottom; i >= top; i--)
                matrix[i][left] = num++;
            ++left;
        }
        return matrix;
    }
} 
```
