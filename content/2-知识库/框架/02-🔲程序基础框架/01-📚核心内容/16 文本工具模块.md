## 主要作用

在游戏开发中，经常会对字符串进行一些处理
而这些处理往往会在多处使用，为了减少代码冗余
往往会把常用的文本操作逻辑，封装到一个工具类中提供给外部使用

因此文本工具模块，主要就是**提取通用的文本处理逻辑并封装为方法**，供外部使用

---

## 制作思路

**新建一个文本工具类，主要提供以下方法：**
1. 字符串拆封公共方法
	- 字符串拆分为字符串数组
	- 字符串拆分为int数组
2. 数字前补0转字符串
3. 秒转时分秒
4. 秒转00:00:00
5. 大数据数值转换
6. 等等

更多内容，在实际开发时根据需求加入到工具类中即可

### 字符串拆分

在配置表中，经常会配置如下格式的字符串数据
`1,10;2,4;32,1`
`2,11|3,12|4,55`
遇到这种数据时，我们往往会使用 `string` 中的 `Split` 方法进行拆分

我们想要通过封装满足以下的需求
1. 避免符号填写错误，英文字符填写成中文字符
2. 能够直接拆分获取到int数组

```csharp
/// <summary>
/// 字符串工具
/// </summary>
public class TextUtil
{
    /// <summary>
    /// 拆分字符串 返回字符串数组
    /// </summary>
    /// <param name="str">字符串</param>
    /// <param name="type">拆分字符类型 1; 2, 3% 4: 5空格 6| 7_ </param>
    /// <returns>字符串数组</returns>
    public static string[] SplitStr(string str, int type = 1)
    {
        if (str == "")
            return new string[0];
        string newStr = str;
        if (type == 1)
        {
            //为了避免填成中文符号 先进行替换
            while (newStr.IndexOf('；') != -1)
                newStr = newStr.Replace('；', ';');
            return newStr.Split(';');
        }
        else if (type == 2)
        {
            while (newStr.IndexOf('，') != -1)
                newStr = newStr.Replace('，', ',');
            return newStr.Split(',');
        }
        else if (type == 3)
        {
            return newStr.Split('%');
        }
        else if (type == 4)
        {
            while (newStr.IndexOf('：') != -1)
                newStr = newStr.Replace('：', ':');
            return newStr.Split(':');
        }
        else if (type == 5)
        {
            return newStr.Split(' ');
        }
        else if (type == 6)
        {
            return newStr.Split('|');
        }
        else if (type == 7)
        {
            return newStr.Split('_');
        }
        return new string[0];
    }
    /// <summary>
    /// 拆分字符串 返回int数组
    /// </summary>
    /// <param name="str">字符串</param>
    /// <param name="type">拆分字符类型 1; 2, 3% 4: 5空格 6| 7_ </param>
    /// <returns>字符串数组</returns>
    public static int[] SplitStrToIntArr(string str, int type = 1)
    {
        //得到拆分后的字符串数组
        string[] strs = SplitStr(str, type);
        if (strs.Length == 0)
            return new int[0];
        //把字符串转换成int数组 返回出去
        return Array.ConvertAll<string, int>(strs, (str) =>
        {
            return int.Parse(str);
        });
    }
    /// <summary>
    /// 专门拆分多组键值对形式数据的 以int返回
    /// 0001,100;0002;99;0003;5
    /// </summary>
    /// <param name="str">字符串</param>
    /// <param name="typeOne">待拆分字符串分隔符 1; 2, 3% 4: 5空格 6| 7_</param>
    /// <param name="typeTwo">键值对的分隔符 1; 2, 3% 4: 5空格 6| 7_</param>
    /// <param name="callBack">回调函数</param>
    public static void SplitToIntArrTwice(string str, int typeOne, int typeTwo, UnityAction<int, int> callBack) 
    {
        string[] strs = SplitStr(str, typeOne); //先拆字符串 返回字符串数组
        if (strs.Length == 0) //为空数组不执行下面逻辑
            return;
        for (int i = 0; i < strs.Length; i++) //再拆字符串转为int数组
        {
            int[] ints = SplitStrToIntArr(strs[i], typeTwo);
            if (ints.Length == 0) //如果int数组长度为0 重新进入循环
                continue;
            callBack?.Invoke(ints[0], ints[1]); //处理逻辑
        }
    }
    /// <summary>
    /// 专门拆分多组键值对形式数据 以string返回
    /// </summary>
    /// <param name="str">字符串</param>
    /// <param name="typeOne">待拆分字符串分隔符 1; 2, 3% 4: 5空格 6| 7_</param>
    /// <param name="typeTwo">键值对的分隔符 1; 2, 3% 4: 5空格 6| 7_</param>
    /// <param name="callBack">回调函数</param>
    public static void SplitStrTwice(string str, int typeOne, int typeTwo, UnityAction<string, string> callBack)
    {
        string[] strs = SplitStr(str, typeOne); //先拆字符串 返回字符串数组
        if (strs.Length == 0) //为空数组不执行下面逻辑
            return;
        for (int i = 0; i < strs.Length; i++) //再拆字符串转为int数组
        {
            string[] strs2 = SplitStr(strs[i], typeTwo);
            if (strs2.Length == 0) //如果int数组长度为0 重新进入循环
                continue;
            callBack?.Invoke(strs2[0], strs2[1]); //处理逻辑
        }
    }
}
```

### 数字转字符串

**1. 转字符串数字前补0**

在游戏开发中界面显示时，经常可能会有这样的需求：要在界面中显示
0001
01
010
也就是说想要让某个数字在转换为字符串时，**可以指定转换的字符串长度**，并且能够在数字前自动的添加0

**2. 转字符串保留n位小数**
在游戏开发中界面显示时，经常可能会有这样的需求，要在界面中显示
3.1415926 保留两位小数 只显示 3.14
也就是说想要让某个数字在转换为字符串时，**可以指定保留小数点后的位数**

```csharp
/// <summary>
/// 得到指定长度的数字转字符串内容
/// 长度不够会在前面补0，超出长度保留原数值
/// </summary>
/// <param name="value">值</param>
/// <param name="length">长度</param>
/// <returns></returns>
public static string GetNumStr(int value, int length)
{
	//ToString 中传入一个Dn 代表把数字转换为长度为n的字符串
	//长度不够会自动在前补0
	return value.ToString($"D{length}");
}
/// <summary>
/// 让浮点数保留小数点后n位
/// 长度不够会在前面补0，超出长度保留原数值
/// </summary>
/// <param name="value">值</param>
/// <param name="length">长度</param>
/// <returns></returns>
public static string GetDecimalStr(float value, int length)
{
	//ToString 中传入一个Fn 代表保留几位小数
	//长度不够会自动在前补0
	return value.ToString($"F{length}");
}
```

### 秒转时分秒


在游戏开发中界面显示时，经常可能会有这样的需求，要在界面中显示 n时n分n秒 的倒计时

而一般都是根据倒计时剩余秒数进行转换的，因此可以将其封装为一个常用方法

**1. 提供一个 秒转 n时n分n秒 的方法，需求：**

1. 函数有两个参数：秒数 和 是否忽略0（若为 0时0分53秒 前面的0应该不拼接 直接显示53秒）
2. 可以传入用于拼接的 时分秒 字符
3. 当忽略0时，如果存在时，分和秒也必须显示；如果存在分，秒也必须显示（避免出现 1时53秒、1时5分 这种显示）

**2. 提供一个 秒转00:00:00 的方法，需求：**
1. 用:作为间隔符进行封装
2. 每个位置的格式为最少2位(即存在这样的格式 01:05:35 秒)

```csharp
private static StringBuilder resultStr = new StringBuilder();
/// <summary>
/// 秒转时分秒字符串
/// </summary>
/// <param name="time">秒</param>
/// <param name="isKeepLength">是否保持2位数字</param>
/// <param name="egZero">是否忽略0 默认不忽略</param>
/// <param name="hourStr">时显示字符</param>
/// <param name="minuteStr">分显示字符</param>
/// <param name="secondStr">秒显示字符</param>
/// <returns></returns>
public static string SecondToHMS(int time, bool isKeepLength = false, bool egZero = false, string hourStr = "时", string minuteStr = "分", string secondStr = "秒")
{
	if (time < 0) //如果时间输入不合法
		time = 0;
	//计算小时
	int hour = time / 3600;
	int minute = time % 3600 / 60;
	int second = time % 60;
	resultStr.Clear();//清空
	//不忽略0 时不为0
	if (!egZero || hour != 0) //拼接时
	{
		resultStr.Append(isKeepLength ? GetNumStr(hour, 2) : hour);
		resultStr.Append(hourStr);
	}
	//不忽略0 分不为0 时不为0
	if (!egZero || minute != 0 || hour != 0) //拼接分
	{
		resultStr.Append(isKeepLength ? GetNumStr(minute, 2) : minute);
		resultStr.Append(minuteStr);
	}
	//不忽略0 秒不为0 分不为0 时不为0
	if (!egZero || second != 0 || minute != 0 || hour != 0)
	{
		resultStr.Append(isKeepLength ? GetNumStr(second, 2) : second);
		resultStr.Append(secondStr);
	}
	if (resultStr.Length == 0)
	{
		resultStr.Append(isKeepLength ? GetNumStr(second, 2) : second);
		resultStr.Append(secondStr);
	}
	return resultStr.ToString();
}
/// <summary>
/// 将秒转为 00:00:00的形式
/// </summary>
/// <param name="time">时间</param>
/// <param name="egZero">是否忽略0 默认不忽略</param>
/// <returns></returns>
public static string SecondToHMSKeepLength(int time, bool egZero = false)
{
	return SecondToHMS(time, true, egZero, ":", ":", "");
}
```

### 大数据数值转字符串


在游戏开发时，为了避免在界面中显示过长的数值，往往需要把过长数值进行转换，比如：
1198678978 转换为 11亿9千万
5328987 转换为 532万8千
因为如果直接在界面上显示过长的数值可能会影响UI布局
而过大的数也可以直接粗略显示
可以直接省略后面的内容
显示一个大概数值

因此可以将其封装为一个常用方法

**需求：**
1. 以 亿 和 万 作为转换临界点

```csharp
public static string GetBigDataToString(int num)
{
    //一亿一千万
    if (num >= 100000000)
    {
        return BigDataChange(num, 100000000, "亿", "千万");
    }
    //一万一千
    else if (num >= 10000)
    {
        return BigDataChange(num, 10000, "万", "千");
    }
    //不满足条件直接返回
    else
    {
        return num.ToString();
    }
}
/// <summary>
/// 内部方法 大数据转为对应的字符串拼接
/// </summary>
/// <param name="num">数字</param>
/// <param name="company">单位数值</param>
/// <param name="bigCompany">大单位</param>
/// <param name="littleCompany">小单位</param>
/// <returns></returns>
private static string BigDataChange(int num, int company, string bigCompany, string littleCompany)
{
    resultStr.Clear();
    resultStr.Append(num / company);
    resultStr.Append(bigCompany);
    int temp = company / 10;
    if (num % company / temp != 0)
    {
        resultStr.Append(num % company / temp);
        resultStr.Append(littleCompany);
    }
    return resultStr.ToString();
}
```
