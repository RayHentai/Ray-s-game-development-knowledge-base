# 06 音效管理模块

**所属模块**：[[00 程序基础框架 总览]]

## 主要作用

游戏开发中有很多会用到音乐的地方
比如：背景音乐、打击音效、UI音效、特效音效等等

而不同的音乐音效会在不同的模块中去进行播放
比如：技能释放、怪物受伤、角色受伤、副本通关、奖励发放

如果不写一个音效管理模块
那么在播放音乐音效时会有很多冗余的代码，不方便管理，整体结构会显得较为杂乱

因此单独实现一个独立的音效管理模块
专门用于管理控制游戏中的所有音乐相关的功能，提供给外部使用

---

## 制作思路

**实现一个音乐管理器，提供一些公共方法给外部使用，比如：**
**背景音乐部分：**
1. 播放背景音乐
2. 停止背景音乐
3. 暂停背景音乐
4. 设置背景音乐大小

**音效部分：**
1. 播放音效
2. 自动移除播放完成的音效
3. 停止播放指定音效
4. 设置音效声音大小
5. 暂停或继续播放所有音效

**注意：**
1. 音效和背景音乐不同
	- 音效存在多个，并且音效需要管理是否结束
	- 因此需要用容器记录音效组件
2. 音效分为循环和非循环
	- 非循环的需要我们检测它播放结束
	- 循环的需要让外部进行管理

## 基本实现

```csharp
public class MusicMgr : BaseManager<MusicMgr>
{
    private AudioSource bkMusic = null;//空对象对象依附的音乐脚本
    private float bkMusicValue = 0.1f;//背景音乐音量大小
    private List<AudioSource> soundList = new List<AudioSource>();//存储音效组件的容器
    private GameObject soundObj; //音效组件依附的对象
    private float soundValue = 0.1f;//音效大小
    private bool isPause = false;//避免音效被暂停时误删
    private MusicMgr() 
    {
        //利用公共mono模块 持续检测不在播放状态的音效组件 然后移除
        //Update间隔时间太短 可以换成FixedUpdate
        MonoMgr.Instance.AddFixedUpdateListener(Update);
    }
    private void Update() 
    {
        if (isPause) //暂停直接不走下面逻辑
            return;
        //反向遍历 避免出问题
        for (int i = soundList.Count - 1; i >= 0; --i) 
        {
            if (!soundList[i].isPlaying)
            {
                GameObject.Destroy(soundList[i]);
                soundList.RemoveAt(i);
            }
        }
    }
    /// <summary>
    /// 播放背景音乐
    /// </summary>
    /// <param name="name">音乐资源名</param>
    public void PlayBKMusic(string name)
    {
        if (bkMusic == null)
        {
            GameObject obj = new GameObject("BKMusic");
            GameObject.DontDestroyOnLoad(obj);
            bkMusic = obj.AddComponent<AudioSource>();
        }
        ABResMgr.Instance.LoadResAsync<AudioClip>("music", name, (clip) =>
        {
            bkMusic.clip = clip;
            bkMusic.volume = bkMusicValue;
            bkMusic.loop = true;
            bkMusic.Play();
        });
    }
    /// <summary>
    /// 停止播放背景音乐
    /// </summary>
    public void StopBKMusic()
    {
        if (bkMusic == null)
            return;
        bkMusic.Stop();
    }
    /// <summary>
    /// 暂停背景音乐
    /// </summary>
    public void PauseBKMusic()
    {
        if (bkMusic == null)
            return;
        bkMusic.Pause();
    }
    /// <summary>
    /// 修改背景音乐音量大小
    /// </summary>
    /// <param name="value">音量值</param>
    public void ChangeBKMusicValue(float value)
    {
        bkMusicValue = value;
        if (bkMusic == null)
            return;
        bkMusic.volume = bkMusicValue;
    }
    /// <summary>
    /// 播放音效
    /// </summary>
    /// <param name="name">音效资源名</param>
    /// <param name="isLoop">是否循环</param>
    /// <param name="callback">回调函数</param>
    /// <param name="isSync">是否同步加载</param>
    public void PlaySound(string name, bool isLoop = false, UnityAction<AudioSource> callback = null, bool isSync = false) 
    {
        if (soundObj == null)
            //音效过场景就消失了 不需要设置过场景不移除
            soundObj = new GameObject("soundObj");
        ABResMgr.Instance.LoadResAsync<AudioClip>("music", name, (clip) =>
        {
            AudioSource audioSource = soundObj.AddComponent<AudioSource>();
            audioSource.clip = clip;
            audioSource.loop = isLoop;
            audioSource.volume = soundValue;
            audioSource.Play();
            soundList.Add(audioSource);//存储
            callback?.Invoke(audioSource);
        }, isSync);
    }
    /// <summary>
    /// 停止播放音效
    /// </summary>
    /// <param name="audioSource">关联的音效脚本</param>
    public void StopSound(AudioSource audioSource) 
    {
        if (soundList.Contains(audioSource))
        {
            audioSource.Stop();
            GameObject.Destroy(audioSource);
            soundList.Remove(audioSource);
        }
    }
    /// <summary>
    /// 改变音效大小
    /// </summary>
    /// <param name="value">音量值</param>
    public void ChangeSoundValue(float value) 
    {
        soundValue = value;
        for (int i = 0; i < soundList.Count; i++)
            soundList[i].volume = value;
    }
    /// <summary>
    /// 播放还是暂停所有音效
    /// </summary>
    /// <param name="isPlay"></param>
    public void PlayOrPauseAllSound(bool isPlay)
    {
        if (isPlay)
        {
            for (int i = 0; i < soundList.Count; i++)
                soundList[i].Play();
            isPause = false;
        }
        else
        {
            for (int i = 0; i < soundList.Count; i++)
                soundList[i].Pause();
            isPause = true;
        }
            
    }
}
```

**潜在问题：**
1. 频繁的删除创建音效组件，比较浪费性能
2. 3D音效需要让音效对象位置产生变化，而当前音效对象的位置不会变化

---

## 优化

**频繁创建组件问题**
目前音效管理器中的音效组件会频繁的创建和删除
这样会产生大量的内存垃圾，并且频繁创建对象也会带来性能消耗
因此可以利用缓存池对其进行优化

**3D音效问题**
由于使用了缓存池，因此AudioSource依附的将会是一个个独立的游戏对象
而3D音效主要考虑的问题是：
1. 音效依附在对象上跟随对象移动
2. 音效中3D相关参数的设置

如果想要使用3D音效，那么只需要获取到音效组件和它依附的对象就可以迎刃而解了
1. 获取音效组件依附的对象，改变它的父对象，设置它的位置，音效便可以跟随对象移动
2. 获取音效组件改变其中的3D音效相关参数即可完成相关设置

目前过场景时，音效相关对象会被自动的删除
但是音效管理器中我们的容器还占着引用，我们应该提供方法清空容器

```csharp
public class MusicMgr : BaseManager<MusicMgr>
{
    private AudioSource bkMusic = null;//空对象对象依附的音乐脚本
    private float bkMusicValue = 0.1f;//背景音乐音量大小
    private List<AudioSource> soundList = new List<AudioSource>();//存储音效组件的容器
    private float soundValue = 0.1f;//音效大小
    private bool isPause = false;//避免音效被暂停时误删
    private MusicMgr() 
    {
        //利用公共mono模块 持续检测不在播放状态的音效组件 然后移除
        //Update间隔时间太短 可以换成FixedUpdate
        MonoMgr.Instance.AddFixedUpdateListener(Update);
    }
    private void Update() 
    {
        if (isPause) //暂停直接不走下面逻辑
            return;
        //反向遍历 避免出问题
        for (int i = soundList.Count - 1; i >= 0; --i) 
        {
            if (!soundList[i].isPlaying)
            {
                soundList[i].clip = null;//置空切片
                PoolMgr.Instance.PushObj(soundList[i].gameObject);//压入缓存池
                soundList.RemoveAt(i);
            }
        }
    }
    /// <summary>
    /// 播放背景音乐
    /// </summary>
    /// <param name="name">音乐资源名</param>
    public void PlayBKMusic(string name)
    {
        if (bkMusic == null)
        {
            GameObject obj = new GameObject("BKMusic");
            GameObject.DontDestroyOnLoad(obj);
            bkMusic = obj.AddComponent<AudioSource>();
        }
        ABResMgr.Instance.LoadResAsync<AudioClip>("music", name, (clip) =>
        {
            bkMusic.clip = clip;
            bkMusic.volume = bkMusicValue;
            bkMusic.loop = true;
            bkMusic.Play();
        });
    }
    /// <summary>
    /// 停止播放背景音乐
    /// </summary>
    public void StopBKMusic()
    {
        if (bkMusic == null)
            return;
        bkMusic.Stop();
    }
    /// <summary>
    /// 暂停背景音乐
    /// </summary>
    public void PauseBKMusic()
    {
        if (bkMusic == null)
            return;
        bkMusic.Pause();
    }
    /// <summary>
    /// 修改背景音乐音量大小
    /// </summary>
    /// <param name="value">音量值</param>
    public void ChangeBKMusicValue(float value)
    {
        bkMusicValue = value;
        if (bkMusic == null)
            return;
        bkMusic.volume = bkMusicValue;
    }
    /// <summary>
    /// 播放音效
    /// </summary>
    /// <param name="name">音效资源名</param>
    /// <param name="isLoop">是否循环</param>
    /// <param name="callback">回调函数</param>
    /// <param name="isSync">是否同步加载</param>
    public void PlaySound(string name, bool isLoop = false, UnityAction<AudioSource> callback = null, bool isSync = false) 
    {
        ABResMgr.Instance.LoadResAsync<AudioClip>("sound", name, (clip) =>
        {
            //从缓存池取出对象
            AudioSource audioSource = PoolMgr.Instance.GetObj("sound/SoundObj").GetComponent<AudioSource>();
            audioSource.Stop();//如果是之前使用过的直接停止播放
            audioSource.clip = clip;
            audioSource.loop = isLoop;
            audioSource.volume = soundValue;
            audioSource.Play();
            //从缓存池拿出对象 有可能是之前使用过的 已经被存在容器中了
            //如果不存在该音效组件才存储
            if (!soundList.Contains(audioSource))
                soundList.Add(audioSource);
            callback?.Invoke(audioSource);
        }, isSync);
    }
    /// <summary>
    /// 停止播放音效
    /// </summary>
    /// <param name="audioSource">关联的音效脚本</param>
    public void StopSound(AudioSource audioSource) 
    {
        if (soundList.Contains(audioSource))
        {
            audioSource.Stop();
            soundList.Remove(audioSource);
            audioSource.clip = null;//不用了 清空切片 避免占用
            PoolMgr.Instance.PushObj(audioSource.gameObject);//压入缓存池
        }
    }
    /// <summary>
    /// 改变音效大小
    /// </summary>
    /// <param name="value">音量值</param>
    public void ChangeSoundValue(float value) 
    {
        soundValue = value;
        for (int i = 0; i < soundList.Count; i++)
            soundList[i].volume = value;
    }
    /// <summary>
    /// 播放还是暂停所有音效
    /// </summary>
    /// <param name="isPlay"></param>
    public void PlayOrPauseAllSound(bool isPlay)
    {
        if (isPlay)
        {
            for (int i = 0; i < soundList.Count; i++)
                soundList[i].Play();
            isPause = false;
        }
        else
        {
            for (int i = 0; i < soundList.Count; i++)
                soundList[i].Pause();
            isPause = true;
        }
    }
    /// <summary>
    /// 清空音效相关记录
    /// 一定要在清空缓存池之前使用
    /// </summary>
    public void ClearSound()
    {
        for (int i = 0; i < soundList.Count; i++)
        {
            soundList[i].Stop();//停止
            soundList[i].clip = null;//置空切片
            soundList.Clear();//清空容器
            PoolMgr.Instance.PushObj(soundList[i].gameObject);//压入缓存池
        }
    }
}
```
