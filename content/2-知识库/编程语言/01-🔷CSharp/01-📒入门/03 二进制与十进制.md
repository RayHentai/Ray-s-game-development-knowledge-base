## 基本规则

- **1 字节 = 8 bit**，用二进制表示：`0000 0000`，可表示 256 个数字（0~255）
- **有符号整型**：最高位（第一个数字）表示正负，0 为正，1 为负
- **无符号整型**：第一个数字也用来存数值，不表示正负

---

## 十进制 → 二进制

**方法**：不断除以 2，取余数，从下往上读。

```
100 ÷ 2 = 50  余 0   ↑ 最后读
 50 ÷ 2 = 25  余 0   ↑
 25 ÷ 2 = 12  余 1   ↑
 12 ÷ 2 =  6  余 0   ↑
  6 ÷ 2 =  3  余 0   ↑
  3 ÷ 2 =  1  余 1   ↑
  1            首位   ↑ 最先读

从下往上读：1  1  0  0  1  0  0 = 1100100
```

> 100（十进制）= 1100100（二进制）✅

---

## 二进制 → 十进制

**方法**：从右往左，第 0 位起，该位是 1 就加 2^n，是 0 跳过。

```
1  1  0  0  1  0  0
↑  ↑  ↑  ↑  ↑  ↑  ↑
6  5  4  3  2  1  0  ← 位数

第6位：1 → 2⁶ = 64
第5位：1 → 2⁵ = 32
第2位：1 → 2² =  4
其余位：0 → 跳过
```

>1100100（二进制） = 64 + 32 + 4 = 100（十进制） ✅
