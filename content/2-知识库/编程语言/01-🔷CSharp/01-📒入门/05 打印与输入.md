## 输出（打印）

| 语法 | 说明 |
|------|------|
| `Console.WriteLine()` | 打印后**自动换行**；括号为空时只输出一个空行 |
| `Console.Write()` | 打印后**不换行** |

```csharp
Console.WriteLine("你好");  // 输出"你好"并换行
Console.Write("你好");      // 输出"你好"不换行
Console.WriteLine();        // 只输出一个空行
```

---

## 输入

| 语法 | 说明 |
|------|------|
| `Console.ReadLine()` | 玩家输入一串内容，**回车结束**，返回 `string` |
| `Console.ReadKey()` | 玩家输入**任意一个键**结束，不需要回车 |

```csharp
string input = Console.ReadLine();
Console.ReadKey();                    // 等待任意键（常用于"按任意键继续"）
char c = Console.ReadKey(true).KeyChar; // 输入不显示在控制台
```

## 练习

> 1. 请简单描述 Console.Write 和 Console.WriteLine 的区别，Console.ReadKey 和 Console.ReadLine 的区别

- `Console.Write()`：打印完后不空行
- `Console.WriteLine()`：打印完后自动空行
- `Console.ReadKey()`：输入任意键后执行下一步
- `Console.ReadLine()`：输入回车后执行下一步

> 2. 请用户输入用户名、年龄、班级

```csharp
Console.WriteLine("请输入用户名、年龄、班级");
Console.ReadLine();
Console.WriteLine("登录成功");
```

> 3. 问用户喜欢什么运动，不管用户输入什么，你都回答："哈哈，好巧我也喜欢这个运动"

```csharp
Console.WriteLine("你喜欢什么运动呢？");
Console.ReadLine();
Console.WriteLine("哈哈，好巧，我也喜欢这个运动");
```

> 4. 在控制台上输出如下 10×10 的空心星型方阵（不用循环，直接打印）

`**********`
`*        *`
`*        *`
`*        *`
`*        *`
`*        *`
`*        *`
`*        *`
`*        *`
`**********`

```csharp
Console.WriteLine("**********");

Console.WriteLine("*        *");
Console.WriteLine("*        *");
Console.WriteLine("*        *");
Console.WriteLine("*        *");
Console.WriteLine("*        *");
```
