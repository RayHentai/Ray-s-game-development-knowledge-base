## 什么是常量

> 声明一些**不会变化的值**，比如最大血量、数学常数 π 等。

```csharp
const 变量类型 变量名 = 初始值;

const int MAX_HP    = 100;
const float PI        = 3.14f;
const string GAME_NAME = "冒险游戏";
```

---

## 常量的特点

- **必须在声明时初始化**（赋值），不能先声明再赋值
- **不能被修改**，尝试修改会报编译错误
- 数值本身（字面量）也是常量，如 `1`、`3.14`、`"hello"`

---

## 练习

> 1. 常量有什么特点？

- 声明时必须赋值
- 不可修改
- 声明句式就是在变量声明前加 `const`

> 2. 请简述你觉得在游戏开发中常量会用来干什么？

- 设定用于后续计算的基础值（如圆周率、重力加速度）
- 设定不会改变的配置值（如最大血量、屏幕宽高）
