## 实用语法

### ReadKey 的两种用法

```csharp
// 获取输入的字符，存为 char 类型
char key = Console.ReadKey().KeyChar;

// 获取输入的键，存为枚举类型（ConsoleKey）
ConsoleKey key = Console.ReadKey().Key;
// 可用于判断方向键：if (key == ConsoleKey.UpArrow)
```

### 获取变量类型的默认值

```csharp
int defaultInt = default(int);     // 0
bool defaultBool = default(bool);    // false
string defaultStr = default(string);  // null
```

## 内存优化意识（早期养成习惯）

- 节约内存：少 `new` 新对象，少产生垃圾
- 尽力减少 GC（垃圾回收）：合理使用 `static`、`string`、`StringBuilder`
- 在循环体**外**声明变量，在循环体**内**修改值，节省内存分配

## 语句块与变量生命周期

> 语句块 `{}` 会影响变量的生命周期：在 `{}` 内声明的变量，出了这对括号就不存在了。

```csharp
if (true)
{
    int b = 10;   // b 只在这个 if 块内有效
}
// Console.WriteLine(b);   // ❌ 这里访问不到 b，会报错
```
