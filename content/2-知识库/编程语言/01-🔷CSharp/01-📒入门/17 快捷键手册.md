## Visual Studio 常用快捷键

| 快捷键            | 功能                 |
| -------------- | ------------------ |
| `F5`           | 控制台运行（启动调试）        |
| `F9`           | 打 / 取消断点           |
| `F12`          | 转到定义（跳转到变量/方法的声明处） |
| `F2`           | 重命名（所有引用处同步修改）     |
| `Ctrl + K + C` | 注释选中代码             |
| `Ctrl + K + U` | 取消注释               |
| `Ctrl + C`（双击） | 复制整行               |
