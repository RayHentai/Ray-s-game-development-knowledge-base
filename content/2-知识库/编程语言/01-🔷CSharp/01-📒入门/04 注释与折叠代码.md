## 注释

注释不会被编译执行，用来给自己或他人解释代码。

| 语法         | 类型       | 用途              |
| ---------- | -------- | --------------- |
| `//`       | 单行注释     | 注释当前行           |
| `/* 内容 */` | 多行注释     | 注释多行            |
| `///`      | XML 文档注释 | 鼠标悬停时显示说明，引用时可见 |


```csharp
// 这是单行注释

/*
   这是
   多行注释
*/

/// <summary>
/// 这是文档注释，调用此方法时悬停会显示这段说明
/// </summary>
void MyMethod() { }
```

## 折叠代码（#region）

用 `#region` 和 `#endregion` 把一段代码折叠起来，方便管理大文件。

```csharp
#region 初始化逻辑
int hp = 100;
int mp = 50;
string name = "勇者";
#endregion
```

> 在 Visual Studio 中点击左侧小箭头可展开/折叠。

---

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + K + C` | 注释选中代码 |
| `Ctrl + K + U` | 取消注释 |
