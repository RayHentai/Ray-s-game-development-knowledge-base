## while

>先判断条件，再不停循环执行代码块的内容

```csharp
while (bool类型值)
{
    // 满足条件就执行，不满足就停止
}

// 死循环（配合 break 使用）
while (true)
{
    if (退出条件) break;
}
```

> 4. 提示用户输入一个数，判断该数是否是素数并打印（只能被1和自身整除，1不是质数，最小质数是2）

```csharp
while (true)
{
    try
    {
        Console.WriteLine("请输入一个数，判断是否是质数");
        int n = int.Parse(Console.ReadLine());
        int i = 2;
        while (i < n) 
        { 
	        if (n % i == 0) 
		        break; 
	        i++; 
        }
        Console.WriteLine(i != n ? "{0}不是质数" : "{0}是质数", n);
    }
    catch { Console.WriteLine("请输入正确的数字"); }
}
```

> 9. 求数列 1, 1, 2, 3, 5, 8, 13... 的第20位数字是多少？

```csharp
int n1 = 1, n2 = 1, result = 0, index = 0;

{
    index++;
    if (index == 1) 
	    result = 1;
    else if (index == 2) 
	    result = 1;
```

## do...while — 先执行再判断

> 先斩后奏：至少执行一次，再判断是否继续。

```csharp
do
{
    // 先执行这里
} while (bool类型值);   // ⚠️ 注意末尾有分号！
```

### 练习

> 1. 要求用户输入用户名和密码，只要不是 admin/8888 就一直提示用户名或密码错误，请重新输入

```csharp
string userName, passWord;
bool isShow = false;
do
{
    if (isShow) 
	    Console.WriteLine("用户名或密码错误，请重新登录");
    Console.WriteLine("请输入用户名"); userName = Console.ReadLine();
    Console.WriteLine("请输入密码");   passWord = Console.ReadLine();
    isShow = true;
} while (userName != "admin" || passWord != "8888");
```

> 2. 不断提示请输入你的姓名，直到输入 q 结束

```csharp
string name;
do
{
    Console.WriteLine("请输入你的姓名");
    name = Console.ReadLine();
} while (name != "q");
```

## for

>用来准确高效得到一个范围中的所有数

- **初始表达式：**一般声明一个临时变量，用来计数
- **条件表达式：**表示进入循环的条件，是一个bool值
- **增量表达式：**用第一个空中的变量，进行自增减运算，填什么都可以

```csharp
for (初始表达式; 条件表达式; 增量表达式)
{
    // 循环体
}

// 三个空位可以不填（分号不能省），不填就是死循环
for (;;) { }
```

**运行逻辑：**
-  进入循环时执行一次初始表达式
-  每次执行前都会判断条件表达式中的条件，满足才执行循环体
-  执行完循环体后执行增量表达式 → 回到第2步

## 流程控制关键字

### break — 跳出循环

```csharp
for (int i = 0; i < 10; i++)
{
    if (i == 5) 
	    break;   // 跳出整个循环
}
```

**注意：** 嵌套循环中 `break` 只跳出**当前这一层**；`switch` 中的 `break` 不会跳出循环

### continue — 跳过本次，继续下次

```csharp
for (int i = 0; i < 10; i++)
{
    if (i % 2 == 0) continue;   // 偶数跳过
    Console.WriteLine(i);       // 输出 1 3 5 7 9
}
```

**注意：** `do...while` 中 `continue` 跳到 `while` 判断处，不是 `do` 开头

## 注意事项

- `while(true)` 是死循环，需要在内部用 `break` 跳出
- `for循环`括号中的空位可填可不填，不填可以在外层或内层补充
- `for循环`中括号不填东西就是死循环
- 可以嵌套其他语句块，循环内的嵌套中声明的变量，是一次性的
- 同层 `for循环`中可以声明同名变量，不同层会报错
- 💡 在循环外声明变量、在循环内修改值，节省内存分配
