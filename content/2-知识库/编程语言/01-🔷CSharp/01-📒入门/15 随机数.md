```csharp
Random random = new Random();
```

---

## 生成随机数

| 语法 | 生成范围 |
|------|----------|
| `random.Next()` | 任意非负整数 |
| `random.Next(x)` | `0 ~ x-1` |
| `random.Next(x, y)` | `x ~ y-1` |

```csharp
Random r = new Random();

int a = r.Next();          // 0 ~ int最大值
int b = r.Next(6);         // 0 ~ 5（模拟骰子：0到5）
int c = r.Next(1, 7);      // 1 ~ 6（模拟骰子：1到6）
int d = r.Next(10, 20);    // 10 ~ 19
```

> 💡 **记住**：`Next(x, y)` 是**左闭右开**，包含 x，不包含 y。

---

## 注意

- `Random` 对象最好**只创建一次**，反复使用同一个对象
- 如果在循环里每次 `new Random()`，短时间内生成的随机数可能完全一样（因为种子相同）

```csharp
// ❌ 不推荐：每次循环都 new，随机数可能重复
for (int i = 0; i < 5; i++)
{
    Random r = new Random();
    Console.WriteLine(r.Next(10));
}

// ✅ 推荐：在循环外创建，循环内使用
Random r = new Random();
for (int i = 0; i < 5; i++)
{
    Console.WriteLine(r.Next(10));
}
```

## 练习

> 1. 唐老狮打小怪兽：

- 唐老狮攻击力为8~12之间的一个值
- 小怪兽防御为10，血量为20

- 在控制台中通过打印信息表现唐老狮打小怪兽的过程，描述小怪兽的掉血情况
- 伤害计算公式：攻击力小于防御力时减血为0，否则减血为攻击力和防御力的差值

```csharp
Random r = new Random();
int monsterHp  = 20;
int monsterDef = 10;
Console.WriteLine("开始战斗");

while (true)
{
    int atk = r.Next(8, 13);
    if (atk > monsterDef)
    {
        int dmg = atk - monsterDef;
        monsterHp -= dmg;
        Console.WriteLine("本次攻击伤害{0}，怪物扣除{1}血量，剩余{2}血量", atk, dmg, monsterHp);
        if (monsterHp <= 0)
        {
            Console.WriteLine("战斗结束，怪物已被打倒");
            break;
        }
    }
    else
    {
        Console.WriteLine("本次攻击伤害{0}，怪物没有受到伤害", atk);
    }
    Console.WriteLine("请按任意键继续");
    Console.ReadKey(true);
    Console.Clear();
}
```
