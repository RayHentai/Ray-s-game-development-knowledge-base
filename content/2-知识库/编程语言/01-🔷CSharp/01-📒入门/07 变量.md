## 变量是什么

> 变量是一个有名字的"盒子"，用来存储数据。
> 通过变量存储后面可能随时需要更改的值，不要把数字写死在代码里。

> 💡 **口诀**：数字用 int，小数用 float，字符串用 string，真假用 bool

---

## 所有基础数据类型

### 有符号整型（可存负数）

| 类型 | 范围 | 字节 |
|------|------|------|
| `sbyte` | -128 ~ 127 | 1 |
| `short` | -32768 ~ 32767 | 2 |
| `int` | ±21亿 | 4 |
| `long` | ±900万兆 | 8 |

### 无符号整型（只存 0 和正数）

| 类型 | 范围 | 字节 |
|------|------|------|
| `byte` | 0 ~ 255 | 1 |
| `ushort` | 0 ~ 65535 | 2 |
| `uint` | 0 ~ 42亿 | 4 |
| `ulong` | 0 ~ 18百万兆 | 8 |

### 浮点数（小数，四舍五入存储）

| 类型 | 有效数字 | 字节 | 声明方式 |
|------|---------|------|----------|
| `float` | 7~8位 | 4 | `0.1f`（必须加 f） |
| `double` | 15~17位 | 8 | `0.1`（默认小数类型） |
| `decimal` | 27~28位 | 16 | `0.1m`（必须加 m） |

### 特殊类型

| 类型 | 说明 | 字节 |
|------|------|------|
| `bool` | true / false | 1 |
| `char` | 单个字符，用单引号 `'A'` | 2 |
| `string` | 字符串，用双引号 `"hello"` | 不固定 |

## 声明变量

```csharp
int age = 18;
float score = 99.5f;    // float 必须加 f
decimal money = 9.9m;   // decimal 必须加 m
string name = "小明";
bool isStudent = true;
char grade = 'A';

// 连续声明同类型变量
int i = 1, j = 2, k = 3;
```

**注意事项**
- `float` 小数必须加 `f`，否则默认识别为 `double`
- `decimal` 小数必须加 `m`
- 声明小数默认是 `double` 类型
- 变量使用前必须先初始化（赋值），否则报错

## 获取变量占用内存

```csharp
int i = sizeof(int); // i = 4
```

## 命名规则

**必须遵守：**
- 不能重名（同一作用域内）
- 不能以数字开头
- 不能使用程序关键字（关键字在 IDE 里显示为蓝色）
- 不能有特殊符号（下划线 `_` 除外）

**约定俗成：**
- **驼峰命名法**：首字母小写，后续单词首字母大写（用于**变量**）→ `myName`
- **帕斯卡命名法**：所有单词首字母大写（用于**函数、类**）→ `MyMethod`

## 练习

> 1. 下面代码的输出结果是什么？

```csharp
double num = 36.6;
Console.WriteLine("num");
// 输出：num（打印的是字符串 "num"，不是变量的值）
```

> 2. 声明 float 类型变量时，为何要在数字后面加 f？

- 生成浮点数变量默认是 double 类型，加 `f` 是告诉编译器这是 float 类型。

> 6. 请将二进制 11000111、001101、01010101 转为十进制

```
11000111 = 2⁷+2⁶+2²+2¹+2⁰ = 128+64+4+2+1 = 199
001101   = 2³+2²+2⁰ = 8+4+1 = 13
01010101 = 2⁶+2⁴+2²+2⁰ = 64+16+4+1 = 85
```

> 7. 请将十进制 99、1024、78937 转为二进制，写出计算过程

```
99    → 1100011
1024  → 10000000000
78937 → 10011010001011001
（以上均为从低位到高位计算，也可用短除法验证）
```

> 8. 下面的变量名哪些是错误的？U3d / No.1 / 3day / $money / class / Main / discount_1 / int / _  2C / ShangHai / abc / print

```
U3d        ✅ 合法（字母开头，含数字）
No.1       ❌ 含特殊符号 .
3day       ❌ 数字开头
$money     ❌ 含特殊符号 $
class      ❌ 关键字
Main       ⚠️ 合法但与入口方法同名，不推荐
discount_1 ✅ 合法
int        ❌ 关键字
_ 2C       ❌ 含空格
ShangHai   ✅ 合法
abc        ✅ 合法
print      ✅ 合法
```
