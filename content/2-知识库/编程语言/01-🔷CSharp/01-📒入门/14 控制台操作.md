## 设置控制台大小

```csharp
// ⚠️ 必须先设置窗口大小，再设置缓冲区大小
Console.SetWindowSize(宽, 高);    // 设置窗口大小
Console.SetBufferSize(宽, 高);    // 设置缓冲区大小

// 读取缓冲区尺寸
int h = Console.BufferHeight;    // 缓冲区的高
int w = Console.BufferWidth;     // 缓冲区的宽
```

**缓冲区说明：**
- 需要先设置窗口大小，再设置缓冲区大小
- 缓冲区 = 可以显示输入内容的范围
- 缓冲区大小**不能小于**窗口大小

---

## 设置光标位置

```csharp
// 从指定位置开始打印内容
Console.SetCursorPosition(横坐标, 纵坐标);
```

**坐标系说明：**
- 控制台**左上角**为原点 `(0, 0)`
- 向右是 X 轴正方向
- 向下是 Y 轴正方向
- ⚠️ 横纵单位不同：视觉上 `1 格Y ≈ 2 格X`（字符是纵向更高）
- ⚠️ 注意边界，光标超出范围会**报错**

## 设置颜色

```csharp
// 设置文字颜色（之后打印的所有内容都是这个颜色）
Console.ForegroundColor = ConsoleColor.Red;

// 设置背景颜色（需要再执行一次 Clear 才能全部生效）
Console.BackgroundColor = ConsoleColor.Black;
Console.Clear();
```

**常用颜色：**
`Black` / `White` / `Red` / `Green` / `Blue` / `Yellow` / `Cyan` / `Gray`

Console.CursorVisible = false;   // 隐藏光标（游戏中常用，避免闪烁）

## 实际项目中的典型用法

```csharp
// 初始化控制台（一般在程序开始时设置一次）
Console.SetWindowSize(50, 30);
Console.SetBufferSize(50, 30);
Console.CursorVisible = false;

// 在指定位置画内容
Console.SetCursorPosition(10, 5);
Console.ForegroundColor = ConsoleColor.Yellow;
Console.Write("★");

// 清空并重绘
Console.Clear();
```

## 练习

> 通过 W/A/S/D 键在控制台中控制一个黄色方块上下左右移动，注意边界问题，知识点：while，switch，输入输出等等

```csharp
// 初始化控制台
Console.SetWindowSize(30, 20);
Console.SetBufferSize(30, 20);
Console.CursorVisible = false;

int x = 14, y = 9;   // 初始位置（中间）

while (true)
{
    // 绘制方块
    Console.SetCursorPosition(x, y);
    Console.ForegroundColor = ConsoleColor.Yellow;
    Console.Write("■");

    // 读取输入
    ConsoleKey key = Console.ReadKey(true).Key;

    // 清除当前位置
    Console.SetCursorPosition(x, y);
    Console.Write(" ");

    // 根据按键移动，注意边界（控制台字符宽高比约1:2）
    switch (key)
    {
        case ConsoleKey.W: if (y > 0)  y--; break;
        case ConsoleKey.S: if (y < 19) y++; break;
        case ConsoleKey.A: if (x > 0)  x--; break;
        case ConsoleKey.D: if (x < 28) x++; break;
        case ConsoleKey.Escape: return; // 按 Esc 退出
    }
}
```
