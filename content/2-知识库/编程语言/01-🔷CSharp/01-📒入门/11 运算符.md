## 算术运算符

| 符号  | 名称  | 注意点           |
| --- | --- | ------------- |
| `=` | 赋值  | 先算右侧，再赋值给左侧   |
| `+` | 加   | 同上            |
| `-` | 减   | 同上            |
| `*` | 乘   | 同上            |
| `/` | 除   | 整数相除会**丢失小数** |
| `%` | 取余  | `4 % 3 = 1`   |

```csharp
// ⚠️ 整数除法陷阱
int a = 1 / 2;       // 结果是 0，不是 0.5！
float b = 1 / 2f;    // 结果是 0.5（其中一个数加 f）

// 💡 能被整除的判断：取余为 0
bool divisible = 10 % 2 == 0;   // true，10 能被 2 整除
```

## 复合运算符

>同样是先看等号 右边再看左边

```csharp
i += 1;   // i = i + 1
i -= 1;   // i = i - 1
i *= 2;   // i = i * 2
i /= 2;   // i = i / 2
i %= 3;   // i = i % 3
```

> ⚠️ 只能进行一种运算，不能混合（`+-*=` 这样写不行）

## 自增减运算符

```csharp
int i = 5;
int a = i++;   // a = 5（先用，后加）→ 执行后 i = 6
int b = ++i;   // i 先变 7，b = 7（先加，后用）

int a = i--;   // a = 7（先用，后减）→ 执行后 i = 6
int b = --i;   // i 先变 6，b = 6（减，后用）
```

## 字符串拼接

### `+` 拼接

**注意**：
- 变量打印不需要引号

```csharp
string name = "小明";
int age = 18;
Console.WriteLine("姓名：" + name + "，年龄：" + age);
```

**字符串拼接的运算顺序（从左往右）：**

```csharp
Console.WriteLine(1 + 2 + "" + 3 + 4);    // 334
Console.WriteLine("1" + 2 + 3 + 4);        // 1234
Console.WriteLine("1" + 2 + (3 + 4));      // 127
```

**注意**：
- 变量与变量拼接时要用字符串去连接，否则整型会相加

### `string.Format()` 占位符拼接

**注意**：
- `{}`里面一定要填数字，不然会**报错**
- `{}`占位符的数量可以大于变量数量，会舍弃多余数量的变量数量
- `{}`占位符的数量不能小于变量的数量，会**报错**
- `WriteLine`的`()`中也可以用这种写法

```csharp
//想要拼接的内容用占位符 {数字} 替代 数字从0~n 依次往后
string str = string.Format("姓名：{0}，年龄：{1}", name, age)
Console.WriteLine("姓名：{0}，年龄：{1}", name, age);
```

## 条件运算符（比较运算符）

| 符号   | 含义     |
| ---- | ------ |
| `>`  | 是否大于   |
| `<`  | 是否小于   |
| `==` | 是否等于   |
| `!=` | 是否不等于  |
| `>=` | 是否大于等于 |
| `<=` | 是否小于等于 |

**注意：**
- 结果只返回 `bool` 类型
- `条件运算符`的优先级低于`算数运算符`
- **不能连续使用**：`5 > a > 1` 会报错，要写 `a > 1 && a < 5`
- 不同数值类型之间可以随意进行条件运算符
- `bool` 和 `string` 只能用 `==` 和 `!=` 比较
- `char` 可以和数值比较（比的是 ASCII 码）

## 逻辑运算符

>没有其他运算符的情况下，逻辑运算也是从左往右依次计算

| 符号   | 名称    | 规则        |
| ---- | ----- | --------- |
| &&   | 与（并且） | 有假则假，同真为真 |
| \|\| | 或（或者） | 有真则真，同假为假 |
| !    | 非（取反） | 真变假，假变真   |
|      |       |           |

```csharp
bool inRange  = i > 1 && i < 5;   // i 在 1~5 之间
bool outRange = i < 1 || i > 5;   // i 不在 1~5 之间
```

**优先级**：`!` > `&&` > `||`，均低于条件运算符

### 短路规则

```csharp
int i = 1;
bool b = i > 0 || ++i >= 1;
// 左边已是 true，右边 ++i 不会执行，i 依然等于 1
```

## 优先级

```
高 → 低：
乘除取余 * / %  >  加减 + -  >  条件运算符  >  逻辑运算符
逻辑运算符中 ! > && > ||
括号最高，多层括号从里往外算
```

## 位运算符

> 把数值转成二进制后按位计算，了解即可。

| 符号       | 名称  | 规则                               |
| -------- | --- | -------------------------------- |
| `A & B`  | 位与  | 连接两个数值进行位计算，将数值转化为2进制，有0则0，同1则1  |
| `A \| B` | 位或  | 有1则1，同0则0                        |
| `A ^ B`  | 异或  | 相同为0，不同为1                        |
| `~A`     | 位取反 | 写在数值前面，将数值转换为2进制，再进行对位运算，0变1，1变0 |
| `A << n` | 左移  | 左移几位，右边加几个0                      |
| `A >> n` | 右移  | 右移几位，右边去掉几个数                     |
|          |     |                                  |

^0bbea8

```csharp
//位与计算
001 & 101 
 ↓
001
101
 ↓
001（二进制）→ 1（十进制）

int result = 1 & 5;   // 001 & 101 = 001 → 1

//取反计算
//需要带上变量类型的所有位数
int i = 1;
~ i
↓  
0000 0000 0000 0000 0000 0000 0000 0001
1111 1111 1111 1111 1111 1111 1111 1110
```

## 三目运算符

> 简单逻辑判断的简写，替代 if-else。

```csharp
// 语法：条件 ? 为真的值 : 为假的值
int max = a > b ? a : b;
string label = age >= 18 ? "成年" : "未成年";
```

**注意：**
- `?` 后两侧返回值类型必须一致
- 必须有变量存储结果
