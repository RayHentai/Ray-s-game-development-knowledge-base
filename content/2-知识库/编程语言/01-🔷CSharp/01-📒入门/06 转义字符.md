## 什么是转义字符

在字符串中，`\` 加上特定字母，可以表示无法直接输入的特殊字符。

---

## 常用转义字符

| 转义字符 | 含义 | 使用频率 |
|----------|------|----------|
| `\'` | 单引号 | 常用 |
| `\"` | 双引号 | 常用 |
| `\\` | 反斜杠本身 | 常用 |
| `\n` | 换行 | 常用 |
| `\t` | 制表符（Tab） | 不常用 |
| `\b` | 退格（覆盖前一个字符） | 不常用 |
| `\0` | 空字符 | 不常用 |
| `\a` | 警报音 | 不常用 |

```csharp
Console.WriteLine("他说：\"你好\"");    // 输出：他说："你好"
Console.WriteLine("第一行\n第二行");   // 换行输出
Console.WriteLine("C:\\Users\\Name"); // 输出：C:\Users\Name
```

---

## 取消转义字符：@ 前缀

```csharp
// ❌ 普通写法
string path = "C:\\Users\\Name\\file.txt";

// ✅ @ 写法
string path = @"C:\Users\Name\file.txt";
```

## 练习

> 1. 请写出至少5种转义字符

```
\'  \"  \\  \n  \t  \b  \a  \0
```

> 2. 请用至少两种方式在控制台打印出多行文字

我是小明
我今年18
我的爱好是制作游戏
我要好好学习天天向上

```csharp
// 方法1：多个 WriteLine
Console.WriteLine("我是小明");
Console.WriteLine("我今年18");
Console.WriteLine("我的爱好是制作游戏");
Console.WriteLine("我要好好学习，天天向上");

// 方法2：\n 换行符
Console.WriteLine("我是小明\n我今年18\n我的爱好是制作游戏\n我要好好学习，天天向上");

// 方法3：Write + 空 WriteLine
Console.Write("我是小明"); Console.WriteLine();
Console.Write("我今年18"); Console.WriteLine();
Console.Write("我的爱好是制作游戏"); Console.WriteLine();
Console.Write("我要好好学习，天天向上");
```
