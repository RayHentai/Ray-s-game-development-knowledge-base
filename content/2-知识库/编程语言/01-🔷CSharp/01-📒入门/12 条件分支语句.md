## if 语句

>让顺序执行的代码产生分支

```csharp
// 形式一
if (bool类型值) { }

// 形式二
if (bool类型值) { } else { }

// 形式三：多条分支，从上往下匹配，满足第一个就走那条
if (条件1) { }
else if (条件2) { }
else if (条件3) { }
else { }
```

**注意：**
- `if` 后面不需要写分号
- `if`语句可以嵌套使用，`else` 可以省略
- 条件判断**从上到下**，满足第一个后，后面的分支不再执行
- 去掉 `else` 每个 `if` 分支都会独立判断一次（性能稍低）

## switch 语句

> 判断一个变量等于哪个固定值，配合枚举使用效果最好。

```csharp
switch (变量)
{
    case 常量1: 
	    /* 满足时执行 */ 
	    break;
    case 常量2: 
	    /* 满足时执行 */ 
	    break;
    default:    
	    /* 都不满足时执行 */ 
	    break;
}
```

### 贯穿（多个 case 做同一件事）
```csharp
case 1:
case 2:
case 3:
    Console.WriteLine("工作日");
    break;
```

**注意：**
- `case`和`break`之间可以写n句语句，也可以嵌套其他语句（`if`、`while`等）
- `switch()` 里必须是**常量**（固定值），可以输入设定好的常量变量，不能写范围
- `switch`只判断，是否等于某一个固定值
- `case`、`break`可以省略
- 条件判断从上到下执行，满足了第一个之后，后面的都不会执行
- 一般配合**枚举**使用

- 学生评价很充实，则评定为B级，不加绩效工资

	    break;

	    salary -= 300; 
	    break;

	    break;
