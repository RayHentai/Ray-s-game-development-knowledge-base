## 成员变量的使用

**规则：**
- 申明在类语句块中，用来描述对象的特征
- 可以是任意变量类型，数量没有限制
- 是否赋值根据需求决定

**成员变量的初始值**
- 值类型（int/float 等）默认都是 **0**
- bool 类型默认 **false**
- 引用类型（string/数组/类）默认 **null**

**注意：**
- 成员变量可以与类名同名
- 如果在类中声明一个和自己相同类型的成员变量时，**不能对它进行实例化**，会报错（内存溢出）

```csharp
// 先实例化，再通过对象名.成员变量名访问
Player p = new Player();
p.name = "张三";  // 访问并赋值
Console.WriteLine(p.name);  // 读取
```

## 访问修饰符

| 修饰符 | 含义 | 使用场景 |
|--------|------|---------|
| `public` | 公共的，内部和外部都能访问使用 | 需要外部访问的成员 |
| `private` | 私有的，只能在内部访问使用（**不写默认为 private**） | 不希望外部直接修改的数据 |
| `protected` | 保护的，内部和子类才能访问使用 | 需要子类访问但不对外公开的成员 |

## 练习

> 1. 3P 是什么？


3P 指的是三种访问修饰符：
- public    — 公共的，内部和外部都能访问使用
- private   — 私有的，只能在内部访问使用（不写默认为 private）
- protected — 保护的，内部和子类才能访问使用

> 4. 请问 p.age 为多少？

```csharp
Person p  = new Person();
p.age = 10;
Person p2 = new Person();
p2.age = 20;
Console.WriteLine(p.age);
// 答：10（p 和 p2 是两个独立对象，p2.age 不影响 p.age）
```

> 5. 请问 p.age 为多少？

```csharp
Person p  = new Person();
p.age = 10;
Person p2 = p;
p2.age = 20;
Console.WriteLine(p.age);
// 答：20（p2 = p 让两者指向同一对象，修改 p2.age 就是修改 p.age）
```

> 6. 请问 s.age 为多少？

```csharp
Student s = new Student();
s.age = 10;
int age = s.age;
age = 20;
Console.WriteLine(s.age);
// 答：10（int 是值类型，age = s.age 是值拷贝，修改 age 不影响 s.age）
```

> 7. 请问 s.deskmate.age 是多少？

```csharp
Student s = new Student();
s.deskmate = new Student();
s.deskmate.age = 10;
Student s2 = s.deskmate;
s2.age = 20;
Console.WriteLine(s.deskmate.age);
// 答：20（s2 = s.deskmate 是引用类型赋值，s2 和 s.deskmate 指向同一对象）
```
