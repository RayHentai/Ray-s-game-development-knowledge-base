## 核心理解

> `object` 是所有类型的基类，它是一个类（引用类型）。
 
---

## 装箱拆箱

**用object存值类型（装箱）**
- 把值类型用引用类型存储
- 栈内存会迁移到堆内存中

**把object存储的值类型转为值对象（拆箱）**
- 把引用类型存储的值类型取出来放入值对象中    
- 堆内存会迁移到栈内存中

```csharp
// 装箱：用 object 存值类型
// 把值类型用引用类型存储，栈内存会迁移到堆内存中
object obj = 1; // 装箱：int → object

// 拆箱：把 object 存储的值类型取出来放入值对象
// 堆内存会迁移到栈内存中
int i = (int)obj; // 拆箱：object → int

// 引用类型用 as
object obj2 = "1234";
string str = obj2 as string;

//实操 
```

**优缺点：**
- 优点：不确定类型时可以用 object 存任何类型，方便参数的存储和传递
- 缺点：存在内存迁移（栈↔堆），增加性能消耗

---

## 什么时候用

> 适用场景，帮助建立使用直觉

- 可以利用里氏替换原则，用 object 容器装所有对象。
- 可以用来表示不确定类型，作为函数参数类型。

---

## 练习

> 1. 请口头描述什么是装箱拆箱


装箱：把值类型用引用类型（object）存储，值从栈内存迁移到堆内存
拆箱：把 object 中存储的值类型取出来，从堆内存迁移回栈内存


> 2. 请用代码描述装箱拆箱

```csharp
// 装箱
object obj = 1;
object obj2 = "1234";

// 拆箱
int i = (int)obj; // 值类型用括号强转拆箱
string str = obj2 as string; // 引用类型用 as 拆箱
```
