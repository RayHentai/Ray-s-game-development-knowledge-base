## 类和对象的区别

> 类的声明和（类）对象的声明是两个概念。

> 类的声明类似枚举和结构体的声明，相当于声明了一个自定义变量类型。

**类（Class）：**
- 具有相同特征、行为的一类事物的抽象，对象的模板
- 可以通过类创建出对象
- 关键词是 `class`，声明位置在 `namespace` 中
- 类前面可以加访问修饰符，用`帕斯卡命名`
- 类不能在同一个 namespace 块中重名
- 类是一个自定义的变量类型

**（类）对象：**
- 对象是类创建出来的，相当于声明一个指定类的变量
- 类创建对象的过程一般称为**实例化对象**
- 类对象都是**引用类型**

## 实例化对象

```csharp
// 前两种声明方式在内存分配上等同
Player p1 = new Player(); // 方式1：声明并实例化
Player p2;
p2 = new Player(); // 方式2：先声明，后实例化

Player p3 = null; // null 指的是栈上是一个空值（不指向任何堆内存）

p3 = new Player();
p4 = new Player();
// 两个新生成的实例化对象，特征、行为等信息都是独有的，彼此没有关系，并不是等同的
```

> 2. A 目前等于多少？

```csharp
GameObject A = new GameObject();
GameObject B = A;
B = null;
// A 仍然指向原来的对象，不为 null
// B = null 只是让 B 不再指向那个对象，不影响 A
```

> 3. A 和 B 有什么关系？

```csharp
GameObject A = new GameObject();
GameObject B = A;
B = new GameObject();
// 答：没有任何关系
// B = new GameObject() 让 B 指向了一个新对象
// A 依然指向原来的对象，两者互不影响
```
