# ⚖️ 28 抽象类与接口的区别

**所属模块**：[[00 CSharp 核心阶段总览]]
**关联**：[[20 抽象类与抽象方法]] | [[21 接口]]
**查阅次数**：0

---

## 相同点

- 都可以被继承
- 都不能直接实例化
- 都可以包含方法声明
- 子类必须实现未实现的方法
- 都遵循里氏替换原则

---

## 不同点

| 对比项 | 抽象类 | 接口 |
|--------|--------|------|
| 构造函数 | ✅ 可以有 | ❌ 不能有 |
| 继承数量 | 只能被**单一继承** | 可以被**继承多个** |
| 成员变量 | ✅ 可以有 | ❌ 不能有 |
| 方法实现 | 可以有普通方法、虚方法、抽象方法、静态方法 | 只能申明没有实现的抽象方法 |
| 访问修饰符 | 方法可以使用 | 建议不写，默认 public |

---

## 如何选择

| 场景                    | 选择      |
| --------------------- | ------- |
| **表示对象**（人、动物等一类事物）   | **抽象类** |
| **表示行为拓展**（能飞、能游泳等行为） | **接口**  |
| 不同对象拥有的共同行为           | **接口**  |

**举例：**
- 动物是一类对象 → 自然选择抽象类
- 飞翔是一个行为 → 自然选择接口
