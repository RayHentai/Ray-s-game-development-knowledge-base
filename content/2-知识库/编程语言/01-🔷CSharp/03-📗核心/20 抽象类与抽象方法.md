## 抽象类

**定义：** 被抽象关键字 `abstract` 修饰的类。

**规则：**
- **不能被实例化的类，但是遵循里氏替换原则**
- 可以包含抽象方法
- 继承抽象类必须重写其抽象方法
- 封装的所有知识点都可以在其中书写（普通方法、属性、构造函数等）

---

## 抽象方法（纯虚方法）

**定义：** 用 `abstract` 关键字修饰的方法，又称**纯虚方法**。

**规则：**
- 只能在抽象类中声明
- **没有方法体（没有大括号），有大括号会报错**
- **不能是私有的**
- **继承后必须实现，用 override 重写**

```csharp
abstract class Animal
{
    public abstract void Speak(); // 只有声明，没有 {}方法体

    // 抽象类中也可以有普通方法
    public void Breathe() 
    { 
	    Console.WriteLine("呼吸"); 
    }
}

class Dog : Animal
{
    public override void Speak() 
    { 
	    Console.WriteLine("汪汪"); 
    }  // 必须实现
}
```

---

## 什么时候用

> 适用场景，帮助建立使用直觉

- 不希望被实例化的对象，相对比较抽象的类（如"人"类）可以使用抽象类
- 父类中的行为不太需要被实现的，只希望子类去定义具体规则的，可以选择抽象类然后使用抽象方法

---

## 虚方法 vs 抽象方法

| | 虚方法 `virtual` | 抽象方法 `abstract` |
|-|---------|---------|
| 方法体 | 有（可以有默认实现） | **无（有大括号报错）** |
| 子类是否必须重写 | 可选 | **必须** |
| 可以在哪定义 | 普通类/抽象类 | **只能在抽象类中** |
| 都可以被子类无限重写 | ✅ | ✅ |

## 练习

> 1. 写一个动物抽象类，写三个子类：人叫、狗叫、猫叫

```csharp
abstract class Animal 
{ 
	public abstract void Speak(); 
}

class Person : Animal 
{ 
	public override void Speak() 
	{ 
		Console.WriteLine("人说话"); 
	} 
}
class Dog : Animal 
{ 
	public override void Speak() 
	{ 
		Console.WriteLine("狗叫汪汪"); 
	} 
}
class Cat : Animal 
{ 
	public override void Speak() 
	{ 
		Console.WriteLine("猫叫喵喵"); 
	} 
}
```

> 2. 创建一个图形抽象类，有求面积和周长两个抽象方法；创建矩形类、正方形类、圆形类继承图形类；实例化矩形、正方形、圆形对象求面积和周长

```csharp
abstract class TuXing
{
    public abstract float MianJi();
    public abstract float ZhouChang();
}

class JvXing : TuXing
{
    private float w, h;
    public JvXing(int w, int h)
    { 
	    this.w = w; 
	    this.h = h; 
    }
    public override float MianJi() 
    { 
	    return w * h; 
    }
    public override float ZhouChang() 
    { 
	    return (w + h) * 2; 
    }
}

class ZhengFangXing : TuXing
{
    private float w;
    public ZhengFangXing(int w)
    { 
	    this.w = w; 
    }
    public override float MianJi()    
    { 
	    return w * w; 
    }
    public override float ZhouChang() 
    { 
	    return w * 4; 
    }
}

class YuanXing : TuXing
{
    private float r;
    public YuanXing(int r)
    { 
	    this.r = r; 
    }
    public override float MianJi()    
    { 
	    return 3.14f * r * r; 
    }
    public override float ZhouChang() 
    { 
	    return 2 * 3.14f * r; 
    }
}

JvXing j = new JvXing(3, 4);             Console.WriteLine(j.MianJi() + " " + j.ZhouChang());
ZhengFangXing z = new ZhengFangXing(3);  Console.WriteLine(z.MianJi() + " " + z.ZhouChang());
YuanXing y = new YuanXing(2);            Console.WriteLine(y.MianJi() + " " + y.ZhouChang());
```
