## 定义

> 用于组织和重用代码。就像是一个工具包，类就像是一件一件的工具，都是申明在命名空间中的。

---

## 使用

```csharp
// 方式1：using 引用（常用）
using System;
using MyGame;
Player p = new Player();

// 方式2：指明出处
MyGame.Player p = new MyGame.Player();
```

---

## 语法

```csharp
namespace 命名空间名
{
    class 类A { }
    class 类B { }
}
```

---

## 访问修饰符

| 修饰符 | 含义 |
|--------|------|
| `public` | 命名空间的类默认访问权限 |
| `internal` | 只能在该程序集中使用 |
| `abstract` | 抽象 |
| `sealed` | 密封 |
| `partial` | 分部 |

---

## 注意事项

- 命名空间可以同名写多个（是一个整体），但是里面的类不能重名
- 不同命名空间中相互使用，需要引用命名空间或指明出处
- 不同的命名空间允许有同名类，如果要使用两个命名空间中的同名类，需要用指明出处的方式使用
- **命名空间可以包裹命名空间**

```csharp
namespace MyGame
{
    namespace UI        // 命名空间嵌套
    {
        class Button { }
    }
}
// 使用：MyGame.UI.Button
```
