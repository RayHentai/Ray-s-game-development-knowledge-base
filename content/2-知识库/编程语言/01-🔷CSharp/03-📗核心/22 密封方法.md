## 定义

> 用密封关键字 `sealed` 修饰的重写函数，写在 `override` 之前。

- 作用：让虚方法或抽象方法之后不能再被重写。

**注意：**
- `sealed` 只能用在 `override` 方法上，不能修饰普通方法

```csharp
class Animal
{
    public virtual void Speak() 
    { 
	    Console.WriteLine("动物叫"); 
    }
}

class Dog : Animal
{
    public sealed override void Speak() 
    { 
	    Console.WriteLine("汪汪"); 
    }
    // sealed 修饰的 override：Dog 之后的子类不能再重写 Speak
}

class GoldenRetriever : Dog
{
    public override void Speak() { } // ❌ 报错，已被密封
}
```

---

## 什么时候用

> 适用场景，帮助建立使用直觉

- 不希望某个行为在更底层的子类中被改变
