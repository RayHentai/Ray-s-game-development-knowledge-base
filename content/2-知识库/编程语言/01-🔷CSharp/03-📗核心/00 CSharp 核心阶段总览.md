## ✅ 阶段自检清单

**封装**
- [ ] 能写一个完整的类（成员变量/方法/属性/构造函数）
- [ ] 理解 public/private/protected 的区别
- [ ] 会用属性保护成员变量，加入逻辑判断
- [ ] 理解静态成员与程序同生共死的特点
- [ ] 会写拓展方法和运算符重载

**继承**
- [ ] 能说清楚继承的规则（单根性/传递性）
- [ ] 理解里氏替换原则，会用父类容器装子类
- [ ] 会用 is/as 做类型判断和转换
- [ ] 知道继承中构造函数的执行顺序，会用 base

**多态**
- [ ] 能说清楚 virtual/override 的作用
- [ ] 理解抽象类和抽象方法的使用场景
- [ ] 理解接口的本质（行为的抽象规范）
- [ ] 能说清楚抽象类和接口的区别

**综合**
- [ ] 独立完成贪吃蛇项目
