## 定义

> 让自定义类和结构体可以使用运算符。关键字：`operator`

**作用：** 让自定义类和结构体可以运算。

---

## 语法

**规则：**
- 一定是一个**公共的静态方法**
- 返回值写在 `operator` 前
- 逻辑处理自定义
- 注意参数列表的变量顺序，计算时不可颠倒
- **条件运算符需要成对实现**（大于/小于，等于/不等于）
- 一个符号可以有多个重载
- 不能使用 ref 和 out

```csharp
// 语法：public static 返回类型 operator 运算符(参数列表)
public static Vector3 operator +(Vector3 a, Vector3 b)
{
    return new Vector3(a.x + b.x, a.y + b.y, a.z + b.z);
}
```

## 可重载与不可重载

**可重载：**
- 算数运算符（全部）
- 逻辑非 `!`
- 位运算符（全部）
- 条件运算符（全部，返回值一般是 bool，也可以是其他）

**不可重载：**
- 逻辑与 `||`、逻辑或 `&&`
- 索引符 `[]`
- 强转运算符 `()`
- 特殊运算符：点 `.`、三目运算符 `?:`、赋值符号 `=`

## 练习

> 1. 定义一个位置结构体或类，为其重载判断是否相等的运算符：`(x1,y1)==(x2,y2)`=> 两个值同时相等才为 true

```csharp
class Test
{
    public int x;
    public int y;
	
    public static bool operator ==(Test t, Test t1)
    {
        if (t.x == t1.x && t.y == t1.y) 
	        return true;
        return false;
    }
	
    public static bool operator !=(Test t, Test t1)
    {
        if (t.x == t1.x && t.y == t1.y) 
	        return false;
        return true;
    }
}
```

> 2. 定义一个 Vector3 类（x,y,z），通过重载运算符实现 +、-、*(乘以数字) 运算

```csharp
class Vector3
{
    public int x, y, z;

    public Vector3(int x, int y, int z)
    {
        this.x = x; this.y = y; this.z = z;
    }

    public static Vector3 operator +(Vector3 v, Vector3 v1)
    {
        return new Vector3(v.x += v1.x, v.y += v1.y, v.z += v1.z);
    }

    public static Vector3 operator -(Vector3 v, Vector3 v1)
    {
        return new Vector3(v.x -= v1.x, v.y -= v1.y, v.z -= v1.z);
    }

    public static Vector3 operator *(Vector3 v, int num)
    {
        return new Vector3(v.x *= num, v.y *= num, v.z *= num);
    }
}
```
