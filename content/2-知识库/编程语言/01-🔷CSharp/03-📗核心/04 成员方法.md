## 声明与使用

**规则**
- 申明在类语句块中，描述对象行为
- 规则和申明函数相同，返回值参数不做限制，方法数量不做限制

**注意：**
- **不要加 static 关键字**（static 函数中不能直接访问非 static 成员，否则会报错）
- 必须实例化出对象，再通过对象使用
- 受到访问修饰符影响，private 方法只能在类内部调用

```csharp
class Person
{
    public string name;

    // 声明：不加 static
    public void Speak(string str)
    {
        Console.WriteLine("{0}说了{1}", name, str);
    }
}

// 使用：先实例化，再调用
Person p = new Person();
p.name = "张三";
p.Speak("你好");  // 通过对象调用
```
