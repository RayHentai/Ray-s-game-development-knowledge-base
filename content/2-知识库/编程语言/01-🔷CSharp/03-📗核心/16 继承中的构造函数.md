## 语法

**执行顺序：**
- 当声明一个子类对象时，**先执行父类构造函数，再执行子类构造函数**。
- 从老祖宗开始，依次一代一代向下执行。

```csharp
class GrandFather
{
	public GrandFather()
	{
		Console.Write("GrandFather的构造函数")
	}
}
class Father : GrandFather
{
	public GrandFather()
	{
		Console.Write("Father的构造函数")
	}
}
class Son : Father
{
	public GrandFather()
	{
		Console.Write("Son的构造函数")
	}
}
Son son = new Son();
//此时会依次打印信息
```

**注意：父类的无参构造很重要**
- **子类实例化时，默认自动调用的是父类的无参构造**
- **如果父类无参构造被有参构造顶掉，会报错**

**解决：**
- 可以通过`base`或者`this`解决
- 子类可以通过`base`关键字代表父类，调用父类构造

**`base`关键字：**
- `base`代表父类的某一个构造函数
- 在子类构造函数后面加 : `base`(父类构造函数传入值)

```csharp
class Father
{
	//public Father() { }
	public Father(int i) { }
}
class Son : Father { }//此时会报错

class Son : Father 
{
	//调用父类的有参构造
	public Son(int i) : base(i) { } //解决
	//调用自己的有参构造 间接调用父类的有参构造
	public Son(int i, string str) : this(i) { } //解决
}
```

## 练习

> 1. 有一个打工人基类，有工种、工作内容两个特征，一个工作方法；程序员、策划、美术分别继承打工人，请用继承中的构造函数知识点，实例化3个对象

```csharp
class WorkMan
{
    public string type;
    public string workNeiRong;

    public WorkMan(string type, string workNeiRong)
    {
        this.type = type;
        this.workNeiRong = workNeiRong;
    }

    public void WorkFangfa() 
    { 
	    Console.WriteLine(type + workNeiRong); 
    }
}

class ChengXuYuan : WorkMan
{
    public ChengXuYuan() : base("程序员", "写代码") { }
}

class CeHua : WorkMan
{
    public CeHua() : base("策划", "写案子") { }
}

class Artist : WorkMan
{
    public Artist() : base("美术", "画画") { }
}

ChengXuYuan c = new ChengXuYuan(); 
c.WorkFangfa();
CeHua ch = new CeHua();       
ch.WorkFangfa();
Artist a = new Artist();      
a.WorkFangfa();
```
