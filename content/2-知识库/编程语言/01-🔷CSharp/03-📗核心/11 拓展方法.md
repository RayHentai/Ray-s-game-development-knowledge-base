## 定义

> 为现有的**非静态**变量类型添加新方法。

**作用：**
- 提升程序拓展性
- 不需要在对象中重新写方法
- 不需要继承来添加方法
- 为别人封装的类型写额外方法

---

## 语法

**规则：**
- 一定是写在**静态类**中
- 一定是个**静态函数**
- 第一个参数为拓展目标，且用 **`this`** 修饰（第一个参数用 this 修饰）

**注意：**
- 自定义添加的拓展方法名如果与类中的方法名重名，**会使用类中的方法**（优先级低于原有方法）

```csharp
// 语法：访问修饰符 static 返回值 函数名(this 拓展类名 参数名, 其他参数...)
static class ExtensionMethods
{
	//为int拓展了一个平方的成员方法
	//成员方法是 需要 实例化对象后 才能使用的
	//i 代表使用该方法的 实例化对象
	//括号内可以继续添加参数，后面的参数就代表传入的参数，和函数一样
    public static int Square(this int i)
    {
        return i * i;
    }
}

int n = 5;
Console.WriteLine(n.Square()); // 25，像调用成员方法一样
```

## 练习

> 1. 为整形拓展一个求平方的方法

```csharp
static class Tool
{
    public static int PingFang(this int i)
    {
        return i * i;
    }
}
```
