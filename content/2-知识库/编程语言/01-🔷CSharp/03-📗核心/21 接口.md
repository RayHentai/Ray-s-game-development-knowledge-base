## 核心理解

> **接口是行为的抽象规范，是抽象行为的基类。**
> 是一种自定义类型，关键字 `interface`。

---
## 语法
### 接口的声明

**规则：**
- 不包含成员变量
- 只包含方法、属性、索引器、事件
- 成员不能被实现（没有方法体）
- 成员可以不用写访问修饰符，不能是私有的
- 接口不能继承类，但是可以继承另一个接口

**命名规范：** 
- 帕斯卡前面加个 `I`（I开头），如 `IFlyable`、`ISwimmable`

```csharp
interface IFlyable
{
	//方法
    void Fly(); // 只有声明，不写访问修饰符（默认 public）
    //属性
    string Name 
    {
	    get;
	    set;
    }
    
    //索引器
    int this[int index]
    
    //委托
    event Action testEvent;
}

interface ISwimmable
{
    void Swim();
}
```

---

### 接口的继承

>接口遵循里氏替换原则

---

#### 类继承接口

**规则：**
- 接口是用来继承的，类可以继承多个接口
- 继承接口后，可以将接口中的行为修饰为抽象行为，供子类去实现，因为是抽象行为，所以子类必须实现，并且必须是public
- 实现的接口函数，可以再加virtual，在子类中重写

```csharp
class Animal : IFlyable,ISwimmable
{
	//必须实现
	public void Fly()
	{
		Console.Write("飞")
	}
	//可以是虚函数
	public virtual void Swim();
}

class Duck : Animal
{
	//必须实现
	public override void Swim()
	{
		Console.Write("鸭子游泳")
	}
}
```

---

#### 接口继承接口

```csharp
// 接口继承接口不需要实现
// 待类继承接口后，类自己去实现所有内容
interface IAnimal 
{ 
	void Breathe(); 
}
interface IDog : IAnimal 
{ 
	void Bark(); 
}

// 接口容器可以装实现了该接口的任何对象（装载各种毫无关系但有相同行为的对象）

IFlyable[] flyers = new IFlyable[3];
flyers[0] = new Duck();
flyers[1] = new Bird();
flyers[2] = new Airplane();
```

#### 显示实现接口（处理同名冲突）

> 当一个类继承两个接口，且接口中存在同名方法时使用。
> **显示实现接口时，不能写访问修饰符。**

```csharp
interface IA 
{ 
	void Show(); 
}
interface IB 
{ 
	void Show(); 
}

class Test : IA, IB
{
    // 接口名.行为名
    void IA.Show() 
    { 
	    Console.WriteLine("IA.Show"); 
    }
    void IB.Show() 
    { 
	    Console.WriteLine("IB.Show"); 
    }
}

Test t = new Test();
(t as IA).Show();  // IA.Show
(t as IB).Show();  // IB.Show
```

## 继承类 vs 继承接口

- 继承类是对象间的继承，包括特征、行为等
- 继承接口是行为间的继承，继承接口的行为规范，按照规范去实现内容

---


## 什么时候用

> 适用场景，帮助建立使用直觉

- 因为接口遵循里氏替换原则，所以可以用接口容器装对象，那么就可以实现，装载各种毫无关系但是却有相同行为的对象
- 用一个接口对象装不确定类型的泛型对象
	-  [[05 项目-PlayerPrefs 数据管理器#^6e2d0c]] 利用IList 和 IDictionary 装List<> 和 Dictionary<>

## 练习

> 1. 人、汽车、房子都需要登记，人需要到派出所登记，汽车需要去车管所登记，房子需要去房管局登记，使用接口实现登记方法

```csharp
interface IDengJi 
{ 
	void DengJi(); 
}

class Person : IDengJi 
{ 
	public void DengJi() 
	{ 
		Console.WriteLine("派出所登记成功"); 
	} 
}
class House : IDengJi 
{ 
	public void DengJi() 
	{ 
		Console.WriteLine("房管局登记成功"); 
	} 
}
class Car : IDengJi 
{ 
	public void DengJi() 
	{ 
		Console.WriteLine("车管所登记成功"); 
	} 
}
```

> 3. 多态来模拟移动硬盘、U盘、MP3 插到电脑上传输数据，电脑提供了一个 USB 接口，请实现电脑的传输数据的功能

```csharp
interface IUSB 
{ 
	void ChuanShuShuJv(); 
}

class CunChuSheBei : IUSB
{
    public string name;
    public CunChuSheBei(string name) 
    { 
	    this.name = name; 
    }
    public void ChuanShuShuJv() 
    { 
	    Console.WriteLine(name + "传输数据"); 
    }
}

class MP3 : IUSB
{
    public void ChuanShuShuJv() 
    { 
	    Console.WriteLine("MP3传输数据"); 
    }
}

class Computer 
{ 
	public IUSB usb; 
}

CunChuSheBei yp = new CunChuSheBei("移动硬盘");
CunChuSheBei up = new CunChuSheBei("U盘");
MP3 mp = new MP3();
Computer c = new Computer();
c.usb = yp; 
c.usb.ChuanShuShuJv();
c.usb = up; 
c.usb.ChuanShuShuJv();
c.usb = mp; 
c.usb.ChuanShuShuJv();
```
