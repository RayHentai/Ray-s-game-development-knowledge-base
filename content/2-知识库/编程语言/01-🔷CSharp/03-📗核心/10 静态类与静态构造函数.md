## 静态类

**定义：** 用 `static` 修饰的类（`Console` 就是一个静态类）。
- 只能包含静态成员
- 不能被实例化

**作用：**
- 将常用的静态成员写在静态类中，方便使用
- 静态类不能被实例化，更能体现工具类的唯一性

```csharp
static class MathHelper
{
    public static float PI = 3.14159f;
    public static float CircleArea(float r) 
    { 
	    return PI * r * r; 
    }
}

// 直接用类名调用，不能 new
Console.WriteLine(MathHelper.CircleArea(2));
```

## 静态构造函数

**定义：** 在构造函数前面加上 `static` 修饰。

**规则：**
- 静态类和普通类都可以有
- 不能使用访问修饰符
- 不能有参数
- **只会自动调用一次**，在第一次使用该类时触发

**作用：** 在静态构造函数中初始化静态变量。

```csharp
class GameConfig
{
    public static int MaxPlayers;
    public static string Version;

    static GameConfig()  // 静态构造函数：无修饰符、无参数、只调用一次
    {
        MaxPlayers = 4;
        Version    = "1.0.0";
    }
}
```

**注意：** 静态构造函数和构造函数不是同一个类型，不会重载。

## 练习

> 1. 写一个用于数学计算的静态类，该类中提供计算圆面积、圆周长、矩形面积、矩形周长、取绝对值等方法

```csharp
static class JiSuan
{
    public static float pi = 3.14159f;

    public static float YuanMianJi(int r)       
    { 
	    return r * pi * pi; 
    }
    public static float YuanZhouChang(int r)    
    { 
	    return 2 * pi * r; 
    }
    public static int JvXingMianJi(int w, int h) 
    { 
	    return w * h; 
    }
    public static int JvXingZhouChang(int w, int h) 
    { 
	    return (w + h) * 2; 
    }
    public static int JueDuiZhi(int i)          
    { 
	    return i >= 0 ? i : -i; 
    }
}

Console.WriteLine(JiSuan.YuanMianJi(2));
Console.WriteLine(JiSuan.YuanZhouChang(2));
Console.WriteLine(JiSuan.JvXingMianJi(4, 5));
Console.WriteLine(JiSuan.JvXingZhouChang(4, 5));
Console.WriteLine(JiSuan.JueDuiZhi(-8));
Console.WriteLine(JiSuan.JueDuiZhi(6));
```
