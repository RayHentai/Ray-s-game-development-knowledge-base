## 核心理解

> 复用封装对象的代码，儿子继承父亲，复用现成代码。

- 一个类 A 继承一个类 B，类 A 将会继承类 B 所有成员
- A 类将会拥有 B 类的所有特征和行为
- 被继承的类称为**父类、基类、超类**；继承的类称为**子类、派生类**
- 子类可以有自己的特征和行为

---

## 语法

**注意：**

- C# 中允许子类和父类有同名成员，但是**不建议使用**。

```csharp
class Father {  }
class Son : Father
{
    // 子类自己的成员
}
```

---

## 两个特点

| 特点 | 说明 |
|------|------|
| **单根性** | 子类只能有**一个**父类（C# 不支持多继承） |
| **传递性** | 子类可以间接继承父类的父类（爷爷→父→子） |

## 练习

> 1. 写一个人类，人类中有姓名、年龄属性，有说话行为；战士类继承人类，有攻击行为

```csharp
class Person
{
    public string Name { get; set; }
    public int    Age  { get; set; }

    public void Speak()
    {
        Console.WriteLine("我叫{0},我今年{1}岁了", Name, Age);
    }
}

class Warrior : Person
{
    public void GongJi(Warrior otherWarrior) { }
}
```
