## 核心理解

- 用于保护成员变量
- 为成员属性的获取和赋值添加逻辑处理
- 解决 3P 的局限性：属性可以让变量成员在外部只能获取不能修改，或只能修改不能获取

---

## 基本语法

**规则：**
- 命名使用`帕斯卡`

**注意：**
- 不能让 get 和 set 的访问权限都低于属性的权限，会报错
- get 和 set 可以只有一个

```csharp
class Player
{
    private int hp;   // 私有字段，外部不能直接访问

    // 属性：帕斯卡命名
    public int Hp
    {
        get 
        { 
	        return hp; 
        } // 获取时执行
        set
        {
            if (value < 0) 
	            hp = 0; // value 表示外部传入的值
            else           
	            hp = value;
        }
    }
}

Player p = new Player();
p.Hp = -10; // set 被调用，hp 被保护为 0
Console.WriteLine(p.Hp); // get 被调用，输出 0
```

## 属性访问修饰符

>get 和 set 可以加访问修饰符
>默认不加会使用属性声明时的访问权限

**注意：**
- 不能让 get 和 set 的访问权限都低于属性的权限，会报错

```csharp
public int Hp
{
    get 
    { 
	    return hp; 
    } // 使用属性声明时的访问权限（public）
    private set 
    { 
	    hp = value; 
    } // 只能在内部赋值
}
```

## 自动属性

```csharp
// 一般用于外部能得不能改的场景
public string Name 
{ 
	get; 
	set; 
} // 外部可读可写
public string Name 
{ 
	get; 
	private set; 
}  // 外部只读，内部可写
```

## 练习

> 1. 定义一个学生类，

- 有姓名、性别、年龄、CSharp成绩、Unity成绩五种属性
- 有打招呼方法和计算总分平均分方法

>使用属性完成：

- 年龄必须0~150
- 成绩必须0~100
- 性别只能男或女
- 实例化两个对象并测试

```csharp
class Student
{
    public int age;
    public int cSharp;
    public int unity;
    public string sex;

    public string Name 
    {
	    get; 
	    set; 
    }

    public string Sex
    {
        get 
        { 
	        return sex; 
        }
        set
        {
            if (value != "男" && value != "女") 
	            sex = "扶他";
            else                               
	            sex = value;
        }
    }

    public int Age
    {
        get 
        { 
	        return age; 
        }
        set
        {
            if (value < 0)   
	            age = 0;
            else if (value > 150) 
	            age = 150;
            else                  
	            age = value;
        }
    }

    public int CShare
    {
        get 
        { 
	        return cSharp; 
        }
        set
        {
            if (value < 0)   
	            cSharp = 0;
            else if (value > 100) 
	            cSharp = 100;
            else                  
	            cSharp = value;
        }
    }

    public int Unity
    {
        get 
        { 
	        return unity; 
        }
        set
        {
            if (value < 0)   
	            unity = 0;
            else if (value > 100) 
	            unity = 100;
            else                  
	            unity = value;
        }
    }

    public void SayHello()
    {
        Console.WriteLine("我叫{0}，今年{1}岁了，是{2}同学", Name, Age, Sex);
    }

    public void JiSuan()
    {
        int sum = CShare + Unity;
        float avg = sum / 2f;
        Console.WriteLine("总分是{0}，平均分是{1}", sum, avg);
    }
}

Student s = new Student();
s.Name = "张";
s.Sex = "女";
s.Age = 180;
s.CShare = 101;
s.Unity  = 99;
s.SayHello();
s.JiSuan();
```
