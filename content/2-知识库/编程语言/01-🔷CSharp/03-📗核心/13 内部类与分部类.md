## 内部类

**定义：** 一个类中再声明一个类，使用时要包裹者点出自己。

```csharp
class Outer
{
    public class Inner
    {
        public void Print() 
        { 
	        Console.WriteLine("我是内部类"); 
        }
    }
}

// 使用：外部类.内部类
Outer.Inner inner = new Outer.Inner();
inner.Print();
```

**作用：** 亲密关系的表现（内部类只属于外部类使用）。
**注意：** 访问修饰符作用很大。

---

## 分部类

**定义：** 把一个类分成几部分生成，关键字 `partial`。

```csharp
// 文件1
partial class Player { public string name; }

// 文件2
partial class Player { public void Attack() { } }
// 两个 partial 合并为一个完整的 Player 类
```

**作用：**
- 分布描述一个类
- 增加程序的拓展性

**规则：**
- 分部类可以写在多个脚本文件中
- 分部类的访问修饰符要一致
- 分部类中不能有重复成员

---

## 分部方法

**定义：** 将方法的声明和实现分离，关键字 `partial`。

**规则：**
- 不能加访问修饰符，默认私有
- 只能在分部类中声明
- 返回值只能是 void
- 可以有参数，但不能用 out 关键字
