## 定义

> 使用 `sealed` 密封关键字修饰的类，让类无法再被继承。

```csharp
sealed class FinalClass { }
// class Child : FinalClass { } // ❌ 报错，密封类不能被继承
```

---

## 什么时候用

> 适用场景，帮助建立使用直觉

- 在面向对象程序的设计中，密封类的主要作用就是不允许最底层子类被继承
- 可以保证程序的规范性、结构性、安全性

## 练习

> 1. 定义一个载具类，有速度、最大速度、可乘人数、司机和乘客等特征，有上车、下车、行驶、车祸等方法，用载具类声明一个对象，并将若干人装载上车

```csharp
class Parson  { }
class Driver : Parson { }
class ChengKe : Parson { }

class Car
{
    public int speed, maxSpeed, num;
    public Parson[] chengKe;

    public Car(int speed, int maxSpeed, int num)
    {
        this.speed = speed;
        this.maxSpeed = maxSpeed;
        this.num = 0;
        chengKe = new Parson[num];
    }

    public void ShangChe(Parson p)
    {
        if (num >= chengKe.Length) 
	        Console.WriteLine("满载"); 
        chengKe[num] = p;
        num++;
    }

    public void XiaChe(Parson p)
    {
        for (int i = 0; i < chengKe.Length; i++)
        {
            if (chengKe[i] == null) 
	            break;
            if (chengKe[i] == p)
            {
                for (int j = i; j < num - 1; j++) 
	                chengKe[j] = chengKe[j + 1];
                chengKe[--num] = null;
                break;
            }
        }
    }

    public void XingShi() { }
    public void CheHuo()  { }
}

Car car = new Car(10, 20, 20);
Driver d = new Driver();
car.ShangChe(d);
ChengKe ck = new ChengKe();
car.ShangChe(ck);
ChengKe ck2 = new ChengKe();
car.ShangChe(ck2);
ChengKe ck3 = new ChengKe();
car.ShangChe(ck3);
car.XiaChe(ck2);
```
