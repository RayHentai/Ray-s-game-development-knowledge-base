## 静态方法

>Object. 可以省略，因为类是继承了object的

**Equals：** 判断两个对象是否相等。
- 最终判断权交由左侧对象的 Equals 方法决定
- 不管值类型引用类型，都会按照左侧对象 Equals 方法的规则来进行比较
- 引用类型是判断两者是不是指向同一个内存地址

```csharp
//值类型
Object.Equals(1, 1);//true

class Test { }
Test t = new Test();
Test t2 = t;
//引用类型
Object.Equals(t, t2);//true
```

**ReferenceEquals：** 比较两个对象是否是相同的引用。
- 用来比较引用类型的对象
- **值对象返回值始终是 false**

```csharp
//值类型
Object.ReferenceEquals(1, 1);//false

class Test { }
Test t = new Test();
Test t2 = t;
//引用类型
Object.ReferenceEquals(t, t2);//true
```

## 成员方法

**GetType：** 获取对象运行时的类型 Type，得到一个 Type 对象。

```csharp
class Test { }
Test t = new Test();
Type type = t.GetType();// 命名空间名.Test
```

**MemberwiseClone：** 获取对象的浅拷贝对象（返回一个新的对象）。
- 新变量的引用变量会和老变量中一致（引用类型字段指向同一对象）
- `MemberwiseClone`是保护类型，只能在类内部调用

```csharp
class Test2
{
	public int i = 2;
}
class Test 
{
	public int i = 1;
	public Test2 t2 = new Test2();
	public Test Clone()
	{
		return MemberwiseClone() as Test;
	}
}
Test t = new Test();
Test t2 = t.Clone();
//克隆对象后
t.i //1
t.t2.i //2
t2.1 //1
t2.t2.i //2

t2.i = 20;
t2.i = 21;
//改变克隆体后
t.i //1 老对象的值类型没有变
t.t2.i //21 老对象的引用类型对象跟着变了
t2.1 //20
t2.t2.i //21
```

## 虚方法（可以重写）

**Equals（虚方法版）：**
- 默认实现还是比较两者是否为同一个引用，相当于 ReferenceEquals
- 微软在所有值类型的基类 System.ValueType 中重写了该方法，用于比较值相等
- **我们可以重写该方法，定义自己的比较相等的规则**

```csharp
class Test
{
	public override bool Equals(object obj) 
	{ 
		//自己定义规则  
		return base.Equals();
	}
}
```

**GetHashCode：** 获取对象的哈希码。
- 通过算法算出的表示对象的唯一编码，不同对象的哈希码可能一样
- **我们可以通过重写该函数来自己定义对象的哈希码算法**，正常情况下很少用

```csharp
class Test
{
	public override int GetHashCode() 
	{ 
		//自己定义规则  
		return base.GetHashCode();
	}
}
```

**ToString：** 返回当前对象字符串。
- **可以重写它定义我们自己的对象转字符串的规则**
- 非常常用：调用打印方法时，默认使用的就是对象的 ToString 方法后打印出的内容

```csharp
class Test
{
	public override string ToString() 
	{ 
		//自己定义规则  
		return base.ToString();
	}
}
```

## 练习

> 1. 有一个玩家类，有姓名、血量、攻击力、防御力、闪避率等特征，请在控制台打印出"玩家XX，血量XX，攻击力XX，防御力XX，闪避率XX"

```csharp
class Player
{
    private string name;
    private int hp, atk, def, miss;

    public Player(string name, int hp, int atk, int def, int miss)
    {
        this.name = name; 
        this.hp = hp; 
        this.atk = atk;
        this.def  = def;  
        this.miss = miss;
    }

    public override string ToString()
    {
        return string.Format("玩家{0}，血量{1}，攻击力{2}，防御力{3}，闪避率{4}", name, hp, atk, def, miss);
    }
}

Player p = new Player("Hentai", 999, 999, 999, 100);
Console.WriteLine(p);
```

> 2. 一个 Monster 类的引用对象 A，Monster 类有攻击力、防御力、血量、技能ID等属性，我想复制一个和 A 对象一模一样的 B 对象，并且改变了 B 的属性，A 不会受到影响，请问如何实现？

```csharp
class Monster
{
    public int Atk { get; set; }
    public int Def { get; set; }
    public int Hp { get; set; }
    public int SkillId { get; set; }

    public Monster() 
    { 
	    Atk = 1; 
	    Def = 1; 
	    Hp = 1; 
	    SkillId = 1; 
    }

    public Monster Clone()
    {
        return MemberwiseClone() as Monster;
    }
}

Monster A = new Monster();
Monster B = A.Clone();
```
