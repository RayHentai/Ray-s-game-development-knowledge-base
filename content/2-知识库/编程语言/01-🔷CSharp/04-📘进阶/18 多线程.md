## 基本概念

**进程（process）：** 是计算机中的程序关于某数据集合的上一次运行活动。简单说，打开一个应用程序就是在操作系统上开启了一个进程。进程之间可以相互独立运行，互不干扰。进程之间可以相互访问、操作。

**线程：** 操作系统能够进行运算调度的最小单位，包含在进程之中。一条线程就是代码从上到下运行的一条管道。我们目前写的程序都在主线程中。

**多线程：** 通过代码开启新的线程，可以同时运行代码的多条管道。线程之间可以相互独立运行，互不干扰。线程之间可以相互访问、操作。

## 语法

```csharp
using System.Threading;  // 需要引用此命名空间

// 声明（线程执行的代码需要封装到一个函数中）
Thread t = new Thread(NewThreadLogic);

static void NewThreadLogic() 
{ 
	//线程执行的逻辑 
}

// 启动
t.Start();

// 设置为后台线程（推荐）
// 注意：当前台线程都结束时，整个程序就结束了，即使还有后台线程在运行
// 如果不设置为后台线程，可能导致进程无法正常关闭
t.IsBackground = true;

// 线程休眠（毫秒）
Thread.Sleep(1000);  // 休眠 1 秒，在哪个线程里执行就休眠哪个线程

// 关闭线程（不是死循环就不用手动关闭）
t = null;  // 释放

// 或用 bool 标识控制死循环退出
static void NewThreadLogic() 
{ 
	while (isRunning) { }
}
//按一个键关闭后台线程
Console.ReadKey();
isRunning = false;
//在按一个键关闭主线程
Console.ReadKey();

//通过线程提供的方法  
t.Abort();
//在.Net core版本中无法中止 会报错 
//当前就 是这个版本 会报错
```

## 线程之间共享数据

> 多线程使用的内存是共享的，当多线程同时操作同一片内存区域时可能会出问题（交叉执行，顺序错乱）。

```csharp
// 加锁：避免线程安全问题
// 当 lock 语句块中的代码执行时，会给 () 中的引用类型变量上锁
// 另一个 lock 中如果是同一个引用类型对象，会等上一个 lock 执行完再执行
object lockObj = new object();
lock (lockObj)
{
    // 需要保护的逻辑
}
```

## 练习

> 1. 控制台中有一个 ■，它会自动移动，请开启一个多线程来检测输入，控制它的转向

```csharp
enum E_MoveDir 
{ 
	up, 
	down, 
	left, 
	right 
}

class Icon
{
    public E_MoveDir dir;
    public int x, y;
    public Icon(int x, int y, E_MoveDir dir) 
    { 
	    this.x = x; 
	    this.y = y; 
	    this.dir = dir; 
    }

    public void Move()
    {
        switch (dir)
        {
            case E_MoveDir.up: 
	            --y;    
	            break;
            case E_MoveDir.down: 
	            ++y;    
	            break;
            case E_MoveDir.left:  
	            x -= 2; 
	            break;
            case E_MoveDir.right: 
	            x += 2; 
	            break;
        }
    }

    public void Draw()  
    { 
	    Console.SetCursorPosition(x, y); 
	    Console.Write("■"); 
    }
    public void Clear() 
    { 
	    Console.SetCursorPosition(x, y); 
	    Console.Write("  "); 
    }
    public void ChangeDir(E_MoveDir dir) 
    { 
	    this.dir = dir; 
    }
}

static Icon icon;

public static void DIR()
{
    while (true)
    {
        switch (Console.ReadKey(true).Key)
        {
            case ConsoleKey.W: icon.ChangeDir(E_MoveDir.up);    
	            break;
            case ConsoleKey.S: icon.ChangeDir(E_MoveDir.down);  
	            break;
            case ConsoleKey.A: icon.ChangeDir(E_MoveDir.left);  
	            break;
            case ConsoleKey.D: icon.ChangeDir(E_MoveDir.right); 
	            break;
        }
    }
}

icon = new Icon(10, 5, E_MoveDir.right);
Thread t = new Thread(DIR);
t.Start();
t.IsBackground = true;
while (true)
{
    Thread.Sleep(500);
    icon.Clear();
    icon.Move();
    icon.Draw();
}
```
