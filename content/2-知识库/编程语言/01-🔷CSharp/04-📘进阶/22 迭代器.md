## 核心理解

> 迭代器（iterator）又称光标（cursor），是程序设计的软件设计模式。
> 迭代器模式提供一个方法顺序 访问一个聚合对象中的各个元素，而又不暴露其内部的标识

>**从表现效果上看**，是可以在容器对象（链表或数组）上遍历访问的接口
>设计人员无需关心容器对象的内存分配的实现细节，可以用`foreach`遍历的类 都是实现了迭代器的

**作用：** 迭代器就是可以让我们在外部直接通过`foreach`遍历对象中的元素 而不需要了解其结构

## 方式一：标准迭代器实现方法

```csharp
using System.Collections;  // 命名空间

// foreach (类型 item in 集合) 本质上是：
// 1. 获取 in 后面对象的 IEnumerator（调用 GetEnumerator 方法）
// 2. 执行 MoveNext 方法，只要返回 true 就得到 Current 赋值给 item
// 3. 重复步骤2直到 MoveNext 返回 false

foreach (int item in list) { }

//关键接口 IEnumerable 既不继承都可以 只要实现了 GetEnumerator方法 都可以用迭代器遍历
//关键接口 IEnumerator
class Test : IEnumerable, IEnumerator
{
    private int[] array;
    private int position = -1;  // 光标，默认 -1

    public Test() 
    { 
	    array = new int[] { 1, 2, 3, 4, 5, 6, 7, 8, 9 }; 
    }

    // IEnumerator.Current：得到当前光标的值
    public object Current 
	{ 
		get 
	    { 
		    return array[position]; 
	    } 
    }

    // IEnumerable.GetEnumerator：返回迭代器（把自己返回出去）
    public IEnumerator GetEnumerator() 
    { 
	    Reset();
	    return this; 
    }

    // IEnumerator.MoveNext：移动光标
    public bool MoveNext() 
    { 
	    return ++position < array.Length; 
    }

    // IEnumerator.Reset：复原光标
    public void Reset() 
    { 
	    position = -1; 
    }
}
```

## 方式二：yield return（语法糖，更简洁）

- `yield return` 是`C#`提供给我们的语法糖

**语法糖 也称糖衣语法**
- 主要作用就是将复杂逻辑简单化 可以增加程序的可读性
- 从而减少程序代码出错的机会
- 关键接口 `IEnumerable`
- 命名空间 `System.Collections`
- 本质就是 系统识别到`yield return` 就会帮我们执行迭代器相关逻辑

```csharp
// 系统识别到 yield return 就会帮我们执行迭代器相关逻辑
class Test2 : IEnumerable
{
    private int[] array;
    public Test2() 
    { 
	    array = new int[] { 1, 2, 3, 4, 5, 6, 7, 8, 9 }; 
    }

    public IEnumerator GetEnumerator()
    {
        yield return array[i];
    }
}
```

## 练习

> 1. 请为一个自定义类用两种方法让其可以被 foreach 遍历

```csharp
// 方式一：实现 IEnumerable + IEnumerator 接口
class Test : IEnumerable, IEnumerator
{
    private int[] array;
    private int position = -1;

    public Test() 
    { 
	    array = new int[] { 1, 2, 3, 4, 5, 6, 7, 8, 9 }; 
    }

    public object Current  
    { 
	    get 
	    { 
		    return array[position]; 
	    } 
    }
    public IEnumerator GetEnumerator() 
    { 
	    Reset(); 
	    return this; 
    }
    public bool MoveNext() 
    { 
	    return ++position < array.Length; 
    }
    public void Reset()    
    { 
	    position = -1; 
    }
}

// 方式二：yield return
class Test2 : IEnumerable
{
    private int[] array;
    public Test2() 
    { 
	    array = new int[] { 1, 2, 3, 4, 5, 6, 7, 8, 9 }; 
    }

    public IEnumerator GetEnumerator()
    {
        for (int i = 0; i < array.Length; ++i)
            yield return array[i];
    }
}

Test test = new Test();
foreach (int item in test) 
	Console.WriteLine(item);
Console.WriteLine("************************");
foreach (int item in test) 
	Console.WriteLine(item);  // 可重复遍历（Reset 生效）
Console.WriteLine("************************");

Test2 test2 = new Test2();
foreach (int item in test2) 
	Console.WriteLine(item);
Console.WriteLine("************************");
foreach (int item in test2) 
	Console.WriteLine(item);
```
