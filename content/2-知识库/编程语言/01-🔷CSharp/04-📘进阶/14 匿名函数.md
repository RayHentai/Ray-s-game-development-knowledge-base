## 核心理解

> 匿名函数 = 没有名字的函数，主要配合委托和事件使用。
> **脱离委托和事件是不会使用匿名函数的，单独使用匿名函数会报错。**

---

## 语法

```csharp
// 声明
delegate(参数列表)
{
    函数逻辑
};
```

---

## 使用方式

```csharp
// 无参无返回值：赋值
Action action = delegate()
{
    Console.WriteLine("Hello");
};
action();

// 有参
Action<int> action2 = delegate(int value)
{
    Console.WriteLine(value);
};

// 有返回
Func<string> func = delegate()
{
    return "123123";
};

// 有参有返回值
Func<int, string> func = delegate(int value)
{
    return value.ToString();
};

// 作为函数参数传递
void DoSomething(Action callback) 
{ 
	callback(); 
}
DoSomething(delegate() 
{ 
	Console.WriteLine("回调执行"); 
});

// 作为函数返回值
class Test
{
	public Action GetFun()
	{
		return delegate(){
			//执行逻辑
		};
	}
}
Test t = new Test();
t.GetFun()();
```

---

## 注意

- 匿名函数内部可以继续封装函数
- **缺点：添加到委托或事件容器后，无法单独/指定移除，只能清空**

---

## 练习

> 1. 写一个函数传入一个整数，返回一个函数；之后执行这个匿名函数时传入一个整数，和之前那个函数传入的数相乘并返回结果

```csharp
public static Func<int, int> Test(int i)
{
    return delegate(int v)
    {
        return v * i;
    };
}

Func<int, int> funk = Test(2);
Console.WriteLine(funk(3));     // 6
Console.WriteLine(Test(2)(3)); // 6
```
