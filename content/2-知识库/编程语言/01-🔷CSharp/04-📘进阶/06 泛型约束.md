## 核心理解

> 泛型约束 = 让泛型的类型有一些限制，用 `where` 关键字指定约束条件。

---

## 约束语法

**注意：**
- 两个泛型约束可以组合使用，用 `,` 隔开
- 如果有 `new()` 约束，必须写在末尾，否则报错

```csharp
// where 泛型字母 : 约束类型

where T : struct // 值类型
where T : class // 引用类型
where T : new() // 存在无参公共构造函数（值类型也可以，因为结构体无参构造不会被顶掉）
where T : 类名 // 某个类本身或其派生类
where T : 接口名 // 某个接口的派生类型
where T : U // 另一个泛型类型本身或其派生类型

// 多个约束组合（用 , 隔开）
where T : class, new()

// 注意：如果有 new()，必须写在末尾
// 多个泛型约束直接写就行，不用分隔符
void Method<T, K>() where T : class where K : new() { }
```

## 练习

> 1. 用泛型实现一个单例模式基类    [[01 单例模式基类]]

```csharp
class InstanceBase<T> where T : new()
{
    private static T instance = new T();
    public static T Instance => instance;
}

class Test : InstanceBase<Test>
{
    // 注意：小缺点，必须保留无参构造函数（单例模式约定不会在外部 new 新对象）
    public int a = 10;
}
Console.WriteLine(Test.Instance.a); // 10
```

> 2. 利用泛型知识点，仿造 ArrayList 实现一个不确定数组类型的类，实现增删查改方法

```csharp
class ArrayList<K>
{
    public K[]  array;
    private int count;

    public ArrayList() 
    { 
	    count = 0; 
	    array = new K[5]; 
    }

    public void Add(K value)
    {
        if (Capacity <= count)
        {
            K[] temp = new K[Capacity * 2];
            for (int i = 0; i < count; i++) 
	            temp[i] = array[i];
            array = temp;
        }
        array[count] = value;
        ++count;
    }

    public void RemoveAt(int index)
    {
        if (index > count - 1 || index < 0) 
        { 
	        Console.WriteLine("请输入合法的值"); 
	        return; 
        }
        for (int j = index; j < count - 1; j++) 
	        array[j] = array[j + 1];
        array[count - 1] = default(K);
        --count;
    }

    public void Remove(K value)
    {
        for (int i = 0; i < count - 1; i++)
        {
            if (array[i].Equals(value)) 
            { 
	            RemoveAt(i); 
	            return; 
            }
        }
    }

    public K this[int index]
    {
        get
        {
            if (index > count - 1 || index < 0) 
            { 
	            Console.WriteLine("请输入合法的值");
	            return default(K); 
            }
            return array[index];
        }
        set
        {
            if (index > count - 1 || index < 0) 
            { 
	            Console.WriteLine("请输入合法的值"); 
	            return; 
            }
            array[index] = value;
        }
    }

    public int Count 
    { 
	    get 
	    { 
		    return count; 
	    } 
    }
    public int Capacity 
    { 
	    get 
	    { 
		    return array.Length; 
	    } 
    }
}
```

## 我的踩坑记录

- `where T : new()` 要求 T 必须有无参构造，忘了会报编译错误
- `new()` 约束必须写在所有约束的最后面
