## 核心理解

> **委托是函数（方法）的容器**，可以理解为表示函数的变量类型，用来存储、传递函数。

> 委托的本质是一个类，用来定义函数的类型（返回值和参数的类型）。

> 不同的函数必须对应和各自"格式"一致的委托（有参无参，有返回无返回）。

---

## 声明委托

**规则：**
- 关键字 `delegate`
- 声明位置 `namespace`（一般）、`class`
- 访问修饰符默认不写 为`public`
- 不能重名，不存在重载，会报错

```csharp
// 访问修饰符 delegate 返回值 委托名(参数列表);
// 简单记忆：函数声明语法前加个 delegate 关键字
delegate void MyDelegate(string msg);

// 泛型委托（同泛型函数声明）
delegate T MyFunc<T, K>(T t, K k);
```

---

## 声明委托变量

```csharp
// 方式1
MyDelegate d = new MyDelegate(函数名);
// 方式2（简化）
MyDelegate d = 函数名;
```

---

## 委托的使用

```csharp
// 有参时括号中填写参数，与函数的使用方法一致
d.Invoke(); // 方式1
d(); // 方式2（简化）
```

---

## 委托变量可以存储多个函数

- 委托变量可以存储多个函数（多播委托）。
- 常用作：类的成员变量、函数的参数。

**注意：**
- 委托为 `null` 时调用会报错，调用前要判空
- 多减不会报错，系统找不到不会处理
- 一般有加必有减，或者选择清空委托
```csharp
// 添加函数（+=）
d += 函数名1;
d += 函数名2;

// 移除函数（-=）
d -= 函数名1;
```

## 系统定义好的委托

> 需要引用 System 命名空间。

```csharp
// Action：无参无返回值委托
Action action = SomeVoidMethod;

// Action<T, ...>：有参无返回值，最多16个参数
Action<int> a1 = (i) => { };
Action<int, int> a2 = (a, b) => { };

// Func<T>：无参有返回值委托（<> 最后一个类型为返回类型）
Func<int> f1 = () => 0;

// Func<T1, T2>：有参有返回值，最多16个参数
Func<int, int> f2 = (i) => i * 2;
Func<int, int, int> f3 = (a, b) => a + b;
```

## 练习

> 1. 一家三口，妈妈做饭，爸爸妈妈和孩子都要吃饭，用委托模拟做饭→开饭→吃饭的过程

```csharp
abstract class Person 
{ 
	public abstract void Eat(); 
}

class Mother : Person
{
    public Action beginEat;
    public override void Eat() 
    { 
	    Console.WriteLine("妈妈吃饭"); 
    }
    public void Cook()
    {
        Console.WriteLine("妈妈做饭");
        Console.WriteLine("开饭啦！");
        if (beginEat != null) 
	        beginEat();
    }
}
class Father : Person 
{ 
	public override void Eat() 
	{ 
		Console.WriteLine("爸爸吃饭"); 
	} 
}
class Son : Person 
{ 
	public override void Eat() 
	{ 
		Console.WriteLine("儿子吃饭"); 
	} 
}

Mother m = new Mother(); 
Father f = new Father(); 
Son s = new Son();
m.beginEat += f.Eat;
m.beginEat += s.Eat;
m.beginEat += m.Eat;
m.Cook();
```

> 2. 怪物死亡后，玩家要加10块钱，界面要更新数据，成就要累加怪物击杀数，请用委托来模拟实现这些功能

```csharp
class Monster
{
    public Action<Monster> monsterDie;
    public int money = 10;
    public void Die()
    {
        Console.WriteLine("怪物死亡");
        if (monsterDie != null) 
	        monsterDie(this);
        monsterDie = null;
    }
}
class Player  
{ 
	private int money = 0; 
	public void Money(Monster m)   
	{ 
		money += m.money; 
		Console.WriteLine("玩家获得{0}块钱", money); 
	} 
}
class UI 
{ 
	private int money = 0; 
	public void Ui(Monster m)      
	{ 
		money += m.money; 
		Console.WriteLine("当前持有{0}块钱", money); 
	} 
}
class ChengJiu 
{ 
	private int num = 0;  
	public void CJ(Monster m)      
	{ 
		++num; 
		Console.WriteLine("当前击杀{0}只怪物", num); 
	} 
}

Monster monster = new Monster();
Player player = new Player();
UI ui = new UI();
ChengJiu chengJiu = new ChengJiu();
monster.monsterDie += player.Money;
monster.monsterDie += ui.Ui;
monster.monsterDie += chengJiu.CJ;
monster.Die();
```
