## ✅ 阶段自检清单

**数据结构**
- [ ] 能说清楚 ArrayList / Stack / Queue / Hashtable 各自的特点和使用场景
- [ ] 理解泛型的作用，能写泛型类、泛型接口、泛型方法
- [ ] 理解泛型约束的语法和用途
- [ ] 能说清楚 List 和 ArrayList 的区别
- [ ] 能说清楚 Dictionary 和 Hashtable 的区别
- [ ] 能说清楚顺序存储和链式存储的区别
- [ ] 能自己实现双向链表

**委托与事件**
- [ ] 理解委托的本质，能声明和使用委托，会用 += 绑定多个函数
- [ ] 理解事件和委托的区别，知道事件的限制
- [ ] 能写匿名函数，理解闭包的注意事项
- [ ] 能写 Lambda 表达式
- [ ] 会用 Lambda 对 List 进行自定义排序

**高级特性**
- [ ] 会创建和启动线程，知道前台线程和后台线程的区别
- [ ] 了解预处理器指令的种类和用途
- [ ] 理解反射的概念，能用反射实例化对象、获取和设置字段值
- [ ] 能创建自定义特性，能通过反射读取特性
- [ ] 能用两种方式实现迭代器
