## 数据结构

> 计算机存储、组织数据的方式（规则）。

- 指相互之间存在一种或多种特定关系的数据元素的集合
- 比如自定义的一个类，也可以称为一种数据结构，自己定义的数据组合规则
- 简单说就是：人定义的，数据存储和表示数据之间关系的规则
- 常用的数据结构：**数组、栈、队列、链表、树、图、堆、散列表**
- 顺序存储和链式存储，是数据结构中的两种存储结构

---

## 线性表

是一种数据结构，是由 n 个具有相同特征的数据元素的有限序列（一个连一个）。
例如：数组、ArrayList、Stack、Queue、链表等。

---

## 顺序存储

> 内存中用一组**地址连续**的存储单元，依次存储线性表的各个数据元素（用下标存，一个连一个）。

- 数组、Stack、Queue、List、ArrayList 都是顺序存储
- 只是数组、Stack、Queue 的组织规则不同而已

![[顺序存储图示.png]]

---

## 链式存储（链接存储）

> 内存中用一组**任意**的存储单元，存储线性表中的各个数据元素。

- 单向链表、双向链表、循环链表（环形链表）都是链式存储
- 每个节点包含：数据 + 指向下一个节点的指针

![[链式存储图示.png]]

---

## 数据结构使用场景总结

**普通线性表（数组、List、LinkedList）**
- 数组：固定的不变的一组数据
- List：经常改变 经常通过下标查找
- LinkedList：不确定长度的 经常临时插入改变 查找不多

**先进后出（栈）**  
- 对于一些可以利用先进后出存储特点的逻辑  
- 比如：ui面板显隐规则

**先进先出 （队列）**
- 对于一些可以利用先进先出存储特点的逻辑  
- 比如：消息队列 有了就往里放 然后慢慢依次处理

**键值对（dictionary）**
- 需要频繁查找的 有对应关系的数据  
- 比如一些数据存储 ID对应内容  
- 道具ID → 道具信息  
- 怪物ID → 怪物对象  
- 等等

```csharp
//第一个节点的下一个元素 等于下一个节点

link2.nextNode.nextNode = new LinkedNode<int>(4);
```

## 为单向链表封装增删

```csharp
// 链表节点
class LinkedNode<T>
{
    public T value;
    public LinkedNode<T> nextNode;
    public LinkedNode(T value)
    {
        this.value = value;
    }
}
    
    /// 单向链表管理节点
class LinkedList<T>
{
    public LinkedNode<T> head;
    public LinkedNode<T> last;
    public void Add(T value)
    {
        //添加节点就要 new一个新节点
        LinkedNode<T> node = new LinkedNode<T>(value);
        //判断头是不是为空
        if (head == null)
        {
            //头尾都指向新节点
            head = node;
            last = node;
        }
        else
        {
            //尾的下一个节点指向新节点
            last.nextNode = node;
            //尾等于新节点
            last = node;
        }
    }
    public void Remove(T value)
    {
        //如果头为空
        if (head == null)
            return;
        //如果头值等于传入值
        if (head.value.Equals(value))
        {
            //头 等于下一个
            head = head.nextNode;
            //如果移除完 头是空 
            if (head == null)
                last = null;
            return;
        }
        //头不等于传入值
        //遍历
        //new一个临时变量 节点 等于 头
        LinkedNode<T> node = head;
        //循环 头的下一个值不等于空
        while (node.nextNode != null)
        {
            //如果头下一个值 等于传入值 
            if (node.nextNode.value.Equals(value))
            {
                //上一个节点的下一个 等于 下一个节点
                node.nextNode = node.nextNode.nextNode;
                //如果下一个指向空 那么这个节点 就是最后一个节点
                if (node.nextNode == null)
                    last = node;
                break;
            }
            //没有找到头 头就等于下一个 继续找
            node = node.nextNode;
        }
    }
}

//增  
LinkedList<int> list = new LinkedList<int>();
list.Add(0);
list.Add(1);
list.Add(2);
list.Add(3);
list.Add(4);
list.Add(5);
list.Add(6);

LinkedNode<int> head = list.head;
while (head != null)
{
    Console.WriteLine(head.value);
    head = head.nextNode;
}

//删
list.Remove(0);
list.Remove(1);
list.Remove(2);
list.Remove(6);    Console.WriteLine("***********************");
head = list.head;
//循环遍历 每一次遍历链表的头节点都会往前移 直到头节点为空则证明遍历完毕
while (head != null)
{
	Console.WriteLine(head.value);
    head = head.nextNode;
}

list.Add(999);
list.Add(999);
list.Add(999);
list.Add(699);
Console.WriteLine("***********************");

head = list.head;
while (head != null)
{
    Console.WriteLine(head.value);
    head = head.nextNode;
}
```

## 优缺点对比

| | 顺序存储 | 链式存储 |
|-|---------|---------|
| 增（中间插入） | ❌ 需要移动元素 | ✅ 只修改指针 |
| 删（中间删除） | ❌ 需要移动元素 | ✅ 只修改指针 |
| 查（随机访问） | ✅ 下标直接访问 | ❌ 需要遍历 |
| 改（随机修改） | ✅ 下标直接修改 | ❌ 需要遍历 |

```csharp
class LinkedNode<T>
{
    public T value;
    public LinkedNode<T> nextNode;
    public LinkedNode<T> previous;
    public LinkedNode(T value) 
    { 
	    this.value = value; 
    }
}

class LinkedList<T>
{
    public int count = 0;
    public LinkedNode<T> head;
    public LinkedNode<T> last;

    public LinkedNode<T> Head => head;
    public LinkedNode<T> Last => last; 
    public int Count => count;

    public void Add(T value)
    {
        LinkedNode<T> node = new LinkedNode<T>(value);
        if (head == null) 
        { 
	        head = node; 
	        last = node; 
        }
        else 
        { 
	        node.previous = last; 
	        last.nextNode = node; 
	        last = node; 
        }
        ++count;
    }

    public void RemoveAt(int index)
    {
        if (index < 0 || index > count - 1)
        {
            Console.WriteLine("只有{0}个节点，请输入合法的数字", count);
            return;
        }
        int tempCount = 0;
        LinkedNode<T> temp = head;
        while (true)
        {
            if (tempCount == index)
            {
                if (temp.previous != null) 
	                temp.previous.nextNode = temp.nextNode;
                if (temp.nextNode  != null) 
	                temp.nextNode.previous  = temp.previous;
                if (index == 0) 
	                head = head.nextNode;
                else if (index == count - 1)
	                last = last.previous;
                break;
            }
            temp = temp.nextNode;
            ++tempCount;
        }
        --count;
    }
}

LinkedList<int> list = new LinkedList<int>();
list.Add(0); list.Add(1); list.Add(2); list.Add(3);

LinkedNode<int> node = list.head;
while (node != null) 
{ 
	Console.WriteLine(node.value); 
	node = node.nextNode; 
}
node = list.last;
while (node != null) 
{ 
	Console.WriteLine(node.value); 
	node = node.previous; 
}

list.RemoveAt(0); list.RemoveAt(0);
```
