## 核心理解

> 泛型 = 类型参数化，用 `<T>` 占位符代表类型，让同一份代码处理多种类型。

> 定义类或方法时使用替代符代表变量类型，真正使用时再具体指定类型。

---

## 语法

```csharp
// <> 中为泛型占位字母，一般为大写，多个用 , 隔开

// 泛型类
class 类名<T> { }

// 泛型接口
interface 接口名<T> { }

// 泛型方法（函数名后加 <>）
返回值 函数名<T>(参数列表) { }
```

---

## 声明和使用

```csharp
//声明
//泛型类
class TestClass<T>
{
	public T Value;
}
TestClass<int> t = new TestClass<int>();
TestClass<string> t2 = new TestClass<string>();

// 泛型接口
insterface TestInsterface
{
	T Value
	{
		get;
		set;
	}
}
//继承时要明确占位符变量
class Test : TestInsterface<int>
{
	public int Value { //处理逻辑 }
}

// 普通类中的泛型方法
//作为参数
public void TestFun<T> (T value) { }

//无参数
public void TestFun<T> () 
{
	//在里面处理一些逻辑
	T t = default(t);
}

//指定类型参数
public T TestFun<T> (stringT value) 
{
	return default(T);
}

//多个参数
public void TestFun<T, K, M> (T t, K k, M m) { }

// 泛型类中的泛型方法（<> 包裹的字母肯定是泛型）
class Test<T> 
{ 
	public void Show<K>(K value) 
	{ 
		Console.WriteLine(value); 
	} 
	//这个不叫泛型方法 T在泛型类声明的时候就以及指定类型了
	//所以不能再动态的变化T的类型了
	public void TestFun(T t) { }
}
```

---

## 注意事项

- 泛型类和普通类是不一样的，可以重名
- 不确定泛型类型时，获取默认值可以使用 `default(占位字符)`
- 泛型真正起作用的时候，是在使用它的时候
- <>包裹的字母，肯定是泛型
- 使用泛型函数时 <类型> 可以不写，系统会自动判断

---

## 什么时候用

> 适用场景，帮助建立使用直觉

- 不同类型对象的相同逻辑处理 ，例如优化ArrayList，一定程度避免装箱拆箱

## 练习

> 1. 定义一个泛型方法，方法内判断该类型为何类型，并返回类型的名称与占有的字节数，如果是int，则返回“整形，4字节”

>只考虑以下类型
- int：整形
- char:字符
- float：单精度浮点数
- string：字符串
- 如果是其它类型，则返回“其它类型”

```csharp
using System.Runtime.InteropServices;

public static string Tool<T>()
{
    string str;
    int i;
    if (typeof(T) != typeof(string))
        i = Marshal.SizeOf(typeof(T));
    else
        return "字符串，?字节";

    if (typeof(T) == typeof(int))   
	    str = "整形";
    else if (typeof(T) == typeof(char))  
	    str = "字符";
    else if (typeof(T) == typeof(float)) 
	    str = "单精度浮点数";
    else 
	    return "其它类型";

    return string.Format("{0}，{1}字节", str, i);
}

Console.WriteLine(Tool<int>()); // 整形，4字节
Console.WriteLine(Tool<string>()); // 字符串，?字节
Console.WriteLine(Tool<float>()); // 单精度浮点数，4字节
Console.WriteLine(Tool<double>()); // 其它类型
Console.WriteLine(Tool<char>()); // 字符，2字节
Console.WriteLine(Tool<bool>()); // 其它类型
```
