## 变长参数 params

> 允许函数接收数量不定的同类型参数，调用时可以传任意多个，也可以不传。

```csharp
// 用 params 关键字，参数类型必须是数组
static void PrintAll(params int[] arr)
{
    if (arr.Length == 0) 
    { 
	    Console.WriteLine("没有参数"); 
	    return; 
    }
    for (int i = 0; i < arr.Length; i++)
        Console.WriteLine(arr[i]);
}

// 调用时可以传任意多个参数
PrintAll(1, 2, 3);
PrintAll(10, 20);
PrintAll();// 一个都不传也可以
PrintAll(new int[] { 1, 2, 3 });// 也可以直接传数组
```

**注意**：
- `params` 参数必须是函数的**最后一个参数**
- 一个函数只能有一个 `params` 参数

## 参数默认值（可选参数）

> 给参数预设一个默认值，调用时可以不传这个参数。

```csharp
static void Greet(string name, string greeting = "你好")
{
    Console.WriteLine(greeting + "，" + name);
}

Greet("张三");// 输出：你好，张三（用默认值）
Greet("张三", "早上好"); // 输出：早上好，张三（覆盖默认值）
```

**注意**：
- 支持多参数默认值，每个参数都可以有默认值
- 有默认值的参数必须放在**没有默认值的参数后面**
- 可选参数不能用于函数重载（会产生歧义）

## 练习

> 1. 使用param参数，求多个数字的和以及平均数

```csharp
static void ShuZi(params int[] arr)
{
    if (arr.Length == 0) 
    {
	    Console.WriteLine("没有参数"); 
	    return; 
    }
    int sub = 0, avg = 0;
    for (int i = 0; i < arr.Length; i++)
        sub += arr[i];
    avg = sub / arr.Length;
    Console.WriteLine("总和{0} 平均值{1}", sub, avg);
}
```

> 2. 使用param参数，求多个数字的偶数和奇数和

```csharp
static void ShuZi1(params int[] arr)
{
    if (arr.Length == 0) 
    { 
	    Console.WriteLine("没有参数"); 
	    return; 
    }
    int subo = 0, subj = 0;
    for (int i = 0; i < arr.Length; i++)
    {
        if (arr[i] % 2 == 0) 
	        subo += arr[i];
        else                  
	        subj += arr[i];
    }
    Console.WriteLine("偶数和{0} 奇数和{1}", subo, subj);
}
```
