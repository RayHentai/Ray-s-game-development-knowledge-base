## 核心理解

> 选择排序 = 每轮从未排序区找出最大/最小值，放到末尾/开头。
> 不像冒泡一直交换，选择排序每轮只交换一次。

---

## 原理示意

```
原始数组：[5, 3, 8, 1]（降序：每轮把最大值放末尾）

第1轮：扫描全部，最大值是 8（index=2），和末尾 arr[3]=1 交换
  → [5, 3, 1, 8]

第2轮：扫描前3个，最大值是 5（index=0），和末尾 arr[2]=1 交换
  → [1, 3, 5, 8]

……以此类推
```

## 代码实现（降序）

- 新建中间商
- 依次比较
- 找出极值
- 放入目标位置
- 比较n轮

```csharp
int[] arr = { 5, 3, 8, 1, 9 };
int temp, index;

for (int m = 0; m < arr.Length; m++)
{
    index = 0;
    // 找当前未排序区最大值的下标
    for (int n = 1; n < arr.Length - m; n++)
    {
        if (arr[index] < arr[n]) index = n;
    }
    // 把最大值和末尾交换
    if (index != arr.Length - 1 - m)
    {
        temp                    = arr[index];
        arr[index]              = arr[arr.Length - 1 - m];
        arr[arr.Length - 1 - m] = temp;
    }
}
```

## 和冒泡排序的区别

| | 冒泡排序 | 选择排序 |
|-|---------|---------|
| 交换次数 | 多（每次比较都可能交换） | 少（每轮只交换一次） |
| 稳定性 | 稳定 | 不稳定 |
| 适用场景 | 交换代价低的情况 | 交换代价高的情况 |

## 练习

> 1. 定义一个数组，长度为20，每个元素值随机0~100，使用选择排序进行升序排序并打印，再进行降序排序并打印

```csharp
int[] arr = new int[20];
Random r = new Random();
int temp, index;
for (int i = 0; i < arr.Length; i++) arr[i] = r.Next(0, 101);

// 降序（每轮把最大值放到末尾）
for (int m = 0; m < arr.Length; m++)
{
    index = 0;
    for (int n = 1; n < arr.Length - m; n++)
        if (arr[index] < arr[n]) index = n;
    if (index != arr.Length - 1 - m)
    {
        temp = arr[index]; arr[index] = arr[arr.Length-1-m]; arr[arr.Length-1-m] = temp;
    }
}
for (int i = 0; i < arr.Length; i++) Console.WriteLine(arr[i]);

// 升序（把 < 改成 >，每轮把最小值放到末尾）
for (int m = 0; m < arr.Length; m++)
{
    index = 0;
    for (int n = 0; n < arr.Length - m; n++)
        if (arr[index] > arr[n]) index = n;
    if (index != arr.Length - 1 - m)
    {
        temp = arr[index]; arr[index] = arr[arr.Length-1-m]; arr[arr.Length-1-m] = temp;
    }
}
for (int i = 0; i < arr.Length; i++) Console.WriteLine(arr[i]);
```
