## 核心理解

>交错数组 = 数组的数组 可以存储同一类型的m行不确定列的数据，每行的列数可以不同

- 类比：一栋楼，每层的房间数不一样。

---

## 声明交错数组

```csharp
// 方式1：只声明
int[][] arr;

// 方式2：指定行数
int[][] arr = new int[3][];

// 方式3：指定行数，每行单独初始化
int[][] arr = new int[3][];
arr[0] = new int[] { 1, 2 };       // 第0行2列
arr[1] = new int[] { 3, 4, 5 };    // 第1行3列
arr[2] = new int[] { 6 };          // 第2行1列

// 方式4：整体初始化
int[][] arr = new int[][]
{
    new int[] { 1, 2 },
    new int[] { 3, 4, 5 },
    new int[] { 6 },
};

// 方式5：最简写法
int[][] arr =
{
    new int[] { 1, 2 },
    new int[] { 3, 4, 5 },
    new int[] { 6 },
};
```

## 使用交错数组

```csharp
// 获取行数
int rows = arr.Length;

// 获取某行的列数
int cols0 = arr[0].Length;   // 第0行的列数

// 获取元素
int val = arr[1][2];   // 第1行第2列

// 修改元素
arr[0][1] = 99;

// 遍历（外层行，内层列）
for (int i = 0; i < arr.Length; i++)
{
    for (int j = 0; j < arr[i].Length; j++)
        Console.Write(arr[i][j] + " ");
    Console.WriteLine();
}
```

## 和二维数组的区别

| 对比 | 二维数组 | 交错数组 |
|------|---------|---------|
| 每行列数 | 相同 | 可以不同 |
| 声明方式 | `int[,]` | `int[][]` |
| 获取行数 | `GetLength(0)` | `arr.Length` |
| 获取某行列数 | `GetLength(1)` | `arr[i].Length` |
| 访问元素 | `arr[i, j]` | `arr[i][j]` |
| 典型场景 | 棋盘/矩阵 | 每行长度不定的数据 |

---

## 注意事项

- 增加/删除元素同一维数组，需要新建数组替换
- 查找元素需要双层遍历

## 练习

> 1. 声明一个交错数组，共3行，第0行存 {1,2}，第1行存 {3,4,5}，第2行存 {6}，遍历打印所有元素

```csharp
int[][] arr =
{
    new int[] { 1, 2 },
    new int[] { 3, 4, 5 },
    new int[] { 6 },
};

for (int i = 0; i < arr.Length; i++)
{
    for (int j = 0; j < arr[i].Length; j++)
        Console.Write(arr[i][j] + " ");
    Console.WriteLine();
}
```

> 2. 声明一个5行的交错数组，第 i 行存储 i+1 个随机数（1~10），打印后计算每行的总和

```csharp
Random r = new Random();
int[][] arr = new int[5][];
for (int i = 0; i < arr.Length; i++)
{
    arr[i] = new int[i + 1];
    int sum = 0;
    for (int j = 0; j < arr[i].Length; j++)
    {
        arr[i][j] = r.Next(1, 11);
        Console.Write(arr[i][j] + " ");
        sum += arr[i][j];
    }
    Console.WriteLine("  总和：" + sum);
}
```

> 3. 用交错数组模拟一个班级的成绩表（3个小组，每组人数不同），找出全班最高分及其所在小组和位置

```csharp
int[][] scores =
{
    new int[] { 85, 90, 78 },       // 第0组 3人
    new int[] { 92, 88, 76, 95 },   // 第1组 4人
    new int[] { 70, 83 },           // 第2组 2人
};

int maxVal = 0, maxGroup = 0, maxPos = 0;
for (int i = 0; i < scores.Length; i++)
{
    for (int j = 0; j < scores[i].Length; j++)
    {
        if (scores[i][j] > maxVal)
        {
            maxVal   = scores[i][j];
            maxGroup = i;
            maxPos   = j;
        }
    }
}
Console.WriteLine("最高分：{0}，第{1}组第{2}人", maxVal, maxGroup, maxPos);
```
