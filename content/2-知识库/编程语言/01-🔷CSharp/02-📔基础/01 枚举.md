## 核心理解

> 枚举 = 被命名的整型常量的集合。
> 用来表示状态、类型等，让代码语义更清晰，避免用裸数字代表含义。

类比：与其用 0/1/2 表示"在线/离线/忙碌"，不如用 `E_Online.Online 一眼看懂。

---

## 声明枚举

>创建一个自定义的枚举类型

```csharp
// 命名规范：以 E 或 E_ 开头，帕斯卡命名
// 语法：enum E_自定义枚举名
enum E_PlayerState
{
	//自定义枚举项名字,
    Idle,       // 默认值从 0 开始依次累加
    Run,        // 1
    Attack,     // 2
    Dead = 10,  // 手动指定值
    Win,        // 11（在上一个基础上 +1）
    ...
    Play        //最后一项可以省略,
}
```

**声明位置**：namespace / class / struct 语句块中，**不能**在函数内部声明。

---

## 声明与使用枚举变量

```csharp
// 声明并赋值
E_PlayerState state = E_PlayerState.Idle;

// 一般配合 switch 使用
switch (state)
{
    case E_PlayerState.Idle:
        Console.WriteLine("站立");
        break;
    case E_PlayerState.Run:
        Console.WriteLine("跑步");
        break;
    case E_PlayerState.Attack:
        Console.WriteLine("攻击");
        break;
}
```

## 枚举的类型转换

```csharp
// 枚举 → int（括号强转）
E_PlayerState state = E_PlayerState.Attack;
int stateNum = (int)state;        // 2

// int → 枚举（括号强转）
E_PlayerState s = (E_PlayerState)2;  // Attack

// 枚举 → string(ToString)，显示的是枚举项名字，不是数值
string str = state.ToString();    // "Attack"

// string → 枚举（Enum.Parse）[[09 类型转换#^2381b2]]
//枚举类型(自定义枚举项名)Enum.Parse(自定义枚举类型的Type, "枚举项名")
E_PlayerState s2 = (E_PlayerState)Enum.Parse(typeof(E_PlayerState), "Attack");
```

---

## 注意事项

- 枚举中的数字是**常量**，不能在运行时修改
- 第一个枚举项默认值是 0，后续依次 +1
- 手动指定某项的值后，其后的项在此基础上继续 +1

---

## 什么时候用

>游戏开发中 对象很多时候会有很多状态

>比如玩家 有一个动作状态 我们需要一个变量或者标识 来表示当前玩家处于哪种状态

- 综合考虑可能会使用int，来表示他的状态
- 但是枚举相比int，可以帮助我们清晰的分析状态的含义

## 练习

> 1. 定义QQ状态的枚举，并提示用户选择一个在线状态，我们接受输入的数字，并将其转换成枚举类型

```csharp
enum E_Online
{
    Zaixian,   // 0
    Lixian,    // 1
    Manglu,    // 2
    Yingshen,  // 3
}

try
{
    Console.WriteLine("请输入一个在线状态，0为在线，1为离线，2为忙碌，3为隐身");
    E_Online online = (E_Online)int.Parse(Console.ReadLine());
    Console.WriteLine(online);
}
catch
{
    Console.WriteLine("请输入正确的数字");
}
```

> 2. 用户去星巴克买咖啡，分为中杯（35元），大杯（40元），超大杯（43元），  请用户选择要购买的类型，用户选择后，打印：您购买了XXX咖啡，花费了xx元  

 例如：``你购买了中杯咖啡，花费了35元``

```csharp
enum E_Cup
{
    Zhong,    // 0
    Da,       // 1
    ChaoDa,   // 2
}

try
{
    Console.WriteLine("请选择要购买的类型，0中杯，1大杯，2超大杯");
    E_Cup cup = (E_Cup)int.Parse(Console.ReadLine());
    switch (cup)
    {
        case E_Cup.Zhong:
            Console.WriteLine("你购买了{0}杯咖啡，花费了{1}元", "中", 35);
            break;
        case E_Cup.Da:
            Console.WriteLine("你购买了{0}杯咖啡，花费了{1}元", "大", 40);
            break;
        case E_Cup.ChaoDa:
            Console.WriteLine("你购买了{0}杯咖啡，花费了{1}元", "超大", 43);
            break;
        default:
            Console.WriteLine("请输入正确的数字");
            break;
    }
}
catch
{
    Console.WriteLine("请输入正确的数字");
}
```

> 3. 请用户选择英雄性别与英雄职业，最后打印英雄的基本属性（攻击力，防御力，技能)  

性别：男(攻击力+50，防御力+100)  
女(攻击力+150，防御力+20)  
职业：战士(攻击力+20，防御力+100，技能：冲锋)  
猎人(攻击+120，防御+30，技能：假死)  
法师(攻击力+200，防御力+10，技能：奥术冲击) 

举例打印：``你选择了“性法师”，攻击：350，防御：30，职业技能：奥术冲击``

```csharp
enum E_Sex   { Man, Woman }
enum E_ZhiYe { ZhanShi, LieRen, FaShi }

try
{
    int playerAtk = 0, playerDef = 0;
    string jiNeng = "", xingBie = "", zhiYe1 = "";

    Console.WriteLine("请选择英雄性别，男0，女1");
    E_Sex sex = (E_Sex)int.Parse(Console.ReadLine());
    switch (sex)
    {
        case E_Sex.Man:   playerAtk += 50;  playerDef += 100; xingBie = "男"; break;
        case E_Sex.Woman: playerAtk += 150; playerDef += 20;  xingBie = "女"; break;
    }

    Console.WriteLine("请选择英雄职业，战士0，猎人1，法师2");
    E_ZhiYe zhiYe = (E_ZhiYe)int.Parse(Console.ReadLine());
    switch (zhiYe)
    {
        case E_ZhiYe.ZhanShi: playerAtk += 20;  playerDef += 100; jiNeng = "冲锋";    zhiYe1 = "战士"; break;
        case E_ZhiYe.LieRen:  playerAtk += 120; playerDef += 30;  jiNeng = "假死";    zhiYe1 = "猎人"; break;
        case E_ZhiYe.FaShi:   playerAtk += 200; playerDef += 10;  jiNeng = "奥术冲击"; zhiYe1 = "法师"; break;
    }

    Console.WriteLine("你选择了\"{0}性{1}\"，攻击力：{2}，防御力：{3}，职业技能：{4}",
        xingBie, zhiYe1, playerAtk, playerDef, jiNeng);
}
catch
{
    Console.WriteLine("请输入正确的数字");
}
```
