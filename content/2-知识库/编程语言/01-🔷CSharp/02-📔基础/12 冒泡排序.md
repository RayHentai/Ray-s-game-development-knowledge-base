## 核心理解

> 冒泡排序 = 相邻两个元素比较，如果顺序错了就交换，像气泡一样大的往后浮。
> 每轮结束，最大（或最小）的元素"浮"到末尾，下一轮就不用再管它。

---

## 原理示意

```
原始数组：[5, 3, 8, 1]

第1轮：
  5,3 → 交换 → [3,5,8,1]
  5,8 → 不换 → [3,5,8,1]
  8,1 → 交换 → [3,5,1,8]  ← 8 沉底

第2轮：
  3,5 → 不换 → [3,5,1,8]
  5,1 → 交换 → [3,1,5,8]  ← 5 沉底

……以此类推
```

## 代码实现

- 两两相邻
- 不停比较
- 不停交换
- 比较n轮

```csharp
// 升序冒泡排序（带优化：某轮没有交换说明已排好，提前退出）
int[] arr = { 5, 3, 8, 1, 9 };
bool isSort = false;
int temp;

for (int m = 0; m < arr.Length; m++)
{
    isSort = false;
    for (int n = 0; n < arr.Length - 1 - m; n++)  // 每轮少比较一个（末尾已排好）
    {
        if (arr[n] > arr[n + 1])
        {
            isSort = true;
            temp       = arr[n];
            arr[n]     = arr[n + 1];
            arr[n + 1] = temp;
        }
    }
    if (!isSort) 
	    break;  // 这一轮没有交换，说明已经排好了
}
```

## 练习

> 1. 定义一个数组，长度为20，每个元素值随机0~100，使用冒泡排序进行升序排序并打印，再进行降序排序并打印

```csharp
int[] arr = new int[20];
Random r = new Random();
for (int i = 0; i < arr.Length; i++)
    arr[i] = r.Next(1, 101);

// 升序
bool isSort = false;
int temp;
for (int m = 0; m < arr.Length; m++)
{
    isSort = false;
    for (int n = 0; n < arr.Length - 1 - m; n++)
    {
        if (arr[n] > arr[n + 1])
        {
            isSort = true;
            temp = arr[n];
            arr[n] = arr[n + 1];
            arr[n + 1] = temp;
        }
    }
    if (!isSort) 
	    break;
}
for (int i = 0; i < arr.Length; i++) 
	Console.WriteLine(arr[i]);

// 降序（把 > 改成 <）
for (int m = 0; m < arr.Length; m++)
{
    isSort = false;
    for (int n = 0; n < arr.Length - 1 - m; n++)
    {
        if (arr[n] < arr[n + 1])
        {
            isSort     = true;
            temp       = arr[n];
            arr[n]     = arr[n + 1];
            arr[n + 1] = temp;
        }
    }
    if (!isSort) 
	    break;
}
for (int i = 0; i < arr.Length; i++) 
	Console.WriteLine(arr[i]);
```

> 2. 写一个函数，实现一个数组的排序，并返回结果。可以根据参数决定是升序还是降序

```csharp
static int[] PaiXu(int[] arr, bool shunXu)
{
    bool isSort;
    int temp;
    bool paiXu;
    for (int i = 0; i < arr.Length; i++)
    {
        isSort = false;
        for (int j = 0; j < arr.Length - 1 - i; j++)
        {
            paiXu = shunXu ? arr[j] > arr[j + 1] : arr[j] < arr[j + 1];
            if (paiXu)
            {
                isSort = true;
                temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
        if (!isSort) 
	        break;
    }
    return arr;
}

int[] arr = new int[20];
Random r = new Random();
for (int i = 0; i < arr.Length; i++) 
	arr[i] = r.Next(1, 101);
PaiXu(arr, false);  // 降序
for (int i = 0; i < arr.Length; i++) 
	Console.WriteLine(arr[i]);
```
