## ✅ 阶段自检清单

**复杂数据类型**
- [ ] 能定义并使用枚举，配合 switch 使用
- [ ] 能声明、遍历、增删查改一维数组
- [ ] 能使用二维数组表示格子/矩阵
- [ ] 理解交错数组和二维数组的区别
- [ ] 能用结构体封装一组相关数据，理解访问修饰符

**值类型与引用类型**
- [ ] 能说清楚值类型和引用类型在赋值时的区别
- [ ] 知道栈和堆的区别
- [ ] 理解 string 的特殊性

**函数**
- [ ] 能写无参无返回、有参有返回、多返回值等各种函数
- [ ] 理解 ref 和 out 的区别和使用场景
- [ ] 会用 params 写变长参数函数
- [ ] 能写函数重载（参数数量/类型/顺序不同）
- [ ] 理解递归，会写简单的递归函数

**排序**
- [ ] 能手写冒泡排序（升序/降序）
- [ ] 能手写选择排序

**综合**
- [ ] 独立完成飞行棋项目
