## 核心理解

> 函数重载 = 同一个名字，不同的参数（数量/类型/顺序不同），编译器自动选择调用哪个。

- 作用：用一个名字处理多种类型的相似逻辑，让代码更整洁。

---

## 重载的条件

在同一个 class 或 struct 中，函数名相同，但满足以下任一条件：

| 条件 | 示例 |
|------|------|
| 参数**数量**不同 | `void F(int a)` vs `void F(int a, int b)` |
| 参数**类型**不同 | `void F(int a)` vs `void F(float a)` |
| 参数**顺序**不同 | `void F(int a, float b)` vs `void F(float a, int b)` |

---

## 代码示例

```csharp
// 参数数量不同
static int Add(int a, int b) 
{ 
	return a + b; 
}
static int Add(int a, int b, int c) 
{ 
	return a + b + c; 
}

// 参数类型不同
static float Max(float a, float b) 
{ 
	return a > b ? a : b; 
}
static double Max(double a, double b) 
{ 
	return a > b ? a : b; 
}

//参数顺序不同
static int Add(int a, int b) 
{ 
	return a + b; 
}
static int Add(int b, int a) 
{ 
	return b + a; 
}

//params可以参与重载
// ref 和 out 也可以参与重载（但同一位置不能同时有 ref 和 out）
static void Info(ref int a)  { a = 1; }
static void Info(out int a)  { a = 2; }  // ❌ 同一位置 ref 和 out 不能重载
```

---

## 注意事项

- 重载**只看参数**，和返回值无关（返回值不同不算重载）
- 可选参数（有默认值的参数）**不能用于重载**，会产生歧义
- `ref` 和 `out` 在同一个位置不能互相重载

---

## 什么时候用

> 适用场景，帮助建立使用直觉

- 用于不同参数的同一类型逻辑处理（不同类型变量求和）
- 命名一组功能相似的函数，减少函数命名的数量，避免命名空间的污染，提升程序的可读性

## 练习

> 1. 请重载一个函数，让其可以比较两个int或两个float或两个double的大小，并返回较大的那个值
```csharp
static int BiJiao(int a, int b) 
{ 
	return a > b ? a : b; 
}
static float BiJiao(float a, float b) 
{ 
	return a > b ? a : b; 
}
static double BiJiao(double a, double b) 
{ 
	return a > b ? a : b; 
}
```

> 2. 请重载一个函数，让其可以比较n个int或n个float或n个double的大小并返回最大的那个值。（用params可变参数来完成）


```csharp
static int JiaoBi(params int[] arr)
{
    if (arr.Length == 0) 
    { 
	    Console.WriteLine("数组值为空值"); 
	    return -1; 
	}
    int max = arr[0];
    for (int i = 0; i < arr.Length; i++)
        if (max < arr[i]) 
	        max = arr[i];
    return max;
}

static float JiaoBi(params float[] arr)
{
    if (arr.Length == 0) 
    { 
	    Console.WriteLine("数组值为空值"); 
	    return -1; 
    }
    float max = arr[0];
    for (int i = 0; i < arr.Length; i++)
        if (max < arr[i]) 
	        max = arr[i];
    return max;
}

static double JiaoBi(params double[] arr)
{

    }

        if (max < arr[i]) 
        max = arr[i];

}
```
