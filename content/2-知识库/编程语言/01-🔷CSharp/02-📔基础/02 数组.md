## 核心理解

> 数组 = 顺序存储的同一类型数据集合，长度固定，通过下标访问。

- 类比：一排有编号的储物柜，只能放同一种东西，数量固定，按编号取。

---

## 声明数组

```csharp
// 方式1：只声明，未初始化
int[] arr;

// 方式2：指定长度，元素默认值（int默认0，bool默认false，string默认null）
int[] arr = new int[5];

// 方式3：指定长度 + 初始值
int[] arr = new int[3] { 10, 20, 30 };

// 方式4：由{}内容决定长度
int[] arr = new int[] { 10, 20, 30 };

// 方式5：最简写法
int[] arr = { 10, 20, 30 };
```

**注意**：
- `{}` 中的值必须与数组类型一致
- 数组初始化后长度不可变
- 数组初始化以后是不能够直接添加、删除新元素的
- 所有变量类型都可以申明为数组

---

## 使用数组

```csharp
int[] arr = { 5, 3, 8, 1, 9 };

// 获取长度
int len = arr.Length;   // 5

// 获取元素（下标从 0 开始，最大 Length-1）
int first = arr[0];     // 5
int last  = arr[4];     // 9

// 修改元素
arr[0] = 99;

// 遍历
for (int i = 0; i < arr.Length; i++)
    Console.WriteLine(arr[i]);
```

**注意：** 
- 数组中的下标和索引都是从0开始的
- 数组的范围是 `Lenght-1`
- 不能越界，会**报错**

## 增加 / 删除 / 查找元素

^f721d1

> 数组长度固定，无法真正增删，只能新建更大/更小的数组来替代。

```csharp
// 增加（新建更大数组，把旧数据复制过去）
int[] oldArr = { 1, 2, 3 };
int[] newArr = new int[oldArr.Length + 1];
for (int i = 0; i < oldArr.Length; i++)
    newArr[i] = oldArr[i];
newArr[newArr.Length - 1] = 99;  // 在末尾加入新元素
oldArr = newArr;

// 删除（新建更小数组，跳过要删除的那个）
int delIndex = 1;  // 删除下标为1的元素
int[] result = new int[oldArr.Length - 1];
int index = 0;
for (int i = 0; i < oldArr.Length; i++)
{
    if (i == delIndex) 
	    continue;
    result[index++] = oldArr[i];
}

// 查找（遍历比较）
int target = 8;
for (int i = 0; i < arr.Length; i++)
{
    if (arr[i] == target)
    {
        Console.WriteLine("找到了，下标是 " + i);
        break;
    }
}
```

---

## 注意事项

- 下标从 0 开始，最大值是 `Length - 1`，越界会抛出异常
- 所有变量类型都可以声明为数组（包括结构体、枚举）
- 数组是**引用类型**，赋值是传递引用，不是复制内容

> 3. 随机（0~100）生成1个长度为10的整数数组

```csharp
Random r = new Random();
int[] c = new int[10];
for (int i = 0; i < c.Length; i++)
```

> 4. 从一个整数数组中找出最大值、最小值、总合、平均值（可以使用随机数1~100）

```csharp
int sub = 0, avg = 0, max = 0, min = 0;
Random r = new Random();
int[] c = new int[10];
for (int i = 0; i < c.Length; i++)
{
    c[i] = r.Next(1, 101);
    Console.WriteLine(c[i]);
    sub += c[i];
    if (i == 0) { max = c[i]; min = c[i]; }
    else
    {
        if (c[i] > max) max = c[i];
        if (c[i] < min) min = c[i];
    }
}
avg = sub / c.Length;
Console.WriteLine("总：{0}，平均：{1}，最大：{2}，最小：{3}", sub, avg, max, min);
```

> 5. 交换数组中的第一个和最后一个、第二个和倒数第二个，依次类推，把数组进行反转并打印

```csharp
int[] a = new int[10];
for (int i = 0; i < a.Length; i++) { a[i] = i; Console.WriteLine(a[i]); }

int temp = 0;
for (int i = 0; i < a.Length / 2; i++)
{
    temp = a[i];
    a[i] = a[a.Length - 1 - i];
    a[a.Length - 1 - i] = temp;
}
for (int i = 0; i < a.Length; i++) Console.WriteLine(a[i]);
```

> 6. 将一个整数数组的每一个元素进行如下的处理：  

- 如果元素是正数则将这个位置的元素值加1
- 如果元素是负数则将这个位置的元素值减1  
- 如果元素是0，则不变

```csharp
Random r = new Random();
int[] d = new int[10];
for (int i = 0; i < d.Length; i++) { d[i] = r.Next(-10, 11); Console.WriteLine(d[i]); }

for (int i = 0; i < d.Length; i++)
{
    if (d[i] > 0) d[i] += 1;
    if (d[i] < 0) d[i] -= 1;
    Console.WriteLine(d[i]);
}
```

> 7. 定义一个有10个元素的数组，使用for循环，输入10名同学的数学成绩，将成绩依次存入数组，然后分别求出最高分和最低分，并且求出10名同学的数学平均成绩

```csharp
int[] e = new int[10];
int sub = 0, avg = 0, max = 0, min = 0;
try
{
    for (int i = 0; i < e.Length; i++)
    {
        Console.WriteLine("请依次输入数学成绩");
        e[i] = int.Parse(Console.ReadLine());
        sub += e[i];
        if (i == 0) { max = e[i]; min = e[i]; }
        else
        {
            if (e[i] > max) max = e[i];
            if (e[i] < min) min = e[i];
        }
    }
    avg = sub / e.Length;
    Console.WriteLine("总分是：{0}，平均分是：{1}，最高分是：{2}，最低分是：{3}",
        sub, avg, max, min);
}
catch { Console.WriteLine("请输入正确的成绩"); }
```

> 8. 请声明一个string类型的数组（长度为25）（该数组中存储着符号），通过遍历数组的方式取出其中存储的符号打印出以下效果
