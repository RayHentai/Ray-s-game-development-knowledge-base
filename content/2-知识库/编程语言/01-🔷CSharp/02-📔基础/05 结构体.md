## 核心理解

>结构体 = 任意变量的数据集合，可以自定义
>把一组相关的变量和方法打包在一起，用来表示一个有意义的数据集合。

-  类比：学生档案，把姓名、年龄、班级等信息捆绑成一个整体。

---

## 声明结构体

**规则：**
- 声明位置 `namespace`
- 关键字 `struct`
- 命名规范 `帕斯卡命名法`

**注意：**
- 结构体中的字段**不能直接初始化**（如 `public int age = 0;` 会报错），只能在构造函数或外部赋值
- 字段类型可以是任意类型（包括其他结构体、枚举、数组），但**不能是自身类型**
- 结构体中的方法**不需要加 `static`**
- 如果声明了构造函数，**必须在其中对所有字段赋值**

```csharp
struct Student
{
    // 1. 字段（变量）
    public string name;
    public int    age;
    public bool   isMale;

    // 2. 构造函数（可选）
    //    - 没有返回值
    //    - 函数名必须和结构体名相同
    //    - 必须有参数
    //    - 必须对所有字段初始化
    //    - 可以重载
    public Student(string name, int age, bool isMale)
    {
        this.name   = name;   // this 区分"结构体内部的 name"和"参数 name"
        this.age    = age;
        this.isMale = isMale;
    }

    // 3. 方法
    public void PrintInfo()
    {
        Console.WriteLine("姓名：{0}，年龄：{1}，性别：{2}",
            name, age, isMale ? "男" : "女");
    }
}
```

### this关键字

>代表这个变量是结构体内部的变量，不是外部传入的变量

---

### 访问修饰符

| 修饰符 | 含义 |
|--------|------|
| `public` | 公共的，可以在结构体外部访问 |
| `private` | 私有的，只能在结构体内部访问 |

> 结构体中的变量和方法**默认是 private**，外部访问不到。
> 需要外部访问的字段和方法都要加 `public`。

---

## 使用结构体

```csharp
// 方式1：直接声明（所有字段需在外部手动赋值后使用）
Student s;
s.name = "张三";
s.age  = 18;

// 方式2：用构造函数初始化（推荐）
Student s1 = new Student("张三", 18, true);
Student s2 = new Student("李四", 17, false);

// 调用方法
s1.PrintInfo();
s2.PrintInfo();
```

> 3. 使用结构体描述矩形的信息，长，宽；创建一个矩形，对其长宽进行初始化，并打印矩形的长、宽、面积、周长等信息

```csharp
struct JvXing
{
    public int length;
    public int width;

    {
        this.length  = length;

    }
}

JvXing j1 = new JvXing(4, 5);
j1.JiSuan();
```

> 4. 使用结构体描述玩家信息，玩家名字，玩家职业
>请用户输入玩家姓名，选择玩家职业，最后打印玩家的攻击信息职业：

- 战士(技能：冲锋)
- 猎人(技能：假死) 
- 法师 (技能：奥术冲击)

>打印结果：`猎人唐老狮释放了假死`

```csharp
enum E_JiNeng { ZhanShi, LieRen, FaShi }

struct Player
{
    public string name;
    public string zhiYe;
    string jiNeng;   // private，外部访问不到

    public Player(string name, E_JiNeng e)
    {
        this.name = name;
        switch (e)
        {
            case E_JiNeng.ZhanShi: zhiYe = "战士"; jiNeng = "冲锋";    
	            break;
            case E_JiNeng.LieRen:  zhiYe = "猎人"; jiNeng = "假死";    
	            break;
            case E_JiNeng.FaShi:   zhiYe = "法师"; jiNeng = "奥术冲击"; 
	            break;
            default:               zhiYe = "";     jiNeng = "";         
	            break;
        }
    }

    public void GongJi()
    {
        Console.WriteLine("{0}{1}释放了{2}", zhiYe, name, jiNeng);
    }
}

try
{
    Console.WriteLine("请输入姓名");
    string name = Console.ReadLine();
    Console.WriteLine("请输入职业（战士0 猎人1 法师2）");
    E_JiNeng e = (E_JiNeng)int.Parse(Console.ReadLine());
    Player p1 = new Player(name, e);
    p1.GongJi();
}
catch { Console.WriteLine("请输入正确的数字"); }
```

> 5. **使用结构体描述小怪兽**

- 定义一个数组存储10个上面描述的小怪兽
- 每个小怪兽的名字为（小怪兽+数组下标)

>举例：小怪兽0，最后打印10个小怪兽的名字+攻击力数值

```csharp
struct Monster
{
    public string name;
    public int    atk;

    public Monster(string name)
    {
        this.name = name;
        atk = new Random().Next(1, 20);
    }

    public void Info() { Console.WriteLine("{0} 攻击力{1}", name, atk); }
}

Monster m = new Monster("牛子怪");
m.Info();

Monster[] m1 = new Monster[10];
for (int i = 0; i < m1.Length; i++)
{
    m1[i] = new Monster("大牛子怪" + i);
    m1[i].Info();
}
```

> 6. 应用已学过的知识，实现奥特曼打小怪兽提示：

- 结构体描述奥特曼与小怪兽
- 定义一个方法实现奥特曼攻击小怪兽
- 定义一个方法实现小怪兽攻击奥特曼

```csharp
struct OutMan
{
    public string name;
    public int atk, def, hp;

    public OutMan(string name) 
    { 
	    this.name = name; 
	    atk = 20; 
	    def = 10; 
	    hp = 100; 
    }

    public void Atk(ref Boss boss)
    {
        boss.hp -= atk - boss.def;
        Console.WriteLine("{0}对{1}造成了{2}点伤害，{3}剩余{4}血量",
            name, boss.name, atk - boss.def, boss.name, boss.hp);
    }
}

struct Boss
{
    public string name;
    public int atk, def, hp;

    public Boss(string name) 
    { 
	    this.name = name; 
	    atk = 20; def = 10; 
	    hp = 100; 
    }

    public void Atk(ref OutMan outman)
    {
        outman.hp -= atk - outman.def;
        Console.WriteLine("{0}对{1}造成了{2}点伤害，{3}剩余{4}血量", name, outman.name, atk - outman.def, outman.name, outman.hp);
    }
}

OutMan outman = new OutMan("奥特曼");
Boss boss = new Boss("牛子王");
while (true)
{
    outman.Atk(ref boss);
    if (boss.hp <= 0) 
	    Console.WriteLine("奥特曼胜利"); 
    break;
    boss.Atk(ref outman);
    if (outman.hp <= 0) 
	    { Console.WriteLine("牛子王胜利"); 
	break;
    Console.WriteLine("按任意键继续"); Console.ReadKey(true);
}
```
