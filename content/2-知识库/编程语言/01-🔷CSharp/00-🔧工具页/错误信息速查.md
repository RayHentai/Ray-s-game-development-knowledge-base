# 🚨 错误信息速查

> 遇到报错直接在这里查，找到原因和解决方向。
> 格式：错误信息 → 中文含义 → 常见原因 → 解决思路

---

## 编译错误（红色，代码无法运行）

### CS0019 · Operator cannot be applied
```
Operator '+' cannot be applied to operands of type 'string' and 'int'
```
**含义：** 运算符不能用于这两种类型
**原因：** 想把 string 和 int 直接相加
**解决：** 先转换类型，用 `.ToString()` 或 `int.Parse()`
**关联：** [[09 类型转换]]

---

### CS0029 · Cannot implicitly convert
```
Cannot implicitly convert type 'float' to 'int'
```
**含义：** 不能隐式转换
**原因：** 精度高的类型赋值给精度低的类型（如 float → int）
**解决：** 加强制转换 `(int)floatValue` 或用 `Convert.ToInt32()`
**关联：** [[09 类型转换]]

---

### CS0103 · The name does not exist in the current context
```
The name 'xxx' does not exist in the current context
```
**含义：** 变量/方法名不存在
**原因：** 拼写错误 / 变量作用域问题（在 if 块里声明的变量，在外面用）/ 忘记引用命名空间
**解决：** 检查拼写 → 检查作用域 → 检查 using
**关联：** [[07 变量]] | [[23 命名空间]]

---

### CS0117 · Does not contain a definition
```
'string' does not contain a definition for 'xxx'
```
**含义：** 该类型没有这个方法/属性
**原因：** 方法名拼错 / 用了不属于这个类型的方法
**解决：** 检查拼写 → 查文档确认该类型有哪些方法
**关联：** [[25 string常用方法]]

---

### CS0120 · An object reference is required
```
An object reference is required for the non-static field/method
```
**含义：** 在静态方法里访问了非静态成员
**原因：** 静态函数不知道是"哪个对象"的非静态成员
**解决：** 要么把成员改为 static，要么先实例化对象再访问
**关联：** [[09 静态成员]]

---

### CS0121 · Ambiguous call
```
The call is ambiguous between the following methods
```
**含义：** 方法调用有歧义，不知道该用哪个重载
**原因：** 写了两个参数类型可以互相隐式转换的重载函数
**解决：** 传参时显式指定类型，或修改重载设计
**关联：** [[10 函数重载]]

---

### CS0161 · Not all code paths return a value
```
Not all code paths return a value
```
**含义：** 有返回值的函数，某些分支没有 return
**原因：** if-else 里漏写了 return / switch 没有 default
**解决：** 在函数末尾加兜底 return
**关联：** [[07 函数基础]]

---

### CS0165 · Use of unassigned local variable
```
Use of unassigned local variable 'xxx'
```
**含义：** 使用了未赋值的局部变量
**原因：** 只声明没赋值就直接用
**解决：** 声明时赋初始值
**关联：** [[07 变量]]

---

### CS0200 · Property or indexer cannot be assigned
```
Property or indexer 'xxx' cannot be assigned to -- it is read only
```
**含义：** 只读属性/索引器，不能赋值
**原因：** 该属性只有 get 没有 set
**解决：** 确认是否需要加 set，或者不要对它赋值
**关联：** [[07 成员属性]]

---

### CS0246 · The type or namespace name could not be found
```
The type or namespace name 'List' could not be found
```
**含义：** 找不到该类型或命名空间
**原因：** 忘记写 using / 拼写错误
**解决：** 检查顶部 using，如 `using System.Collections.Generic;`
**关联：** [[23 命名空间]] | [[07 List]]

---

### CS0305 · Using generic type requires type arguments
```
Using the generic type 'List<T>' requires 1 type arguments
```
**含义：** 泛型类没有指定类型参数
**原因：** 写了 `List list` 而不是 `List<int> list`
**解决：** 加上具体类型 `List<T>`
**关联：** [[05 泛型]]

---

### CS0501 · Must declare a body
```
'xxx' must declare a body because it is not marked abstract
```
**含义：** 非抽象方法没有方法体
**原因：** 普通方法写成了没有大括号的样子（只有签名）
**解决：** 加上方法体 `{ }` 或加 `abstract` 关键字
**关联：** [[20 抽象类与抽象方法]]

---

### CS0534 · Does not implement inherited abstract member
```
'Dog' does not implement inherited abstract member 'Animal.Speak()'
```
**含义：** 继承了抽象类，但没有实现所有抽象方法
**解决：** 为缺少的抽象方法补上 `override` 实现
**关联：** [[20 抽象类与抽象方法]]

---

### CS1061 · Does not contain a definition and no extension method
```
Type does not contain a definition for 'xxx' and no accessible extension method
```
**含义：** 找不到这个方法（包括拓展方法也没有）
**原因：** 方法名拼错 / 拓展方法所在的命名空间没有引用
**解决：** 检查拼写 → 检查 using
**关联：** [[11 拓展方法]]

---

### CS1503 · Argument cannot convert
```
Argument 1: cannot convert from 'int' to 'string'
```
**含义：** 传参类型不匹配
**原因：** 函数期望 string，你传了 int
**解决：** 转换后再传，用 `.ToString()` 或相应转换
**关联：** [[09 类型转换]]

---

## 运行时错误（代码能运行，执行时崩溃）

### NullReferenceException
```
NullReferenceException: Object reference not set to an instance of an object
```
**含义：** 对 null 调用了方法或访问了属性
**常见原因：**
- 声明了对象变量但忘记 `new`
- 从函数返回了 null，没有判断就直接用
- Unity 中组件没挂载，`GetComponent<>()` 返回 null
**解决：** 在使用前判空 `if (obj != null)` 或用 `?.` 操作符
**关联：** [[02 类与对象]] | [[23 特殊语法补充]]

---

### IndexOutOfRangeException
```
IndexOutOfRangeException: Index was outside the bounds of the array
```
**含义：** 数组下标越界
**常见原因：** 循环条件写错（用 `<=` 而不是 `<`）/ 下标从 1 开始数
**解决：** 检查循环终止条件，确认下标范围 `0 ~ Length-1`
**关联：** [[02 数组]]

---

### InvalidCastException
```
InvalidCastException: Specified cast is not valid
```
**含义：** 强制类型转换失败
**常见原因：** 用 `(int)obj` 拆箱时类型不匹配
**解决：** 转换前用 `is` 判断类型，或用 `as` 代替（失败返回 null）
**关联：** [[15 里氏替换原则与is as]] | [[17 万物之父object与装箱拆箱]]

---

### StackOverflowException
```
StackOverflowException
```
**含义：** 调用栈溢出
**常见原因：** 递归函数没有终止条件，无限递归
**解决：** 检查递归函数的 base case（终止条件）是否正确
**关联：** [[11 递归函数]]

---

### KeyNotFoundException
```
KeyNotFoundException: The given key was not present in the dictionary
```
**含义：** Dictionary 中不存在该键
**常见原因：** 直接用 `dic[key]` 访问不存在的键
**解决：** 先用 `ContainsKey(key)` 判断，或用 `TryGetValue()`
**关联：** [[08 Dictionary]]

---

### ArgumentOutOfRangeException
```
ArgumentOutOfRangeException: Index was out of range
```
**含义：** 参数超出有效范围
**常见原因：** List 下标越界 / Substring 参数超过字符串长度
**解决：** 访问前检查 `Count` / `Length`
**关联：** [[07 List]] | [[25 string常用方法]]

---

### DivideByZeroException
```
DivideByZeroException: Attempted to divide by zero
```
**含义：** 除以了零
**解决：** 除法前判断除数不为零
**关联：** [[11 运算符]]

---

### FormatException
```
FormatException: Input string was not in a correct format
```
**含义：** 字符串格式不对，无法转换为目标类型
**常见原因：** `int.Parse("abc")` 或 `int.Parse("")`
**解决：** 用 `int.TryParse()` 代替 `int.Parse()`，或加 try-catch
**关联：** [[10 异常捕获]] | [[09 类型转换]]
