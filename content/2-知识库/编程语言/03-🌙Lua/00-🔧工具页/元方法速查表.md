# 🪝 元方法速查表

> 快速查某个元方法的触发时机、参数含义和用法模板。
> 元方法统一定义在元表中，通过 `setmetatable(子表, 元表)` 绑定生效。

---

## 元表操作函数

| 函数 | 作用 | 说明 |
|------|------|------|
| `setmetatable(t, mt)` | 设置表 `t` 的元表为 `mt` | 返回 `t` 本身 |
| `getmetatable(t)` | 获取表 `t` 的元表 | 若无元表返回 nil |
| `rawget(t, k)` | 直接读取 `t[k]`，**绕开** `__index` | 只查自身，不触发元方法 |
| `rawset(t, k, v)` | 直接写入 `t[k] = v`，**绕开** `__newindex` | 只写自身，不触发元方法 |

---

## 访问类元方法

### `__index`

**触发时机：** 访问表中**不存在**的键时触发。

| 赋值形式 | 行为 |
|----------|------|
| `__index = 另一张表` | 去那张表里查找该键 |
| `__index = function(t, k)` | 执行函数，`t` 是子表，`k` 是被访问的键 |

```lua
meta.__index = { age = 18 }          -- 形式一：转发到表
meta.__index = function(t, k)        -- 形式二：自定义逻辑
    return "默认值"
end
setmetatable(myTable, meta)
print(myTable.age)  -- 触发 __index
```

> 支持**套娃**：子表 → 元表 → 元表的元表，依次向上查找。

**关联：** [[12 元表]] | [[13 面向对象]]

---

### `__newindex`

**触发时机：** 向表中**不存在**的键赋值时触发。

| 赋值形式 | 行为 |
|----------|------|
| `__newindex = 另一张表` | 把值写到那张表里（子表自身不变）|
| `__newindex = function(t, k, v)` | 执行函数，`t` 子表、`k` 键、`v` 值 |

```lua
meta.__newindex = {}                        -- 形式一：写入代理表
meta.__newindex = function(t, k, v)         -- 形式二：自定义逻辑
    rawset(t, k, v)  -- 若想写入自身，必须用 rawset
end
```

> 若要在 `__newindex` 内写入子表自身，必须用 `rawset`，否则死循环。

**关联：** [[12 元表]]

## 行为类元方法

### `__tostring`

**触发时机：** `tostring(表)` 或 `print(表)` 时触发。

```lua
meta.__tostring = function(t)
    return t.name  -- 第一个参数是子表自身
end
setmetatable(myTable, meta)
print(myTable)  -- 输出 myTable.name 的值
```

**关联：** [[12 元表]]

---

### `__call`

**触发时机：** 将表当作**函数调用**时触发。

```lua
meta.__call = function(t, a, b)
    -- t 是调用者（子表自身），a、b 是传入的参数
    print(t.name, a, b)
end
setmetatable(myTable, meta)
myTable(1, 2)  -- 触发 __call
```

**关联：** [[12 元表]]

## 运算符重载

> ⚠️ 使用比较运算符（`==` `<` `<=`）时，**两个操作数的元表必须一致**，否则不触发重载。

### 算术运算符

| 元方法 | 对应运算符 | 参数 |
|--------|-----------|------|
| `__add` | `+` | `(t1, t2)` |
| `__sub` | `-`（减法）| `(t1, t2)` |
| `__mul` | `*` | `(t1, t2)` |
| `__div` | `/` | `(t1, t2)` |
| `__mod` | `%` | `(t1, t2)` |
| `__pow` | `^` | `(t1, t2)` |
| `__unm` | `-`（取负，一元）| `(t)` |

```lua
meta.__add = function(t1, t2)
    return t1.value + t2.value
end
```

---

### 拼接运算符

| 元方法 | 对应运算符 | 参数 |
|--------|-----------|------|
| `__concat` | `..` | `(t1, t2)` |

```lua
meta.__concat = function(t1, t2)
    return t1.name .. t2.name
end
```

---

### 长度运算符

| 元方法 | 对应运算符 | 参数 |
|--------|-----------|------|
| `__len` | `#` | `(t)` |

```lua
meta.__len = function(t)
    return t.count
end
```

---

### 比较运算符

| 元方法 | 对应运算符 | 参数 | 注意 |
|--------|-----------|------|------|
| `__eq` | `==` | `(t1, t2)` | 两表元表须一致 |
| `__lt` | `<` | `(t1, t2)` | 两表元表须一致 |
| `__le` | `<=` | `(t1, t2)` | 两表元表须一致 |

```lua
meta.__eq = function(t1, t2)
    return t1.id == t2.id
end
-- ⚠️ myTable4 和 myTable5 必须设置的是同一个元表
setmetatable(myTable4, meta)
setmetatable(myTable5, meta)
print(myTable4 == myTable5)  -- 触发 __eq
```

## 完整模板

```lua
local meta = {
    -- 访问
    __index    = function(t, k) return nil end,
    __newindex = function(t, k, v) rawset(t, k, v) end,
    -- 行为
    __tostring = function(t) return "表" end,
    __call     = function(t, ...) end,
    -- 算术
    __add      = function(t1, t2) end,
    __sub      = function(t1, t2) end,
    __mul      = function(t1, t2) end,
    __div      = function(t1, t2) end,
    __mod      = function(t1, t2) end,
    __pow      = function(t1, t2) end,
    __unm      = function(t) end,
    -- 拼接 & 长度
    __concat   = function(t1, t2) end,
    __len      = function(t) end,
    -- 比较（两表须同一元表）
    __eq       = function(t1, t2) end,
    __lt       = function(t1, t2) end,
    __le       = function(t1, t2) end,
}

local myTable = {}
setmetatable(myTable, meta)
```
