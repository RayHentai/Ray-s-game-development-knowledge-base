# ⚖️ 语法对比速查（Lua vs C#）

> 专门收录"长得像但不是同一回事"的语法，以及 Lua 独有的坑。
> 从 C# 转过来学 Lua 时，这里就是排雷手册。

---

## 变量与类型

### 变量声明

| | Lua | C# |
|-|-----|----|
| 全局变量 | `a = 1`（默认即全局） | ❌ 无全局变量概念 |
| 局部变量 | `local a = 1` | `var a = 1` / `int a = 1` |
| 类型推断 | 自动（无需声明类型） | `var`（编译期推断）|
| 未声明变量 | 返回 `nil`，不报错 | 编译报错 |

**关联：** [[02 变量]] | [[09 多Lua脚本执行]]

---

### null 与 nil

| | Lua | C# |
|-|-----|----|
| 空值关键字 | `nil` | `null` |
| 适用范围 | 所有类型（变量赋 nil 即删除） | 引用类型 |
| 未声明变量的默认值 | `nil` | 编译报错（局部变量必须初始化）|

---

### 真假判断

| | Lua | C# |
|-|-----|----|
| 为假的值 | **只有** `nil` 和 `false` | `false` / `null` |
| 数字 `0` | ✅ **是真** | ❌ bool 上下文不适用 |
| 空字符串 `""` | ✅ **是真** | ❌ bool 上下文不适用 |

> ⚠️ Lua 中 `0` 和 `""` 都是真，这是最容易踩的坑之一。

**关联：** [[10 特殊用法]]

---

### 数据类型对比

| Lua 类型 | C# 对应 | 说明 |
|----------|---------|------|
| `nil` | `null` | 空值 |
| `number` | `int` / `float` / `double` | 统一为一种数值类型 |
| `string` | `string` | `''` 和 `""` 均为字符串 |
| `boolean` | `bool` | |
| `function` | `delegate` / `Func` / `Action` | 函数是一等公民 |
| `table` | `Array` / `List` / `Dictionary` / `class` | 万能复杂类型 |
| `userdata` | 无直接对应 | 宿主语言数据（如 C 结构体）|
| `thread` | `Coroutine`（Unity）| 协程对象 |
| char 类型 | ❌ **没有** | `''` 在 Lua 里也是 string |

**关联：** [[02 变量]]

## 运算符

### 比较运算符

| 操作 | Lua | C# |
|------|-----|----|
| 不等于 | `~=` ⚠️ | `!=` |
| 等于 | `==` | `==` |
| 大于/小于/大于等于/小于等于 | 相同 | 相同 |

---

### 逻辑运算符

| 操作 | Lua | C# |
|------|-----|----|
| 与 | `and` | `&&` |
| 或 | `or` | `\|\|` |
| 非 | `not` | `!` |

> ⚠️ `and` / `or` 在 Lua 中不只返回 bool，而是返回操作数之一（短路值）。

**关联：** [[04 运算符]] | [[10 特殊用法]]

---

### 算术运算符差异

| 操作 | Lua | C# |
|------|-----|----|
| 幂运算 | `^`（不是异或！） | `Math.Pow()` |
| 异或 | ❌ 不支持位运算 | `^` |
| 自增 | ❌ `i = i + 1` | `i++` / `++i` |
| 复合赋值 | ❌ `a = a + 1` | `a += 1` |
| 位运算 | ❌ 需自己实现 | `&` `\|` `^` `<<` `>>` |

---

### 三目运算符

| | Lua | C# |
|-|-----|----|
| 三目 | ❌ 用 `条件 and x or y` 模拟 | `条件 ? x : y` |

```lua
-- Lua 模拟三目
local res = (x > y) and x or y
```

> ⚠️ 当 x 本身是 `false` 或 `nil` 时，模拟三目会失效，需注意。

**关联：** [[10 特殊用法]]

---

### 字符串拼接

| | Lua | C# |
|-|-----|----|
| 拼接运算符 | `..` | `+` |
| 格式化 | `string.format("%d", n)` | `$"{n}"` / `string.Format()` |

## 控制流

### 条件分支

| | Lua | C# |
|-|-----|----|
| 基本结构 | `if 条件 then ... end` | `if (条件) { }` |
| else if | `elseif`（⚠️ 连在一起，不能有空格）| `else if` |
| switch | ❌ 不支持 | `switch / case / default` |

```lua
-- Lua
if a > 0 then
    print("正")
elseif a == 0 then  -- ⚠️ 不是 else if
    print("零")
else
    print("负")
end
```

**关联：** [[05 条件分支语句]]

---

### 循环

| | Lua | C# |
|-|-----|----|
| while | `while 条件 do ... end` | `while (条件) { }` |
| do-while | `repeat ... until 结束条件` ⚠️ | `do { } while (继续条件)` |
| for（计数）| `for i=1,5 do ... end` | `for (int i=0; i<5; i++)` |
| for（步进）| `for i=5,1,-1 do` | `for (int i=5; i>0; i--)` |
| for（遍历）| `for k,v in pairs(t)` | `foreach (var item in list)` |
| break | ✅ 支持 | ✅ 支持 |
| continue | ❌ 不支持 | ✅ 支持 |

> ⚠️ `repeat...until` 的条件是**结束条件**（为 true 时停止），和 C# 的 `do-while` 逻辑相反。

> ⚠️ Lua 的 for 循环**包含结束值**（`for i=1,5` 会执行到 i=5）。

**关联：** [[06 循环语句]]

## 函数

| | Lua | C# |
|-|-----|----|
| 声明方式 | `function 名() end` | `返回类型 名() { }` |
| 匿名函数 | `function() end` | `() => { }` / `delegate { }` |
| 多返回值 | ✅ 原生支持 `return a, b, c` | ❌ 需用 `out` / `Tuple` / 结构体 |
| 函数重载 | ❌ 不支持，后定义的覆盖前者 | ✅ 支持 |
| 变长参数 | `...`，用 `{...}` 转为表 | `params T[] args` |
| 闭包 | ✅ 支持（upvalue）| ✅ 支持 |
| 参数个数不匹配 | 不报错，多退少补 nil | 编译报错 |

**关联：** [[07 函数]]

## 表（Table）vs 数组 / 字典 / 类

### 数组

| | Lua | C# |
|-|-----|----|
| 索引起始 | **1** ⚠️ | **0** |
| 获取长度 | `#表` | `.Length` / `.Count` |
| 遍历 | `ipairs(t)`（顺序，遇 nil 停）| `foreach` / for |
| 中间有 nil | 会导致 `#` 结果不准确 | 无此问题 |

---

### 字典

| | Lua | C# |
|-|-----|----|
| 类型 | 用 table 模拟（字符串键）| `Dictionary<K,V>` |
| 访问 | `t["key"]` 或 `t.key` | `dict["key"]` |
| 遍历 | `pairs(t)` | `foreach (var kv in dict)` |
| 删除 | `t["key"] = nil` | `dict.Remove("key")` |

---

### 类 / 面向对象

| | Lua | C# |
|-|-----|----|
| 声明类 | 用 table 模拟 | `class 类名 { }` |
| 实例化 | 手动实现 `:new()` 方法 | `new 类名()` |
| 继承 | 手动实现 + `setmetatable` | `class B : A` |
| 多态 | 子表重写同名方法 | `override` 关键字 |
| this | `self`（`:` 语法糖自动传入）| `this` |
| 调用父类方法 | `self.base.方法(self)`（⚠️ 不能用 `:`）| `base.方法()` |
| 访问修饰符 | ❌ 没有（全靠约定）| `public` / `private` / `protected` |
| 函数重载 | ❌ 不支持 | ✅ 支持 |

**关联：** [[08 复杂数据类型 表]] | [[13 面向对象]]

## 其他语言特性

| 特性 | Lua | C# |
|------|-----|----|
| 多脚本加载 | `require("文件名")` | `using` / 项目引用 |
| 脚本卸载 | `package.loaded["名"] = nil` | ❌ 无此概念 |
| GC | 自动（可手动 `collectgarbage()`）| 自动（可手动 `GC.Collect()`）|
| 协程 | `coroutine` 库，原生支持 | `IEnumerator` + Unity 驱动 |
| 字符串长度 | `#str` | `str.Length` |
