## 核心要点

**1. lua中的数据类型**
- 简单变量类型 
	- nil
	- number 
	- string 
	- boolean
- 复杂数据类型 
	- function（函数）
	- 表table（表）
	- userdata（数据结构）
	- thread（线程，表现为协同程序）

**2. lua中所有的变量声明都不需要申明变量类型，他会自动的判断类型，类似C#里面的var**

**3. lua中使用没有声明过的变量不会报错 默认值是nil**

**4. lua里没有char `''` / `""` 都是字符串**

**5. type函数，可以得到变量的类型，返回值是string类型**

---

## 代码演示

```lua
--声明变量
a = nil --nil类似于C#中的null
print(type(a)) --null

a = 1 --所有的数值都是number
print(type(a)) --number

a = "1234"
print(type(a)) --string
a = '1234' 
print(type(a)) --string

a = true
a = false
print(type(a)) --boolean

print(type(type(a))) --string

print(b);--nil lua中使用没有声明过的变量不会报错 默认值是nil
```
