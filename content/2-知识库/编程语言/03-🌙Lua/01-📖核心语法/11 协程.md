## 核心要点

**1. 创建协程**
- `coroutine.create`(函数) 返回值是一个thread对象
- `coroutine.wrap`(函数) 返回值是一个函数

**2. 启动协程**
- `coroutine.resume`(协程对象) 默认返回值第一个为是否开启成功
- 直接调用函数（wrap创建的）

**3. 挂起协程**
- `coroutine.yield`(返回值)

**4. 协程的状态**
- `coroutine.status`(协程对象或者协程函数)  dead、suspended、running
- `coroutine.running`() 返回一个线程编号

## 语法 / 用法

```lua
--协同程序
--协程的创建
--方法一 coroutine.create
fun = function()
	print(123)
end
co = coroutine.create(fun)
co = coroutine.create(function()
	print(123)
end)
print(co)--thread
print(type(co))--thread

--方法二 coroutine.wrap
co = coroutine.wrap(fun)
co = coroutine.wrap(function()
	print(123)
end)
print(co) --function
print(type(co)) --function

--协程的运行
--方法一 coroutine.resume
fun = function()
	print(123)
end
co = coroutine.create(fun)
coroutine.resume(co)

--方法二
co = coroutine.wrap(fun)
co()

--协程的挂起
fun2 = function()
	local i = 1
	while true do
		print(i)
		i = i + 1
		coroutine.yield() --协程挂起函数
	end
end

--create
co = coroutine.create(fun2)
--coroutine.yield 会让协程暂时挂起 再次开启协程后从挂起的地方继续执行 直到再次被挂起
coroutine.resume(co)
coroutine.resume(co)
coroutine.resume(co)

--wrap
co = coroutine.wrap(fun2)
co()
co()
co()

--yield存在返回值
fun2 = function()
	local i = 1
	while true do
		print(i)
		i = i + 1
		coroutine.yield(i) --在这里填入参数 表示返回值 可以有多个
	end
end

--create
co = coroutine.create(fun2)
--第一个参数代表是否启动成功
--第二个参数代表tield里面的返回值
temp1, temp2 = coroutine.resume(co)
print(temp1, temp2)
temp1, temp2 = coroutine.resume(co)
print(temp1, temp2)
temp1, temp2 = coroutine.resume(co)
print(temp1, temp2)

--wrap
--这种方式只会有一个返回值 默认的第一个返回值就没有了
co = coroutine.wrap(fun2)
print("返回值："..co())
print("返回值："..co())
print("返回值："..co())

--协程的状态 coroutine.status(协程对象或协程函数)
--dead 结束
--suspended 暂停
--running 进行中
fun3 = function()
	print(coroutine.status(co)) --running
	print(coroutine.running()) --获取正在运行的线程号
end
co = coroutine.create(fun3)
coroutine.resume(co)
print(coroutine.status(co)) --dead
```
