## 核心要点

**1. elseif 中间不能打空格，会报错**
**2. 结构**
- if 条件 then 逻辑 end
- if 条件 then 逻辑 else 逻辑 end
- if 条件 then 逻辑 elseif 条件 逻辑 else 逻辑 end

---

## 语法 / 用法

```lua
--if 
--if 条件 then 逻辑 end
a = 10
if a < 5 then
	print("a小于5")
end

--双分支
if a < 5 then
	print("a小于5")
else
	print("a小于等于5")
end

--多分支
if a < 5 then
	print("a小于5")
elseif a == 6 then
	print("a等于6")
elseif a == 7 then
	print("a等于7")
elseif a == 8 then
	print("a等于8")
elseif a == 9 then
	print("a等于9")
else
	print("a有自己的想法");
end
```
