## 核心要点

**1. 封装**
- 表就是表现类的一种形式
- 新建一个万物之父Object作为所有对象子表的元表
- 实例化对象的方法
	- 本质上就是创建一个空表的方法
	- 设置万物之父的__index为自己
	- 把万物之父设置为新表的元表 return 创建的空表 
	- 外部调用方法就实现了实例化对象
- 修改创建出来的对象的属性变量时 是为这个空表对象新建了一个成员属性

**2. 继承**
- 继承的方法
	- 通过大G表，根据字符串在大G表中创建一个新表
	- 把新表的元表设置为万物之父
	- 设置万物之父的__index为自己
	- 返回这个新表

**3. 多态**
- 直接用子类中重写这个方法
- 如果想要保留父类逻辑，在继承的方法中构建一个base代表父类
- 通过base调用父类方法时一定不能用冒号调用，而是通过点调用然后传入自己，进入父类函数内部

## 封装、继承、多态

```lua
--封装
--面向对象 类 都是基于table实现的
Object = {}
Object.id = 1
function Object:Test()
	print(self.id)
end
--实例化对象的方法
--对象就是变量 返回一个新的变量
--返回出去的内容本质上就是 一个新的表对象
function Object:new()
	local obj = {}
	--把Object自己作为__index
	self.__index = self
	--把Object作为新对象的元表
	setmetatable(obj, self)
	return obj
end
local myObj = Object:new()
print(myObj)
print(myObj.id) --1
myObj.id = 2
myObj:Test() --2 注意 是myObj的id属性 不是Object的id属性 谁调用:传入的参数就是谁

--继承
--写一个用于继承的方法
function Object:subClass(className)
	_G[className] = {} --_G总表中 所有声明的全局变量都以键值对的形式存储在其中 
	local obj = _G[className]
	self.__index = self
	setmetatable(obj, self)
end
Object:subClass("Person")
local p1 = Person:new()
print(p1.id) --1
p1.id = 100
print(p1.id) --100
p1:Test() --100

--多态
--相同行为 不同表现
Object:subClass("GameObject")
GameObject.posX = 0
GameObject.posY = 0
function GameObject:Move()
	self.posX = self.posX + 1
	self.posY = self.posY + 1
	print(self.posX)
	print(self.posY)
end

GameObject:subClass("Player")
local player = Player:new()
player:Move() --1 1

function Player:Move()
	--在C#中 base代表父类 可以在继承的方法中手动构建一个父类
	self.base:Move()
end
player:Move() --1 1

local player1 = Player:new()
--注意！！ 这样写有坑 不同对象使用的成员变量 是相同的成员变量
player1:Move() --2 2

function Player:Move()
	--问题来自这里 当对象调用Move()时
	--这里的self.base代表Object
	--冒号调用代表把Object作为参数传进去调用函数 
	--相应的 打印出来的结果是Object成员属性执行方法之后的结果
	--而不是对象自己的属性执行之后的结果
	self.base:Move()
end

function Player:Move()
	--所以应该改用点调用 把对象自己作为参数传进去
	self.base.Move(self)
end
player.posX = 10
player.posY = 100
player:Move() --11 101
player1.posX = 9
player1.posY = 99
player1:Move() --10 100
```

## 完整实现

```lua
--面向对象
--万物之父 所以对象的基类
Object = {}

--实例化方法
function Object:new()
	--创建空对象
	local obj = {}
	--为空对象设置元表 以及元表__index
	self.__index = self
	setmetatable(obj, self)
	return obj
end

--继承方法
function Object:subClass(className)
	--根据名字生成表 也就是类
	_G[className] = {}
	local obj = _G[className]
	--设置自己的父类
	obj.base = self
	--为子表设置元表 以及__index
	self.__index = self
	setmetatable(obj, self)
end

Object:subClass("GameObject")
GameObject.posX = 0
GameObject.posY = 0
function GameObject:Move()
	self.posX = self.posX + 1
	self.posY = self.posY + 1
end

local gameObject = GameObject:new()
print(gameObject.posX)
gameObject:Move()
print(gameObject.posX)

GameObject:subClass("Player")
function Player:Move()
	self.base.Move(self)
end
local player1 = Player:new()
local player2 = Player:new()
print(player1.posX)
player1:Move()
print(player1.posX)
print(player2.posX)
player2:Move()
print(player2.posX)
```
