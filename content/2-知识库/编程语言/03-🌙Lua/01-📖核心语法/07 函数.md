## 核心要点

**1. lua是从上到下执行的 执行函数不能写在声明函数的上面**

**2. 声明 直接命名声明 或者不命名用一个变量存储**
- function 函数名() end
- 变量名 = 函数名() end

**3. 传递参数时 参数个数传多了或者传少了不会报错**
- 多了会丢弃
- 少了会补空

**4. 函数可以有多个返回值 外部用多个变量来接取 接多接少不会报错**
- 多了会丢弃
- 少了会补空

**5. 函数的类型是 function**

**6. lua不支持函数重载，调用的是最后声明的函数**

**7. 变长参数 传入`...` 先用一个表接住传入参数 再使用**
- function Func4( ... ) arg = {...} end

**8. 函数嵌套 可以在函数里面声明函数**
- 支持闭包：可以改变临时变量的声明周期

## 代码演示

```lua
--lua是从上到下执行的 执行函数不能写在声明函数的上面
--无参无返回值
--function 函数名() end
function Func1()
	print("函数")
end
Func1()

--变量名 = 函数名() end
--这种写法有点类似C#中的委托和事件
f = function()
	print("第二种写法")
end
f()

--有参
--function 函数名(变量) end
function Func2(a)
	print(a)
end
Func2(1) --1
Func2("1") --1
Func2(true) --true
Func2() --nil
Func2(1, 2) --1 如果传入的参数和参数个数不匹配 不会报错，只会补空或者丢弃 

--有返回值
--function 函数名(参数) return 返回值1, 返回值2, ... end
function Func3(a)
	return a, "123", true
end
temp1, temp2, temp3 = Func3(a) --多返回值 在前面声明多个变量接取即可
print(temp1)-- 如果接取的参数的数量和返回值数量不匹配 不会报错，只会补空或者丢弃
print(temp2)
print(temp3)

--函数类型
f1 = function(a)
	return a
end
print(type(f1)) --function

--函数的重载
--lua中不支持重载 调用的是最后声明的函数

--变长参数
function Func4( ... )
	arg = {...} --用一个表存起来再用
	for i = 1, #arg do
		print(arg[i])
	end
end
Func4(1, "123", true, 4, 5, 6) --1 123 true 4 5 6

--函数嵌套
function Func5()
	return function()
		print("内部函数")
	end
end
f3 = Func5()
f3()

--闭包
function Func6(x)
	return function(y)
		return x + y
	end
end
f4 = Func6(4)
print(f4(5)) --9
```
