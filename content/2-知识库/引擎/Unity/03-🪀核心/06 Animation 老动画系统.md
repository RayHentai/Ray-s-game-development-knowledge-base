## Animation 窗口

>Animation窗口就是动画窗口
>它主要用于在Unity内部创建和修改动画
>所有在场景中的对象都可以通过Animation窗口为其制作动画

**原理：**
- 制作动画时：记录在定时间点对象挂载的脚本的变量变化
- 播放动画时：将制作动画时记录的数据在固定时间点进行改变，产生动画效果

**关键词：**
1. **动画时间轴：**
   每一个动画文件都有自己的一个生命周期，从动画开始到结束//我们可以在动画时间轴上编辑每一个动画生命周期中变化

2. **动画中的帧：**
   假设某个动画的帧率为60帧每秒，意味着该动画1秒钟最多会有60次改变机会
   每一帧的间隔时间是1s/60≈16.67毫秒，也就是说我们最快可以每16.67毫秒改变一次对象状态

3. **关键帧：**
   动画在时间轴上的某一个时间节点上处于的状态

### 创建动画

**创建动画**
1. 在场景中选中想要创建动画的对象
2. 在Animation窗口中点击创建
3. 选择动画文件将要保存到的位置

**保存动画文件时，Unity会帮助我们完成以下操作**
1. 创建一个 `Animator Controller`（动画控制器或称之为动画状态机） 资源（新动画系统）
2. 将新创建的动画件添加到 `Animator Controller` 中
3. 为动画对象添加 `Animator` 组件
4. 为 `Animator `组件关联创建的 `Animator Controller` 文件

## 老动画系统相关

> 老动画系统主要用于处理老版本项目和简单的一些自制动画，新项目都是用新系统

**Unity中有两套动画系统**
- 新：Mecanim动画系统主要用Animator组件控制动画
- 老：Animation动画系统一主要用Animation组件控制动画（Unity4之前的版本可能会用到）

**目前我们为对象在Animation窗口创建的动画都会被新动画系统支配**，有特殊需求或者针对一些简易动画，才会使用老动画系统

**注意：**
- 创建动画之前为对象添加Animation组件之后再制作动画
- 这时制作出的动画和之前的动画格式存在区别

**窗口参数**
![[Animatio 脚本参数.png]]

**代码控制：**

```csharp
private Animation animation;
//得到脚本
animation = GetComponent<Animation>();
//1 播放动画
animation.Play("动画名");
//2 淡入播放 （在两个动画之间）自动产生过渡效果
//要播放的动画的开始状态和当前状态不一样时 会产生过渡效果
animation.CrossFade("动画名");
//3 前一个播放完 再播放下一个
animation.PlayQueued("动画名"); 
animation.CrossFadeQueued("动画名"); //淡入
//4 停止播放所有动画
animation.Stop();
//5 是否在播放某个动画
if (animation.IsPlaying("动画名"))
//6 播放模式设置
animation.wrapMode = WrapMode.枚举项;
//7 其他（了解）
//层级和权重以及混合（老动画系统需要通过代码来达到动画的遮罩、融合等效果）
//设置层级
animation["动画名"].layer = 1;
//设置权重
animation["动画名"].weight = 1;
//混合模式 叠加还是混合
animation["动画名"].blendMode = AnimationBlendMode.枚举项;
//设置混合相关的骨骼信息
animation["动画名"].AddMixingTransform(Transform);
```

**动画事件**

在时间轴中添加事件 -> 在脚本中新建函数 -> 在动画事件中选择函数（函数可传参数）
