## 刚体

>2D物理系统中的刚体和3D中的刚体基本是一样的
>最大的区别是对象只会在XY平面中移动
>并且只在垂直于该平面的轴上旋转

**刚体类型**
![[刚体类型 动态类型.png]]

![[刚体类型 运动学类型.png]]

![[刚体 静态类型.png]]

**如何选择不同类型的刚体**
- Dynamic 动态刚体：受力的作用，要动要碰撞的对象
- kinematic 运动学刚体：通过刚体API移动的对象，不受力的作用，但是想要进碰撞检测
- Static 静态刚体：不动不受力作用的静态物体，但是想要进碰撞检测

**API**
跟3D基本一致
```csharp
//[[20 刚体 Rigidbody#^fdf488]]
//加力
Rigidbody2D rigid = GetComponent<Rigidbody2D>();
rigid.AddForce(new Vector2(0, 100));
//速度
rigid.velocity = new Vector2(1, 0);
```

### 练习

> 1. 在之前练习题的基础上，请用2D刚体和碰撞器制作一个玩家，它可以在一个平台上移动跳跃。（注意：移动和跳跃都通过刚体的API进制作，类似3D物理系统中刚体加力和给速度

```csharp
public class Collider_MoveJump : MonoBehaviour
{
    private float h;
    private SpriteRenderer sr;
    private Rigidbody2D rb;
    private void Start()
    {
        sr = GetComponent<SpriteRenderer>();
        rb = GetComponent<Rigidbody2D>();
    }
    void Update()
    {
        h = Input.GetAxis("Horizontal");
        //transform.Translate(Vector3.right * 20 * h * Time.deltaTime);
        rb.velocity = new Vector2(10 * h, rb.velocity.y);//加速度
        if (h < 0)
            sr.flipX = true;
        else if (h > 0)
            sr.flipX = false;
        transform.Translate(Vector3.up * 20 * Input.GetAxis("Vertical") * Time.deltaTime);
        if (Input.GetKeyDown(KeyCode.Space))
        {
            GameObject obj = Instantiate(Resources.Load<GameObject>("Bullet"), transform.position + new Vector3(sr.flipX ? -0.3f : 0.3f, 0.4f, 0), transform.rotation);
            obj.GetComponent<Sprite_bullet>().ChangeDir(sr.flipX ? Vector3.left : Vector3.right);
        }
        if (Input.GetKeyDown(KeyCode.J))
        {
            rb.AddForce(Vector2.up * 300);
        }
    }
}
```

## 恒定力 Constant Force

> 恒定力是一个特殊的脚本
> 它可以给一个2D刚体持续添加一个力
> 在做一些随着时间推移而加速的对象时很适用，比如类似火箭发射等效果
> 恒定力脚本会线性的为对象添加力和扭矩力让其移动和旋转
