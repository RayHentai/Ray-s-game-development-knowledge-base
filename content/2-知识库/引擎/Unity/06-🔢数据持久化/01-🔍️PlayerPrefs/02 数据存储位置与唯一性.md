## 不同平台的存储位置

| 平台          | 存储位置                                                            |
| ----------- | --------------------------------------------------------------- |
| **Windows** | 注册表：`HKCU\Software\[公司名称]\[产品名称]`（公司和产品名称在 ProjectSettings 中设置） |
| **Android** | `/data/data/包名/shared_prefs/pkg-name.xml`                       |
| **iOS**     | `/Library/Preferences/[应用ID].plist`                             |
|             |                                                                 |

## PlayerPrefs 数据唯一性

> **PlayerPrefs 中不同数据的唯一性由 key 决定。**

- 不同的 `key` 决定了不同的数据
- 同一项目中如果不同数据 `key` 相同，**会造成数据丢失**
- 要保证数据不丢失，**就要建立一个保证 `key` 唯一的规则**

### key 唯一性设计示例

```csharp
// 以玩家唯一标识作为前缀，避免不同玩家数据互相覆盖
public void Save(string playerKey)
{
    PlayerPrefs.SetString(playerKey + "_name", name); // "p1_name"、"p2_name" 互不干扰
    PlayerPrefs.SetInt(playerKey + "_age", age);
    PlayerPrefs.SetInt(playerKey + "_atk", atk);
}

// 存储多个玩家时
p1.Save("p1");
p2.Save("p2");
```

public class PlayerInfo

    private string playerKey;

    {
        PlayerPrefs.SetString(playerKey + "_name", name);

    }

    //只要执行读取，Player对象就会有一个唯一的键

}

> 2、要在游戏中做一个排行榜功能
> 排行榜主要记录玩家名（可重复），玩家得分，玩家通关时间
> 请用PlayerPrefs存储读取排行榜相关信息

```csharp
public class RankInfoList
{
    public List<RankInfo> rankInfos;

    //写一个无参构造，外部new一个对象 就会自动调取Load 省得自己要手动new [[优化]] 
    public RankInfoList()
    {
        Load();
    }

    //新加排行榜信息 方便外部调用
    public void Add(string name, int score, int time)
    {
        rankInfos.Add(new RankInfo(name, score, time));
    }
    public void Save()
    {
        PlayerPrefs.SetInt("num",rankInfos.Count);
        for (int i = 0; i < rankInfos.Count; i++) 
        {
            PlayerPrefs.SetString("name", rankInfos[i].playerName);
            PlayerPrefs.SetInt("score", rankInfos[i].playerScore);
            PlayerPrefs.SetInt("time", rankInfos[i].playerTime);
        }
        PlayerPrefs.Save();
    }
    //优化 变成私有  
    private void Load()
    {
        int num = PlayerPrefs.GetInt("num", 0);
        rankInfos = new List<RankInfo>();
        for (int i = 0; i < num; i++)
        {
            RankInfo info = new RankInfo(PlayerPrefs.GetString("name"), 
                                         PlayerPrefs.GetInt("score"),
                                         PlayerPrefs.GetInt("time"));
            rankInfos.Add(info);
        }
    }
}

public class RankInfo
{
    public string playerName;
    public int playerScore;
    public int playerTime;

    public RankInfo(string name, int score, int time)
    {
        playerName = name;
        playerScore = score;
        playerTime = time;
    }
}

public class Rank : MonoBehaviour
{
    void Start()
    {
        RankInfoList list = new RankInfoList();
        print(list.rankInfos.Count);
        for (int i = 0; i < list.rankInfos.Count; i++)
        {
            print(list.rankInfos[i].playerName);
            print(list.rankInfos[i].playerScore);
            print(list.rankInfos[i].playerTime);
        }
        list.Add("Hentai", 9999999, 100);
        list.Save();
    }
}
```
