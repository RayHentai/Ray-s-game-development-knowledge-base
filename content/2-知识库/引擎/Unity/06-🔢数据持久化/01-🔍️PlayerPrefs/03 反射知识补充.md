## IsAssignableFrom

> **判断一个类型的对象是否可以让另一个类型为自己分配空间（父类装子类）。**

```csharp
class father { }
class son : father { }
type fatherType = typeof(Father)
type sonType = typeof(Son)

//判断一个类型的对象是否可以让另一个类型为自己分配空间
//调用者通过该方法进行判断判断是否可以通过传入的类型为自己分配空间
//简单说就是父类能不能装子类
if (fatherType.IsAssignableFrom(sonType))//返回值bool
{
	Father f = Activator.CreateInstance(sonType) as Father;
}

//应用案例：[[05 项目-PlayerPrefs 数据管理器#^6e2d0c]]
```

## 通过反射获取泛型类型

```csharp
List<int> list = new List<int>();
Type listType = list.GetType();

Dictionary<string, float> dic = new Dictionart<string, float>();
Type dicType = dic.GetType();

//通过反射获取泛型类型
type[] listTypes = listType.GetGenericArguments();
type[] dicTypes = dicType.GetGenericArguments();

listTypes[0] //返回泛型对应的Type
dicTypes[0] //返回键Type
dicTypes[1] //返回值Type

//应用案例：[[05 项目-PlayerPrefs 数据管理器#^6e2d0c]]
```

## 什么时候用

- 场景二：想得到泛型的`Type`

## 我的踩坑记录

- `IsAssignableFrom` 方向容易搞错：`A.IsAssignableFrom(B)` 是问"B 能不能赋值给 A"（B 是 A 的子类或 B 就是 A）
