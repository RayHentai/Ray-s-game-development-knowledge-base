## ✅ 自检清单

### PlayerPrefs
- [ ] Set 只写内存，必须 Save() 才写硬盘
- [ ] 三种类型：int / float / string，其他类型需要转换
- [ ] key 唯一性设计（前缀规则防止数据覆盖）
- [ ] 加密的三个层次：找不到 / 看不懂 / 解不出

### XML
- [ ] XML 文件结构：声明 / 根节点 / 子节点 / 属性
- [ ] C# 读取：`XmlDocument` 加载 → `SelectSingleNode` 查找 → 读取值
- [ ] C# 写入：创建节点 → 设置值 → 保存
- [ ] 序列化：`XmlSerializer` 直接把对象转 XML，反序列化逆向还原

### Json
- [ ] Json 格式：对象用 `{}`，数组用 `[]`，键值对用 `:`
- [ ] JsonUtility：Unity 内置，只支持有 `[Serializable]` 的类
- [ ] LitJson：第三方库，支持 Dictionary 和更复杂的结构
- [ ] JsonUtility vs LitJson 怎么选

### 二进制
- [ ] 各类型转字节：`BitConverter.GetBytes()` / `Encoding.UTF8.GetBytes()`
- [ ] 文件流：`FileStream` / `BinaryWriter` / `BinaryReader`
- [ ] `BinaryFormatter` 序列化对象（需要 `[Serializable]`）
- [ ] 异或加密：同一操作既可加密也可解密
