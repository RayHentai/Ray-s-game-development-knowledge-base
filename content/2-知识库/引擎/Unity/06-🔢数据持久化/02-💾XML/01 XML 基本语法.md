## 固定内容

**注意：** 要写在最前面

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!--version 代表版本 encoding 代表编码格式-->
<!--编码格式 就是读取文件时 解析字符串使用的编码
不同的字符 在内存中的二进制是不一样的 每一个字符对应一个数字
不同的编码格式 对应的二进制 是不一样的  -->
```

## 基本语法

**基本规则：**
1. 每个元素都必须有关闭标签 </元素名>
2. 元素命名规则基本遵照C#中变量名命名规则
3. XML标签对大小写敏感
4. XML文档必须有根元素
5. 特殊的符号应该用实体引用
	- `&lt` - <小于
	- `&gt` - >大于
	- `&amp` - &和号
	- `&apos` - ' 单引号
	- `&quot` - " 引号

```xml
<!--xml的基本语法就是 <元素标签> </元素标签> 配对出现-->
<!--必须要有一个根节点-->
<!--<节点名> 可以填写数据 或者包裹别的节点 </节点名>-->
<PlayerInfo>
	<name>name</name>
	<atk>10</atk>
	<ItemList>
		<Item>
			<id>1</id>
			<num>1</num>
		</Item>
		<Item>
			<id>2</id>
			<num>5</num>
		</Item>
		<Item>
			<id>3</id>
			<num>10</num>
		</Item>
	</ItemList>
</PlayerInfo>
```
