## 需求分析

菜单栏有一个工具窗口，里面有一个生成数据，点击生成数据可以把excel文件转存为二进制文件，并生成对应的数据结构类、数据容器

根据Excel表的结构，把数据转存为二进制文件
数据结构类，表示excel表中一行的数据
容器类，把一行行的数据以字典的方式存储起来，方便用一个键快速得到某一行的数据

## 二进制文件格式

> 写入和读取必须严格对称，理解这个格式是理解整个工具的关键

```
┌─────────────────────────────────────────┐
│  数据行数（4字节 int）                    │  = 总行数 - 4（去掉表头4行）
├─────────────────────────────────────────┤
│  主键字段名长度（4字节 int）              │
│  主键字段名（N字节 UTF8字符串）           │  例如 "id"
├─────────────────────────────────────────┤
│  第1行数据                               │
│    int字段：直接写4字节                   │
│    float字段：直接写4字节                 │
│    bool字段：写1字节                      │
│    string字段：先写4字节长度，再写N字节内容 │  ← 长度不固定，所以需要先存长度
├─────────────────────────────────────────┤
│  第2行数据...                             │
│  ...                                     │
└─────────────────────────────────────────┘
```

**为什么 string 要先写长度？**
int/float/bool 字节数固定（4/4/1），读取时直接按固定长度读。
string 编码后长度不固定（中文字符UTF8占3字节，英文占1字节），读取时不知道该读多少字节，所以写入时先存一个 int 表示长度，读取时先读这个长度，再按长度读字符串内容。

## 核心设计思路

**关键逻辑一：为什么用反射实现 `LoadTable<T, K>()`**

如果不用反射，每张表都要单独写一个加载方法：

```csharp
// ❌ 不用反射：有多少张表就要写多少个方法，字段改了代码也要跟着改
public void LoadPlayerInfo() {
    var obj = new PlayerInfo();
    obj.id = BitConverter.ToInt32(bytes, index); index += 4;
    obj.name = Encoding.UTF8.GetString(...);
    // ...
}
public void LoadTowerInfo() { /* 又是一堆重复代码 */ }
```

用反射之后，一个方法处理所有表：

```csharp
// ✅ 用反射：运行时动态查询字段，通用逻辑处理任意表
Type classType = typeof(K);                    // 运行时才知道是哪张表
FieldInfo[] infos = classType.GetFields();     // 动态问：你有哪些字段？
object obj = Activator.CreateInstance(classType); // 动态创建对象
foreach (FieldInfo info in infos)
    info.SetValue(obj, 读出来的值);             // 动态赋值
```

**关键逻辑二：为什么 tableDic 用 `Dictionary<string, object>`**

一个字典只能声明一种 Value 类型，但要同时存 `PlayerInfoContainer`、`TowerInfoContainer` 等不同类型：

```csharp
// ❌ 无法同时存不同类型
Dictionary<string, PlayerInfoContainer> tableDic;   // 只能存PlayerInfo
Dictionary<string, TowerInfoContainer> tableDic;    // 只能存TowerInfo

// ✅ 用公共基类 object 统一存储，取出时 as T 转回具体类型
Dictionary<string, object> tableDic;
tableDic.Add("PlayerInfoContainer", playerContainerObj);  // 存
tableDic["PlayerInfoContainer"] as PlayerInfoContainer;   // 取
```

**关键逻辑三：字段顺序的隐式约定**

`GetFields()` 返回字段的顺序不被官方保证，但此工具依赖一个约定：
`ExcelTool` 生成 `.cs` 文件时，字段声明顺序 = Excel 列从左到右的顺序。
写入和读取都按这个顺序走，所以能对上。一旦手动修改生成的 `.cs` 文件改变字段顺序，读取就会错位。

## 完整工程

**配置规则**
```
1.Excel文件应该放置在ArtRes/Excel当中
如果想要修改 就去修改ExcelTool当中的EXCEL_PATH路径变量

2.配置表规则
	第一行：字段名
	第二行：字段类型（字段类型一定不要配置错误，字段类型目前只支持int float bool string）
	如果想要再添加类型，需要在ExcelTool的GenerateExcelBinary方法中
	和BinaryDataMgr的LoadTable方法当中对应添加读写的逻辑
	第三行：主键是哪一个字段 需要通过key来标识主键
	第四行：描述信息（只是给别人看，不会有别的作用）
	第五行~第n行：就是具体数据信息
	下方的表名决定了数据结构类，容器类，2进制文件的文件名
	
3.生成的容器类和数据结构类可以在ExcelTool当中修改DATA_CLASS_PATH和DATA_CONTAINER_PATH
变量来进行更改

4.生成2进制文件的路径可以在BinaryDataMgr当中的DATA_BINARY_PATH变量来进行修改
```

| 配置项        | 变量名                   | 位置                 |

### ExcelTool.cs

```csharp
public class ExcelTool
{
    public static string EXCEL_PATH = Application.dataPath + "/ArtRes/Excel/"; //读取文件的路径
    public static string DATA_CLASS_PATH = Application.dataPath + "/Scripts/ExcelData/ExcelClass/"; //存储文件的路径
    public static string DATA_CONTAINER_PATH = Application.dataPath + "/Scripts/ExcelData/Container/"; //存储容器类的路径
    public static int BEGIN_INDEX = 4; //写入数据开始行的索引

    [MenuItem("GameTool/GenerateExcel")]
    private static void GenerateExcelInfo()
    {
        DirectoryInfo dInfo = Directory.CreateDirectory(EXCEL_PATH); //得到Excel目录的文件夹信息
        FileInfo[] files = dInfo.GetFiles(); //得到Excel目录下的所有文件信息
        DataTableCollection tableCollection; //所有表数据
        for (int i = 0; i < files.Length; i++)
        {
            if (files[i].Extension != ".xlsx" && files[i].Extension != ".xls") //排除后缀不为Excel的文件
                continue;
            using (FileStream fs = files[i].Open(FileMode.Open, FileAccess.Read)) //打开所有文件
            {
                IExcelDataReader excelReader = ExcelReaderFactory.CreateOpenXmlReader(fs); //获取Excel数据
                tableCollection = excelReader.AsDataSet().Tables;// 得到所有表数据
                fs.Close();
            }

            foreach (DataTable table in tableCollection) //遍历表数据
            {
                //生成数据结构类
                GenerateExcelDataClass(table);
                //生成容器类
                GenerateExcelContainer(table);
                //生成二进制数据
                GenerateExcelBinary(table);
            }
        }
    }
    /// <summary>
    /// 生成数据结构类
    /// </summary>
    private static void GenerateExcelDataClass(DataTable table)
    {
        DataRow rowName = GetVariableNameRow(table); //得到变量名所在行所有信息
        DataRow rowType = GetVariableTypeRow(table); //得到变量字段所在行所有信息
        if (!Directory.Exists(DATA_CLASS_PATH)) //如果没有文件夹则创建
            Directory.CreateDirectory(DATA_CLASS_PATH);

        // 通过字符串拼接创建数据结构类
        string classStr = "public class " + table.TableName + "\n{\n";
        for (int i = 0; i < table.Columns.Count; i++)
            classStr += "\tpublic " + rowType[i].ToString() + " " + rowName[i].ToString() + ";\n";
        classStr += "}";
        File.WriteAllText(DATA_CLASS_PATH + table.TableName + ".cs", classStr);
        AssetDatabase.Refresh();
    }
    /// <summary>
    /// 生成容器类
    /// </summary>
    private static void GenerateExcelContainer(DataTable table)
    {
        if (!Directory.Exists(DATA_CONTAINER_PATH)) //没有文件夹则创建
            Directory.CreateDirectory(DATA_CONTAINER_PATH);
        int index = GetKeyIndex(table); //得到变量名所在行所有信息
        DataRow rowType = GetVariableTypeRow(table); //得到变量字段所在行所有信息
        string containerStr = "using System.Collections.Generic;\n";
        containerStr += "public class " + table.TableName + "Container" +  "\n{\n";
        containerStr += "\tpublic Dictionary<" + rowType[index].ToString() + ", " + table.TableName + "> ";
        containerStr += "dataDic = new Dictionary<" + rowType[index].ToString() + ", " + table.TableName + ">();\n";
        containerStr += "}";
        File.WriteAllText(DATA_CONTAINER_PATH + table.TableName + "Container.cs", containerStr);
        AssetDatabase.Refresh();
    }
    /// <summary>
    /// 写入二进制数据
    /// </summary>
    private static void GenerateExcelBinary(DataTable table) 
    {
        if (!Directory.Exists(BinaryDataMgr.DATA_BINARY_PATH)) //没有文件夹则创建
            Directory.CreateDirectory(BinaryDataMgr.DATA_BINARY_PATH);
        using (FileStream fs = new FileStream(BinaryDataMgr.DATA_BINARY_PATH + table.TableName + ".data", FileMode.OpenOrCreate, FileAccess.Write))
        {
            fs.Write(BitConverter.GetBytes(table.Rows.Count - 4), 0, 4);//写入 需要读取的行数 减去了描述信息的行 留下纯数据
            string KeyName = GetVariableNameRow(table)[GetKeyIndex(table)].ToString(); //得到 键对应的字段名
            byte[] bytes = Encoding.UTF8.GetBytes(KeyName);
            fs.Write(BitConverter.GetBytes(bytes.Length), 0, 4);//写入 键名对应的字段名对应比特数
            fs.Write(bytes, 0, bytes.Length);//写入 键名对应的字段名

            DataRow row;
            DataRow rowType = GetVariableTypeRow(table); //得到 所有字段信息
            for (int i = BEGIN_INDEX; i < table.Rows.Count; i++) //遍历表的每一行
            {
                row = table.Rows[i];//得到表中一行的数据
                for (int j = 0; j < table.Columns.Count; j++) //遍历每一列
                {
                    switch (rowType[j].ToString())
                    {
                        case "int"://写入 int类型数据
                            fs.Write(BitConverter.GetBytes(int.Parse(row[j].ToString())), 0, 4);
                            break;
                        case "float"://写入 float类型数据
                            fs.Write(BitConverter.GetBytes(float.Parse(row[j].ToString())), 0, 4);
                            break;
                        case "bool"://写入 bool类型数据
                            fs.Write(BitConverter.GetBytes(bool.Parse(row[j].ToString())), 0, 1);
                            break;
                        case "string"://写入 string类型数据
                            bytes = Encoding.UTF8.GetBytes(row[j].ToString());
                            fs.Write(BitConverter.GetBytes(bytes.Length), 0, 4);
                            fs.Write(bytes, 0, bytes.Length);
                            break;
                        //可以继续添加 其他变量类型
                    }
                }
            }
            fs.Close();
        }
        AssetDatabase.Refresh();
    }
    /// <summary>
    /// 获取变量名所在行所有信息
    /// </summary>
    private static DataRow GetVariableNameRow(DataTable table) { return table.Rows[0]; }
    /// <summary>
    /// 获取变量类型所在行所有信息
    /// </summary>
    private static DataRow GetVariableTypeRow(DataTable table) { return table.Rows[1]; }
    /// <summary>
    /// 获取Key所在列的索引
    /// </summary>
    private static int GetKeyIndex(DataTable table)
    {
        DataRow row = table.Rows[2];//Key 在第三行 获取第三行
        for (int i = 0; i < table.Columns.Count; i++)
        {
            if (row[i].ToString() == "Key")
                return i;
        }
        Debug.Log("没有找到键对应的索引");
        return 0;
    }
}
```

### BinaryDataMgr.cs

```csharp
public class BinaryDataMgr
{
    //因为打包后会删除Editor中的文件 所以把二进制文件的路径移动到这里
    //让这个变量优先加载 放到单例对象初始化前面 避免单例对象加载完毕后 路径还没有加载的情况
    public static string DATA_BINARY_PATH = Application.streamingAssetsPath + "/Binary/"; //二进制数据的路径
    private static BinaryDataMgr instance = new BinaryDataMgr();
    public static BinaryDataMgr Instance => instance;
    private Dictionary<string, object> tableDic = new Dictionary<string, object>(); //存储所有表信息的容器
    private BinaryDataMgr() { }
    private string SAVE_PATH = Application.persistentDataPath + "/Data/";

    /// <summary>
    /// 初始化表容器
    /// </summary>
    public void InitData() 
    {
        //LoadTable<TowerInfoContainer, TowerInfo>();
        //LoadTable<PlayerInfoContainer, PlayerInfo>();
        //LoadTable<TestInfoContainer, TestInfo>();
    }

    /// <summary>
    /// 加载Excel二进制表的数据到容器中
    /// </summary>
    /// <typeparam name="T">容器类名</typeparam>
    /// <typeparam name="k">数据结构类名</typeparam>
    public void LoadTable<T, K>()
    {
        //读取Excel表对应的二进制文件 进行解析
        using (FileStream fs = File.Open(DATA_BINARY_PATH + typeof(K).Name + ".data", FileMode.Open, FileAccess.Read))
        {
            byte[] bytes = new byte[fs.Length];//获取所有字节
            fs.Read(bytes, 0, bytes.Length); //读取所有字节
            fs.Close();
            int index = 0;//记录当前读到第几个字节
            int count = BitConverter.ToInt32(bytes, index);//读取多少行数据
            index += 4;
            int keyNameLength = BitConverter.ToInt32(bytes, index);//读取主键的字节数
            index += 4;
            string keyName = Encoding.UTF8.GetString(bytes, index, keyNameLength);//读取主键的名字
            index += keyNameLength;
            
            //通过反射创建容器类对象
            Type containerType = typeof(T);//得到容器类的Type
            object containerObj = Activator.CreateInstance(containerType);//通过反射实例化容器对象
            Type classType = typeof(K);//得到数据结构类的Type
            FieldInfo[] infos = classType.GetFields();//得到数据结构类所有字段信息
            for (int i = 0; i < count; i++) //读取每一行的信息
            {
                object dataObj = Activator.CreateInstance(classType);//通过反射实例化数据结构类对象
                foreach (FieldInfo info in infos) //遍历数据结构类所有字段信息
                {
                    //根据字段类型决定如何读取数据
                    if (info.FieldType == typeof(int))
                    {
                        info.SetValue(dataObj, BitConverter.ToInt32(bytes, index));//把读出来的值赋值到字段
                        index += 4;
                    }
                    else if (info.FieldType == typeof(float))
                    {
                        info.SetValue(dataObj, BitConverter.ToSingle(bytes, index));//把读出来的值赋值到字段
                        index += 4;
                    }
                    else if (info.FieldType == typeof(bool))
                    {
                        info.SetValue(dataObj, BitConverter.ToBoolean(bytes, index));//把读出来的值赋值到字段
                        index += 1;
                    }
                    else if (info.FieldType == typeof(string))
                    {
                        int stringLength = BitConverter.ToInt32(bytes, index);
                        index += 4;
                        info.SetValue(dataObj, Encoding.UTF8.GetString(bytes, index, stringLength));//把读出来的值赋值到字段
                        index += stringLength;
                    }
                    //更多类型可以继续补充
                }
                //读完一行数据 应该把数据存到容器中
                object dicObj = containerType.GetField("dataDic").GetValue(containerObj);//得到容器对象用来存数据的的字段
                MethodInfo mInfo = dicObj.GetType().GetMethod("Add");//得到字典对象的Add方法
                object keyValue = classType.GetField(keyName).GetValue(dataObj);//得到主键对应的值
                mInfo.Invoke(dicObj, new object[] { keyValue, dataObj });//传入键和值
            }
            tableDic.Add(typeof(T).Name, containerObj);//把键值添加到存储所有表的容器类中
            fs.Close();
        }
    }
    /// <summary>
    /// 获取表
    /// </summary>
    /// <typeparam name="T">容器类类名</typeparam>
    /// <returns></returns>
    public T GetTable<T>() where T : class
    {
        string tableName = typeof(T).Name;
        if (tableDic.ContainsKey(tableName))
            return tableDic[tableName] as T;
        return null;
    }

    public void SavaData(object data, string fileName)
    {
        if (!Directory.Exists(SAVE_PATH))//判断是否有文件夹
            Directory.CreateDirectory(SAVE_PATH);//没有则创建
        using (FileStream fs = new FileStream(SAVE_PATH + fileName + ".data", FileMode.OpenOrCreate, FileAccess.Write))
        {
            BinaryFormatter bf = new BinaryFormatter();
            bf.Serialize(fs, data);//序列化
            fs.Close();
        }
    }
    public T LoadData<T>(string fileName) where T : class
    {
        if (!File.Exists(SAVE_PATH + fileName + ".data"))
            return default(T);
        T data;
        using (FileStream fs = File.Open(SAVE_PATH + fileName + ".data", FileMode.Open, FileAccess.Read))
        {
            BinaryFormatter bf = new BinaryFormatter();
            data = bf.Deserialize(fs) as T; //反序列化
            fs.Close();
        }
        return data;
    }
}
```

### 语法问题

**反射中Invoke的用法**

`mInfo` 是字典的 `Add` 方法，字典的 `Add` 签名是：
```csharp
void Add(TKey key, TValue value)
```

它需要传两个参数：一个键，一个值。
`mInfo.Invoke` 是反射调用方法的写法，格式是：
```csharp
方法.Invoke(调用者对象, new object[] { 参数1, 参数2, ... })
```

所以 `new object[]` 只是反射调用方法时**打包多个参数**的固定写法，和传什么内容无关。
把这一行还原成正常代码，等价于：
```csharp
// 反射写法
mInfo.Invoke(dicObj, new object[] { keyValue, dataObj });

// 等价于正常写法
dataDic.Add(keyValue, dataObj);
// 比如
dataDic.Add(1, playerInfoObj);
```
