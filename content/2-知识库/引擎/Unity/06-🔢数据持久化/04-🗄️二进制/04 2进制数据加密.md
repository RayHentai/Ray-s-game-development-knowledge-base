## 核心理解

**加密的作用**
- 将类对象转换为2进制数据时进行加密，将2进制数据转换为类对象时进行解密
- 这样如果第三方获取到2进制数据
- 他们不知道加密规则和解密秘钥时就无法获取正确的数据，起到保证数据安全的作用

**注意：**
- 加密只是提高破解门槛，没有100%保密的数据
- 通过各种尝试始终是可以破解加密规则的，只是时间问题
- 加密只能起到提升一定的安全性

## 简单异或加密演示

```csharp
Person p = new Person();
byte key = 199;
using (MemoryStream ms = new MemoryStream())
{
    BinaryFormatter bf = new BinaryFormatter();
    bf.Serialize(ms, p);
    byte[] bytes = ms.GetBuffer();
    //异或加密
    for (int i = 0; i < bytes.Length; i++)
    {
        bytes[i] ^= key;
    }
    File.WriteAllBytes(Application.dataPath + "/Test.test", bytes);
}

//解密
byte[] bytes2 = File.ReadAllBytes(Application.dataPath + "/Test.test");
for (int i = 0; i < bytes2.Length; i++)
{
    bytes2[i] ^= key;
}
using (MemoryStream ms = new MemoryStream(bytes2))
{
    BinaryFormatter bf = new BinaryFormatter();
    Person p2 = bf.Deserialize(ms) as Person;
    ms.Close();
}
```
