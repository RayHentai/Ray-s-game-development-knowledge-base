## Rect Transform Utility

RectTransformUtility 公共类是—个RectTransform的辅助类
主要用于进行一些坐标的转换等等操作

其中对于我们目前来说最重要的函数是将屏幕空间上的点，转换成UI本地坐标下的点

---

## 屏幕坐标转换为UI本地坐标

```csharp
//参数一：相对父对象
//参数二：屏幕点
//参数三：摄像机
//参数四：最终得到的点
//一般配合拖拽事件使用
public void OnDrag(PointerEventData eventData)
{
	private Vector2 nowPos;
	RectTransformUtility.ScreenPointToLocalPointInRectangle(
		transform.parent as RectTransform, 
		eventData.position, 
		eventData.enterEventCamera, 
		out nowPos);
	transform.localPosition = nowPos;
}

```

## 练习

> 1. 在上节课的练习题基础上，修改其中的代码，用新学的屏幕坐标转UI坐标的知识，制作一个UGUI摇杆可以控制场景上的对象移动

```csharp
//这个写法比偏移量的写法更精准
//当鼠标回到遥感范围内时，摇杆会精准定位到鼠标位置，而基于偏移量的写法则不会
private void JoyDrag(BaseEventData data) //拖动时执行的逻辑
{
    PointerEventData eventData = data as PointerEventData;
    Vector2 nowPos;
	RectTransformUtility.ScreenPointToLocalPointInRectangle(joyPos.parent as RectTransform, eventData.position, eventData.enterEventCamera,out nowPos);
    //joyPos.transform.position += new Vector3(eventData.delta.x, eventData.delta.y, 0);//当前位置加上偏移位置
    joyPos.localPosition = nowPos;
    if (joyPos.anchoredPosition.magnitude >= 130)//边界判断 模长大于130
        joyPos.anchoredPosition = joyPos.anchoredPosition.normalized * 130;//拉回 单位向量 * 130
    player.Move(joyPos.anchoredPosition);//传入摇杆方向向量
}
```
