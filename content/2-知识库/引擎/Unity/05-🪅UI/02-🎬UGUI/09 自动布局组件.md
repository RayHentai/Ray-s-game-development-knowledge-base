## 核心理解

>UGUI中提供了很多可以帮助我们对UI控件进行自动布局的组件，他们可以自动的设置UI控件的位置和大小等

**自动布局的工作方式一般是：自动布局控制组件 + 布局元素 = 自动布局**

**自动布局控制组件：** Unity提供了很多用于自动布局的管理性质的组件用于布局
**布局元素：** 具备布局属性的对象们，这主要是指具备RectTransform的uI组件

---

## 布局元素的布局属性

位置：预览框左边下拉列表 -> Layout Properties

**要参与自动布局的布局元素必须包含布局属性**

**布局属性主要有以下几条**
1. **Minmum width：** 该布局元素应具有的最小宽度
2. **Minmumheight：** 该布局元素应具有的最度
3. **Preferredwidth：** 在分配额外可用宽度之前，此布局元素应具有的宽度
4. **Preferred height：** 在分配额外可用高度之前，此布局元素应具有的高度
5. **Flexiblewidth：** 此布局元素应相对于其同级而填充的额外可用宽度的相对量
6. **Flexibleheight：** 此布局元素应相对于其同级而填充的额外可用高度的相对量
**在进行自动布局时都会通过计算布局元素中的这6个属性得到控件的大小位置**

**在布局时，布局元素大小设置的基本规则是**
1. 首先分配最小小Minmum width和Minmum height
2. 如果父类容器中有够的可空间，则分配Preferred width和Preferred height
3. 如果上面两条分配完成后还有额外空间，则分配Flexiblewidth和Flexibleheight

**一般情况下布局元素的这些属性都是0**
但是特定的UI组件依附的对象布局属性会被改变，比如Image和Text

**一般情况下不会手动修改他们，但是如果有需求**
可以动添加个LayoutElement组件可以修改这些布局属性
