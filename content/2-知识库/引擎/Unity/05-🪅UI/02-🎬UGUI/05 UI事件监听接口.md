## 接口

---

### 作用

**目前所有的控件都只提供了常用的事件监听列表**
如果想做一些类似**长按**，**双击**，**拖拽**等功能是无法制作的
或者想让Image和Text，Raw Image 三大基础控件能够响应玩家输入也是无法制作的

**而事件接口就是用来处理类似问题，让所有控件都能够添加更多的事件监听来处理对应的逻辑**

**缺点：不方使管理，需要自己写脚本继承接口，并挂载到对应控件上，比较麻烦**

---

### 事件接口

**常用：**

| 接口名                  | 接口函数名          | 备注                           |
| -------------------- | -------------- | ---------------------------- |
| IPointerEnterHandler | OnPointerEnte  | 当指针进入对象时调用（鼠标进入）             |
| IPointerExitHandler  | OnPointerExit  | 当指针退出对象时调用（鼠标离开）             |
| IPointerDownHandler  | OnPointerDown  | 在对象上按下指针时调用（按下）              |
| IPointerUpHandler    | OnPointerUp    | 松开指针时调用（在指针正在点击的游戏对象上调用）（抬起） |
| IPointerClickHandler | OnPointerClick | 在同一对象上按下再松开指针时调用（点击）         |
| IBeginDragHandler    | OnBeginDrag    | 即将开始拖动时在拖动对象上调（开始拖拽）         |
| IDragHandler         | OnDrag         | 发生拖动时在拖动对象上调用（拖拽中）           |
| IEndDragHandler      | OnEndDrag      | 拖动完成时在拖动对象上调（结束拖拽）           |

**不常用：**

| 接口名                             | 接口函数名                     | 备注                  |
| ------------------------------- | ------------------------- | ------------------- |
| IInitializePotentialDragHandler | OnInitializePotentialDrag | 在找到拖动目标时调用，可用于初始化值  |
| IDropHandler                    | OnDrop                    | 在拖动目标对象上调用          |
| IScrol1Handler                  | OnScroll                  | 当鼠标滚轮滚动时调用          |
| IUpdateSelectedHandler          | OnUpdateSelected          | 每次勾选时在选定对象上调用       |
| ISelectHandler                  | OnSelect                  | 当对象成为选定对象时调用        |
| IDeselectHandler                | OnDeselect                | 取消选择选定对象时调用         |
| IMoveHandler                    | OnMove                    | 发生移动事件（上、下、左、右等）时调用 |
| ISubmitHandler                  | OnSubmit                  | 按下 Submit（回车） 按钮时调用 |
| ICancelHandler                  | OnCancel                  | 按下 Cancel（取消） 按钮时调用 |

---

### 使用接口

```csharp
//1 继承 MonoBehavior 的脚本继承对应的事件接口，引用命名空间
//2 实现接口中的内容
//3 将该脚本挂载到想要监听自定义事件的UI控件上
public class Test : MonoBehaviour, IPointerEnterHandler, IPointerExitHandler, IPointerDownHandler, IPointerUpHandler
{
    public void OnPointerDown(PointerEventData eventData)
    {
        print("鼠标（触摸）按下");
    }
    public void OnPointerEnter(PointerEventData eventData)
    {
        print("鼠标进入");
    }
    public void OnPointerExit(PointerEventData eventData)
    {
        print("鼠标离开");
    }
    public void OnPointerUp(PointerEventData eventData)
    {
        print("鼠标（触摸）抬起");
    }
}
```

### PointerEventData 参数

```csharp
//父类： BaseEventData
pointerid： 鼠标左右中键点击鼠标的ID（-1左，-2右，-3中），通过它可以判断右键点击
position： 当前指针位置（屏幕坐标系）（一直刷新）
pressPosition： 按下的时候指针的位置（一次）
delta： 指针移动增量（位置偏移）
clickCount： 连击次数
clickTime： 点击时间（系统时间）
pressEventCamera： 最后一个onPointerPress按下事件关联的摄像机
enterEvetnCamera： 最后一个onPointerEnter进入事件关联的摄像机
```

### 练习

> 1. 在上节课的练习题基础上，请用现在所学知识，制作一个这样的功能：长按一个UI按钮0.2s后开始蓄能，松开按钮后结束蓄能蓄能满后HP+10，如果没有松开按钮继续蓄能

```csharp
public class EventInterface_Pro : MonoBehaviour, IPointerDownHandler, IPointerUpHandler
{
    public event UnityAction up;
    public event UnityAction down;
    public void OnPointerDown(PointerEventData eventData) { down?.Invoke(); }
    public void OnPointerUp(PointerEventData eventData) { up?.Invoke(); }
}
public class EventInterface_Panel : MonoBehaviour
{
    public EventInterface_Pro btnPro;
    public GameObject hpRoot;
    public RectTransform hpBK;
    private bool isDown = false;
    private float time = 0;
    public int speed = 50;
    private int nowHp = 0;
    void Start()
    {
        btnPro.up += Up;
        btnPro.down += Down;
        hpRoot.SetActive(false);
    }
    private void Update()
    {
        if (isDown) 
        {
            time += Time.deltaTime;
            if (time >= 0.2f)
            {
                hpRoot.SetActive(true);
                hpBK.sizeDelta += new Vector2(speed * Time.deltaTime, 0);
                if (hpBK.sizeDelta.x >= 700)
                {
                    hpBK.sizeDelta = new Vector2(0, 60);
                    nowHp += 10;
                    print("+10hp，当前hp" + nowHp);
                }
            }
        }
    }
    private void Down() 
    {
        isDown = true;
        time = 0;
        hpBK.sizeDelta = new Vector2(0, 60);
    }
    private void Up() 
    {
        isDown = false;
        hpRoot.SetActive(false);
    }
}
```

## Event Trigger 事件触发器

>事件触发器是 Event Trigger 组件
>它是一个集成了所有事件接口的脚本，可以更方便的为控件添加事件监听

**使用：**
首先为控件添加 `Event trigger` 脚本，然后点击 `Add New Event Type` 添加需要的事件监听
1. 方法一：拖曳脚本进行关联

```csharp
//事件监听的函数需要传入参数 BaseEventData
public void TestPointerEnter(BaseEventData data) 
{
	PointerEventData eventDate = data as PointerEventData;
}
```

2. 方法二：代码添加

public EventTrigger et;

//2 申明事件的类型

### 练习

> 1. 在上节课的练习题基础上，请用现在所学知识，制作一个这样的功能：制作一个UGUI摇杆可以控制场景上的对象移动

```csharp
public class EventInterface_Panel : MonoBehaviour
{
    public EventInterface_Player player;
    void Start()
    {
        EventTrigger.Entry entry = new EventTrigger.Entry();//为拖动中注册事件
        entry.eventID = EventTriggerType.Drag;
        entry.callback.AddListener(JoyDrag);
        etJoy.triggers.Add(entry);
        entry = new EventTrigger.Entry();//为结束拖动注册事件
        entry.eventID = EventTriggerType.EndDrag;
        entry.callback.AddListener(EndJoyDrag);
        etJoy.triggers.Add(entry);
    }
    public EventTrigger etJoy;
    public RectTransform joyPos;
    private void JoyDrag(BaseEventData data) //拖动时执行的逻辑
    {
        PointerEventData eventData = data as PointerEventData;
        joyPos.transform.position += new Vector3(eventData.delta.x, eventData.delta.y, 0);//当前位置加上偏移位置
        if (joyPos.anchoredPosition.magnitude >= 130)//边界判断 模长大于130
            joyPos.anchoredPosition = joyPos.anchoredPosition.normalized * 130;//拉回 单位向量 * 130
        player.Move(joyPos.anchoredPosition);//传入摇杆方向向量
    }
    private void EndJoyDrag(BaseEventData data) //取消拖动时执行的逻辑
    {
        PointerEventData eventData = data as PointerEventData;
        joyPos.anchoredPosition = Vector2.zero;//回到中心点
        player.Move(Vector2.zero);//传入0向量
    }
}
public class EventInterface_Player : MonoBehaviour
{
    private Vector3 moveDir = Vector3.zero;//移动方向
    public float moveSpeed = 10;
    public float roundSpeed = 10;
    public void Move(Vector2 dir)//传入的是一个二维向量
    {
        moveDir.x = dir.x;
        moveDir.y = 0;
        moveDir.z = dir.y;//二维向量转三维向量
    }
    private void Update()
    {
        if (moveDir != Vector3.zero)
        {
            transform.Translate(Vector3.forward * moveSpeed * Time.deltaTime);
            transform.rotation = Quaternion.Slerp(transform.rotation, Quaternion.LookRotation(moveDir), roundSpeed * Time.deltaTime);
        }
    }
}
```
