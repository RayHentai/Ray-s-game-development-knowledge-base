# 🎨 09 自定义整体样式

**所属模块**：[[00 GUI 总览]]
**查阅次数**：0

---

## 全局颜色

```csharp
// 改变全局颜色（会影响所有后续控件）
GUI.color = Color.red;

// 改变文本颜色（会和全局颜色相乘）
GUI.contentColor = Color.yellow;

// 改变背景元素颜色（会和全局颜色相乘）
GUI.backgroundColor = Color.blue;
```

---

## GUIStyle 单个控件样式

> 在调用控件时传入最后一个 GUIStyle 参数，可以自定义该控件的外观。

```csharp
GUIStyle style = new GUIStyle();
style.fontSize = 24;
style.fontStyle = FontStyle.Bold;
style.normal.textColor  = Color.white;
style.normal.background = someTexture;

GUI.Label(rect, "文字", style);
```

---

## GUISkin 整体皮肤（批量样式）

> 比单独设置 GUIStyle 更方便，可以统一设置所有控件的样式。

```csharp
// 在 Project 窗口右键 → Create → GUI Skin 创建皮肤资源
GUISkin skin;  // 在 Inspector 中拖入皮肤资源

void OnGUI()
{
    // 设置为 null 使用默认皮肤
    GUI.skin = null;

    // 设置自定义皮肤（之后绘制的所有控件都使用该皮肤）
    GUI.skin = skin;

    // 注意：如果控件的方法调用时使用了 GUIStyle 参数，皮肤对该控件就没用了
}
```
