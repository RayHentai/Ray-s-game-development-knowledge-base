## GUI 控件的共同点

- 都是 **GUI 公共类中提供的`静态函数`**，直接调用即可
- 参数大同小异，每种控件都有多种重载（各参数的排列组合）
- 必备参数：**位置信息** + **显示信息**
- **GUI 的原点 (0, 0) 是屏幕左上角**（与 Unity 世界坐标不同）

---

## 核心参数类型

| 参数类型 | 说明 |
|----------|------|
| `Rect` | 位置参数：x、y 位置，w、h 尺寸 |
| `string` | 显示文本 |
| `Texture` | 图片信息 |
| `GUIContent` | 综合信息（Text 文本 + Image 图片 + Tooltip 提示，图片在左边，文本在右边） |
| `GUIStyle` | 自定义样式 |

---

## 文本控件 Label

```csharp
Rect rect = new Rect(100, 100, 200, 50);
Texture texture;
// 基本使用
GUI.Label(rect, "文本内容");
GUI.Label(rect, texture); // 显示图片

// 综合使用（GUIContent）
GUIContent content = new GUIContent("文本", texture, "Tooltip提示");
GUI.Label(rect, content);

// 获取当前鼠标或键盘选中控件的 Tooltip 信息
string tip = GUI.tooltip;

// 自定义样式
GUIStyle style = new GUIStyle();
GUI.Label(rect, content, style);

```

## 按钮控件 Button

```csharp
// 普通按钮（在按钮范围内按下鼠标抬起时检测一次，返回 bool）
if (GUI.Button(rect, "按钮文字"))
{
    // 点击后的逻辑
}

// 综合 + 自定义样式
GUI.Button(rect, content, style);

// 长按按钮（持续按住返回 true）
if (GUI.RepeatButton(rect, content, style))
{
    // 持续按住时的逻辑
}
```

## 练习

> 1. 请用 GUI 制作一个游戏开始界面，上面有开始游戏、退出游戏、设置等按钮，点击开始游戏可以切换到游戏场景

```csharp
using UnityEngine.SceneManagement;

public class Button : MonoBehaviour
{
    //标题变量
    public Rect titlePos;
    public GUIContent content;
    public GUIStyle style;
    //按钮变量
    public Rect button1Pos;
    public Rect button2Pos;
    public Rect button3Pos;
    //共同的变量 共用就行
    public GUIStyle buttonStyle;
    private void OnGUI()
    {
        GUI.Label(titlePos, content, style);
        //按下处理逻辑
        if (GUI.Button(button1Pos, "开始游戏", buttonStyle))
            //切换场景
            SceneManager.LoadScene("Game");
        if (GUI.Button(button2Pos, "设置", buttonStyle))
        {
			//设置逻辑
        }
        if (GUI.Button(button3Pos, "结束游戏", buttonStyle))
        {
			//结束游戏逻辑
        }
    }
}
// （图片：开始界面效果，见练习文档图片1/2/3）
```
