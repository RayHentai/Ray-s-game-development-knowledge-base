## 分组 Group

**作用：** 用于批量控制控件位置，可以理解为给包裹的控件加了一个"父对象"，通过控制分组来控制包裹控件的位置。

```csharp
// 分组内的控件坐标，是相对分组 rect 的本地坐标
GUI.BeginGroup(rect);  // 开始分组

    GUI.Label(new Rect(10, 10, 100, 30), "我是分组内的控件");
    GUI.Button(new Rect(10, 50, 100, 30), "按钮");

GUI.EndGroup(); // 结束分组
```

## 滚动视图 ScrollView

```csharp
Vector2 scrollPos = Vector2.zero;

void OnGUI()
{
    // 参数1：可见范围（视口大小）
    // 参数2：当前滚动条的位置（Vector2，要存储起来）
    // 参数3：内容实际尺寸（超出可见范围会出现滚动条）
    // 参数4/5（可选）：是否始终显示水平/垂直滚动条
    scrollPos = GUI.BeginScrollView(visibleRect, scrollPos, contentRect);
        // 在这里绘制实际内容
        for (int i = 0; i < 20; i++)
            GUI.Label(new Rect(0, i * 30, 100, 30), "Item " + i);
    GUI.EndScrollView();
}
```

## 练习

> 1. 制作一个滚动视图，外部有一个字符串数组，通过在 Inspector 修改这个字符串数组数量，会在滚动视图内部动态创建 Label 依次显示内容

```csharp
public class BeginScrollView : MonoBehaviour
{
    public Rect svPos;
    public Rect showPos;
    public Vector2 nowPos;
    public string[] strs;
    private void OnGUI()
    {
        //动态变化内容的高度
        showPos.height = strs.Length * 30;
        nowPos = GUI.BeginScrollView(svPos, nowPos, showPos);
        for (int i = 0; i < strs.Length; i++) 
        {
            //动态变化字符串的y轴位置
            GUI.Label(new Rect(100, i * 30, 100, 30), strs[i]);
        }
        GUI.EndScrollView();
    }
}
```

## 我的踩坑记录

- `BeginScrollView` 和 `EndScrollView` 必须成对使用
- `scrollPos` 需要在 `OnGUI` 外部声明（持久化存储），否则每帧重置
