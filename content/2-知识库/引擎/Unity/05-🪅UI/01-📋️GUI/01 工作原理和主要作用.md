## 定义

> **IMGUI（即时模式游戏用户交互界面）**，在 Unity 中简称 GUI。
> 是一个**代码驱动**的 UI 系统。

---

## 作用

- 作为程序员的**调试工具**，创建游戏内调试工具
- 为脚本组件创建**自定义监视面板**
- 创建新的**编辑器窗口和工具**以拓展 Unity 本身（一般用作内置游戏工具）
- **不适合做玩家 UI**

---

## 缺点

- 重复工作量多
- 控件绘制相关代码很多
- 必须运行才能查看结果
- 不支持分辨率自适应

---

## 工作原理

> 在继承 MonoBehaviour 的脚本中的特殊函数里，调用 GUI 提供的方法，类似生命周期函数。

### OnGUI 函数

```csharp
void OnGUI()
{
    // 在这里调用 GUI 相关方法绘制控件
    GUI.Label(new Rect(0, 0, 100, 30), "Hello GUI");
}
```

**注意：**
- 每帧执行，相当于专门用于绘制 GUI 界面的函数
- 一般只在其中执行 GUI 相关界面绘制和操作逻辑
- **执行时机：** 在 `OnDisable` 之前，`LateUpdate` 之后执行
- 只要是继承 `Mono` 的脚本，都可以在 `OnGUI` 中绘制 GUI
