## 工具栏 Toolbar

```csharp
int toolbarIndex = 0;
string[] options = { "强化", "进阶", "幻化" };

void OnGUI()
{
    // 返回当前选中的 int 索引值
    toolbarIndex = GUI.Toolbar(rect, toolbarIndex, options);
    toolbarIndex = GUI.Toolbar(rect, toolbarIndex, options, style);
}
```

---

## 选择网格 SelectionGrid

> 比工具栏多了一个参数 `xCount`，代表水平方向最多显示的数量。

```csharp
// 超出 xCount 的选项会换行显示
toolbarIndex = GUI.SelectionGrid(rect, toolbarIndex, options, xCount);
toolbarIndex = GUI.SelectionGrid(rect, toolbarIndex, options, xCount, style);
```

## 练习

> 1. 制作三个页签，选择不同的页签将在下方显示不同的信息

```csharp
public class Toolbar : MonoBehaviour
{
    public int toolbarIndex = 0;
    public Rect pos;
    public Rect pos2;
    public string[] str = { "强化","进阶","幻化" };

    public Rect textPos;
    private void OnGUI()
    {
        toolbarIndex = GUI.Toolbar(pos, toolbarIndex, str);
        toolbarIndex = GUI.SelectionGrid(pos2, toolbarIndex, str, 1);
        switch (toolbarIndex)
        {
            case 0:
                GUI.Label(textPos, str[0]);
                break;
            case 1:
                GUI.Label(textPos, str[1]);
                break;
            case 2:
                GUI.Label(textPos, str[2]);
                break;
        }
    }
}
```
