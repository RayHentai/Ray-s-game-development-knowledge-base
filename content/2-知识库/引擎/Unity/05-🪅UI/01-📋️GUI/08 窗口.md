## 普通窗口 Window

```csharp
// 参数1：窗口唯一 ID（不要和其他窗口重复）
//   不仅可以区分不同窗口，还可以在一个函数中处理多个窗口的逻辑
// 参数3：委托参数，传入专门用来绘制窗口内控件的函数
GUI.Window(id, rect, DrawWindowFunc, "窗口标题");
GUI.Window(id, rect, DrawWindowFunc, content, style);

void DrawWindowFunc(int id)
{
    // 在这里绘制窗口内的控件
    // 控件坐标是相对窗口的本地坐标
    GUI.Label(new Rect(10, 30, 100, 30), "窗口内容");
}
```

## 模态窗口 ModalWindow

> 可以让其他控件不再响应，该窗口在最上层，其他按钮都不能点，只能点击该窗口上的控件（常用作**警告弹窗**）。

```csharp
GUI.ModalWindow(id, rect, DrawWindowFunc, "提示", style);
```

## 可拖动窗口 DragWindow

```csharp
Rect windowRect = new Rect(100, 100, 300, 200);

void OnGUI()
{
    // 返回值是拖动后新的 Rect
    windowRect = GUI.Window(0, windowRect, DrawWindowFunc, "可拖动窗口");
}

void DrawWindowFunc(int id)
{
    GUI.Label(new Rect(10, 30, 200, 30), "拖动窗口标题栏即可移动");

    // 写在窗口函数中：使整个窗口可拖动
    // 可以填入 rect 参数指定只有特定区域可拖动
    GUI.DragWindow();  // 不填参数 默认所有位置都可以拖动
    GUI.DragWindow(new Rect(0, 0, 300, 20));  // 只有标题栏可拖动
}
```

## 练习

> 1. 在之前制作的开始界面中，点击退出游戏按钮，弹出一个模态窗口，让玩家确认是否退出（退出/取消），点击取消后关闭窗口，点击退出后退出游戏

```csharp
public class Exit : MonoBehaviour
{
    public Rect pos;
    public Rect pos2;
    public Rect pos3;
    public Rect pos4;

    private static Exit instance;
    public static void ShowMe()
    {
        if (instance != null)
        {
            instance.gameObject.SetActive(true);
        }
    }
    public static void HidMe()
    {
        if (instance != null)
            instance.gameObject.SetActive(false);
    }
    private void Awake()
    {
        instance = this;
    }
    private void OnGUI()
    {
        GUI.ModalWindow(1, pos, DrawWindow,"提示");
    }
    public void DrawWindow(int id)
    {
        GUI.Label(pos4,"是否退出");
        if (GUI.Button(pos2, "退出"))
            Application.Quit();
        if (GUI.Button(pos3, "取消"))
        {
            HidMe();
            Button.ShowMe();
        }
    }
}
```
