## 图片绘制 DrawTexture

```csharp
Texture tex;
ScaleMode mode;
bool alpha;
float imageAspect;

GUI.DrawTexture(rect, tex);
GUI.DrawTexture(rect, tex, mode);
GUI.DrawTexture(rect, tex, mode, alpha, imageAspect);
```

### ScaleMode 缩放模式

| 模式 | 说明 |
|------|------|
| `ScaleMode.ScaleAndCrop` | 通过宽高比计算图片，**会进行裁减** |
| `ScaleMode.ScaleToFit` | 自动根据宽高比计算，**不会拉变形，始终完整显示** |
| `ScaleMode.StretchToFill` | **始终填充满**传入的 rect 范围 |

### 其他参数

```
bool alpha → 控制图片是否开启透明通道，false = 不透明
float imageAspect  → 自定义宽高比，不填默认为 0（使用图片原始宽高比）
```

## 练习

> 1. 为之前的各个面板加上背景图片

```csharp
public class Login : MonoBehaviour
{
    public Rect groundPos;
    public Texture tex;
    private void OnGUI()
    {
        GUI.DrawTexture(groundPos, tex);
    }
}
```

> **注意：** OnGUI 中先调用的控件在底层，后调用的在上层。背景图片要最先绘制。
