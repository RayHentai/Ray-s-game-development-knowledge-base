## 核心理解

> 在指定点发射一个指定方向的射线，判断该射线与哪些碰撞器相交，得到对应对象

**比如：**
- 鼠标选择场景上的物体
- fps射击游戏（不产生实际的子弹对象移动）

### 声明射线

**注意：单独的射线没有任何实际意义，需要结合物理系统进行射线碰撞判断**

```csharp
//声明射线
//1 3D世界中的射线
//假设有一条 起点为坐标(1，0，0) 方向为世界坐标z轴正方向的射线
//参数1 起点
//参数2 方向（一定记住不是两点决定射线方向，第二个参数直接就代表方向向量）
Ray r = new Ray(Vector3.right, Vector3.forward);
//Ray中的参数
r.origin //起点
r.direction // 方向

//2 摄像机发出的射线
//得到一条从屏幕位置作为起点 摄像机视口方向为方向的 射线
Ray r2 = Camera.main.ScreenPointToRay(Input.mousePosition);
```

### 碰撞检测函数

**注意：射线检测是瞬时的，执行代码时进行一次射线检测**

```csharp
//1 最原始的射线检测 只能判断是否有对象被射线碰撞 不能得到对象的碰撞器信息 用的少
//准备一条射线
Ray r = new Ray(Vector3.zero, Vector3.forward);
//进行射线检测 
//参数1 射线
//参数2 检测的最大距离
//参数3 检测指定层级
//参数4 是否忽略触发器
//返回值 bool 碰撞到对象返回true 没有 返回 false
if (Physics.Raycast(r, 1000, 1 << LayerMask.NameToLayer("Monster"), QueryTriggerInteraction.UseGlobal));
{
	Print("碰撞到了对象")
}
//另一种重载 不用传入射线 直接传入起点和方向 也可以判断
if (Physics.Raycast(Vector3.zero, Vector3.forward, 1000, 1 << LayerMask.NameToLayer("Monster"), QueryTriggerInteraction.UseGlobal));
{
	Print("碰撞到了对象")
}

//2 获取相交的单个物体信息
//物体信息类 RaycastHit
RaycastHit hitInfo;
//参数1 射线
//参数2 RaycastHit 是结构体 是值类型 Unity会通过out 在函数内部处理RaycastHit 得到碰撞数据后重新赋值 让RaycastHit在外部也同时改变
//参数3 距离
//参数4 检测指定层级
//参数5 是否忽略触发器
//返回值 bool 碰撞到对象返回true 没有 返回 false
if (Physics.Raycast(r, out hitInfo, 1000, 1 << LayerMask.NameToLayer("Monster"), QueryTriggerInteraction.UseGlobal))
{
	Print("碰撞到了对象，得到了信息")
	hitInfo.collider //碰撞到物体的碰撞器信息
	hitInfo.point //碰撞点
	hitInfo.normal //法线
	hitInfo.transform.posiiton
	hitInfo.distance //从起点到碰撞点的距离
}
//还有一种重载 不用传射线 传起点和方向
if (Physics.Raycast(Vector3.zero, Vector3.forward, out hitInfo, 1000, 1 << LayerMask.NameToLayer("Monster"), QueryTriggerInteraction.UseGlobal))

//3 获取相交的多个物体
//返回 RaycastHit 数组 如果没有 返回容量为0的数组
//参数1 射线
//参数2 距离
//参数3 检测指定层级
//参数4 是否忽略触发器
RaycastHit[] hitInfos = Physics.Raycast(r, 1000, 1 << LayerMask.NameToLayer("Monster"), QueryTriggerInteraction.UseGlobal);
for (int i = 0; i >= hitInfos.Length; i++)
	pring("碰撞到的所有物体 名字分别是" + hitInfos[i].collider.gameObject.name);	
//重载 不用传射线 传起点和方向
RaycastHit[] hitInfos = Physics.Raycast(Vector3.zero, Vector3.forward, 1000, 1 << LayerMask.NameToLayer("Monster"), QueryTriggerInteraction.UseGlobal);

//4 返回碰撞的数量 通过out得到数据
if (Physics.RaycastNonAlloc(r, hitInfo, 1000, 1 << LayerMask.NameToLayer("Monster"), QueryTriggerInteraction.UseGlobal) > 0) { }
```

## 练习

> 1. 请用资料区给的资源，实现鼠标点击场景上一面墙，在点击的位置创建子弹特效和弹孔

```csharp
public class Physics_shoot : MonoBehaviour
{
    private Ray r;
    private RaycastHit hitInfo;
    void Update()
    {
        if (Input.GetMouseButton(0))
        {
            r = Camera.main.ScreenPointToRay(Input.mousePosition);
            if (Physics.Raycast(r, out hitInfo, 1000, 1 << LayerMask.NameToLayer("Wall")))
            {
                print("检测到射线碰撞");
                GameObject hitEff = Instantiate(hitEff = Resources.Load<GameObject>("Effect/HitEff"), hitInfo.point + hitInfo.normal * 0.1f, Quaternion.LookRotation(hitInfo.normal));
                GameObject danKongEff = Instantiate(danKongEff = Resources.Load<GameObject>("Effect/DanKongEff"), hitInfo.point + hitInfo.normal * 0.1f, Quaternion.LookRotation(hitInfo.normal));
                Destroy(hitEff, 0.5f);
            }
        }
    }
}
```

> 2. 场景上有一个平面，有一个立方体，当鼠标点击选中立方体时，长按鼠标左键可以拖动立方体 在平面上移动，点击鼠标右键取消选中

```csharp
public class Physice_Move : MonoBehaviour
{
    public Transform nowSelObj;
    private Ray r;
    private RaycastHit hitInfo;
    public float offSelY = 0.5f;
    void Update()
    {
        if (Input.GetMouseButtonDown(0))
        {
            r = Camera.main.ScreenPointToRay(Input.mousePosition);
            if (Physics.Raycast(r, out hitInfo, 1000, 1 << LayerMask.NameToLayer("Player")))
            {
                print("选中方块");
                nowSelObj = hitInfo.transform;
            }
        }
        else if (Input.GetMouseButtonDown(1))
        {
            r = Camera.main.ScreenPointToRay(Input.mousePosition);
            if (Physics.Raycast(r, out hitInfo, 1000, 1 << LayerMask.NameToLayer("Player")))
            {
                print("取消选中方块");
                nowSelObj = null;
            }
        }
        if (Input.GetMouseButton(0) && nowSelObj != null)
        {
            r = Camera.main.ScreenPointToRay(Input.mousePosition);
            if (Physics.Raycast(r, out hitInfo, 1000, 1 << LayerMask.NameToLayer("Floor")))
                nowSelObj.transform.position = hitInfo.point + Vector3.up * offSelY;
        }
    }
}
```

## API 速查


- Camera.main.ScreenPointToRay（方法 屏幕坐标转射线）
- Physics.Raycast （方法 射线碰撞检测）
- Physics.RaycastNonAlloc （方法 得到射线碰撞对象的数量）

- Ray（射线类）
	- Ray.origin （成员属性 起点）
	- Ray.direction （成员属性 射线方向）

- RaycastHit （物体信息 结构体）
	- RaycastHit.collider （成员属性 碰撞到物体的碰撞器信息）
	- RaycastHit.point （成员属性 碰撞点）
	- RaycastHit.normal （成员属性 法线）
	- RaycastHit.transform （成员属性 位置信息）
	- RaycastHit.distance （成员属性 从起点到碰撞点的距离）
