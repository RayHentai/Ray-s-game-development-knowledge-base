## 角度和弧度

> 度量角的单位
---

### 数学

```csharp
角度： 1°
弧度： 1 radian(rad)
圆一周的角度： 360°
圆一周的弧度： 2π radian(rad)
```

> 转换

```csharp
π rad = 180°
1 rad = (180 / π)° => 1 rad = 180 / 3.14 ≈ 57.3°
1° = (π / 180) rad => 1° = 3.14 / 180 ≈ 0.01745 rad
↓
弧度 * 57.3 = 对应角度
角度 * 0.01745 = 对应弧度
```

---

### 代码

```csharp
//弧度转角度
float rad = 1;
float anger = rad * Mathf.Rad2Deg;

//角度转弧度
anger = 1;
rad = anger * Mathf.Deg2Rad;
```

## 反三角函数

>反三角函数是初等函数之一，包括反正弦函数、反余弦函数等
>作用是通过反三角函数计算正弦值或余弦值对应的弧度值

```csharp
float rad = Mathf.Asin(0.5f); // 正弦函数值为 1/2 时对应角度的弧度
rad * Mathf.Rad2Deg; // 30

float rad = Mathf.ACos(0.5f); // 余弦函数值为 1/2 时对应角度的弧度
rad * Mathf.Rad2Deg; // 60
```

## 练习

> 1. 实现一个物体按曲线移动（正弦或者余弦曲线）

```csharp
public class SinMove : MonoBehaviour
{
    //面朝向的移动速度
    public float forwardSpeed = 5;
    //左右曲线移动变化速度
    public float rightSpeed = 5;
    //左右曲线移动距离
    public float moveSize = 0.5f;

    private float time = 0;
    void Update()
    {
        //面朝向移动
        transform.Translate(Vector3.forward * forwardSpeed * Time.deltaTime);
        //左右曲线移动
        time += Time.deltaTime * rightSpeed;
        transform.Translate(Vector3.right * moveSize * Mathf.Sin(time));
    }
}
```

要点：
- 如果要给三角函数传入一个0到正无穷的值 Time 就是一个可选值
