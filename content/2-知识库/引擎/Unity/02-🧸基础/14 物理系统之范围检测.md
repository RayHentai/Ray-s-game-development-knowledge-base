## 回顾碰撞检测

**碰撞产生的必要条件：**
1. 至少一个物体有刚体
2. 两个物体都必须有碰撞器

**碰撞和触发：**
碰撞会产生实际的物理效果
触发看起来不会产生碰撞但是可以通过函数监听触发

**碰撞检测主要用于实体物体之间产生物理效果时使用**

---

## 核心理解

> 范围判断可以得到处于指定范围内的对象，并对对象进行逻辑处理

**游戏中瞬时的攻击范围判断一般会使用范围检测：**
- 玩家在前方5m处释放一个地刺魔法，在此处范围内的对象将受到地刺伤害
- 玩家攻击，在前方1米圆形范围内对象都受到伤害
- 等等

**类似这种没有实体物体，只想要检测在指定某一范围是否让敌方受到伤害时，便可以使用范围判断**

### 关于层级

- 通过名字得到层级编号LayerMask.NameToLayer
- 我们需要通过编号左移构建二进制数
- 这样每一个编号的层级，都是对应位为1的2进制数
- 我们通过位运算可以选择想要检测层级
- 好处一个int就可以表示所有想要检测的层级信息

**层级编号是0~31刚好32位，是一个int数，每一个编号代表的都是二进制的一位**

```csharp
//[[11 运算符#^0bbea8]]
0 → 1 << 0— 0000 0000 0000 0000 0000 0000 0000 0001 = 1 
1 → 1 << 1— 0000 0000 0000 0000 0000 0000 0000 0010 = 2 
2 → 1 << 2— 0000 0000 0000 0000 0000 0000 0000 0100 = 4
3 → 1 << 3— 0000 0000 0000 0000 0000 0000 0000 1000 = 8
4 → 1 << 4— 0000 0000 0000 0000 0000 0000 0001 0000 = 16
.........

//用对应层的二进制数，和传入的二进制数进行 位与（&）计算 如果结果为0 就不需要检测这一层
//如果想检测多个层，可以用 位或 (|) 连接
//假设传入 
1 << 5 | 1 << 0
0000 0000 0000 0000 0000 0000 0000 0001 +
0000 0000 0000 0000 0000 0000 0001 0000 =
1 0001(33) 代表第 0 层 和 第5层
33(10001) & 1 (00001) ≠ 0
33(10001) & 32(10000) ≠ 0 //就会检测这两层（0、5）

33(10001) & 8(01000) = 0 //就不会检测这一层（2）
```

### 范围检测API

**注意：**
- 范围检测相关API只有当执行该句代码时进行一次范围检测,它是瞬时的
- 范围检测相关API并不会真正产生一个碰撞器只是碰撞判断计算而已
- 检测指定层级相关参数 不能只传入层级序号，而是传入 1 << 层级序号

```csharp
//范围检测API
//1 盒状范围检测
//参数1 立方体中心点 
//参数2 立方体三边大小 的一半
//参数3 立方体角度 
//参数4 检测指定层级（不填检测所有层）
//参数5 是否忽略触发器 UseGlobal 使用全局设置 Collide 检测触发器 Ignore 忽略触发器 不填使用全局设置
//返回值 在该范围内的触发器（得到了对象触发器就可以得到对象的所有信息）
Collider[] colliders = Physics.OverlapBox(Vector3.zero, Vector3.one, Quaternion.AngleAxis(45, Vector3.up), 
				   1 << LayerMask.NameToLayer("UI") |
				   1 << LayerMask.NameToLayer("Default"), 
				   QueryTriggerInteraction.UseGlobal )
//另一个API
//参数 传入一个数组进行存储
//返回值 触碰到的碰撞器数量
if (colliders = Physics.OverlapBoxNonAlloc(比 OverlapBox 多了一个新参数 在第三位) != 0)

//2 球型范围检测
//参数1 中心点
//参数2 球半径
//参数3 检测指定层级（不填检测所有层）
//参数4 是否忽略触发器
//返回值 在该范围内的触发器
Collider[] collides = Physics.OverlapSphere(参数);
//另一个API
if (colliders = Physics.OverlapSphereNonAlloc(比 OverlapSphere 多了一个新参数 在第三位) != 0)

//3 胶囊范围检测
//参数1 上半圆中心点
//参数2 下半圆中心点 两个点可以决定高度和角度
//参数3 半圆半径
//参数4 检测指定层级（不填检测所有层）
//参数5 是否忽略触发器
//返回值 在该范围内的触发器
Collider[] collides = Physics.OverlapCapsule(参数);
if (colliders = Physics.OverlapCapsuleNonAlloc(参数) != 0)
```

## 练习

> 1. 世界坐标原点有一个立方体，键盘WASD键可以控制其前后移动和旋转请结合所学知识实现

1. 按J键在立方体面朝向前方1米处进行立方体范围检测
2. 按K键在立方体前面5米范围内进行胶囊范围检测
3. 按L键以立方体脚下为原点，半径10米内进行球形范围检测

```csharp
public class Physics_Overlap : MonoBehaviour
{
    public float moveSpeed = 10;
    public float RotateSpeed = 10;
    Collider[] cBox;
    Collider[] cCapsule;
    Collider[] cSphere;
    void Update()
    {
        transform.Translate(Vector3.forward * Input.GetAxis("Vertical") * moveSpeed * Time.deltaTime);
        transform.Rotate(Vector3.up * Input.GetAxis("Horizontal") * RotateSpeed * Time.deltaTime);
        if (Input.GetKeyDown(KeyCode.J))
        {
            cBox = Physics.OverlapBox(transform.position + transform.forward, Vector3.one * 0.5f, transform.rotation, 1 << LayerMask.NameToLayer("Monster"));
            for (int i = 0; i < cBox.Length; i++)
                print(cBox[i].gameObject.name + "被击中");
        }
        if (Input.GetKeyDown(KeyCode.K))
        {
            cCapsule = Physics.OverlapCapsule(transform.position, transform.position + transform.forward * 5, 0.4f, 1 << LayerMask.NameToLayer("Monster"));
            for (int i = 0; i < cCapsule.Length; i++)
                print(cCapsule[i].gameObject.name + "被击中");
        }
        if (Input.GetKeyDown(KeyCode.L))
        {
            cSphere = Physics.OverlapSphere(transform.position, 10f, 1 << LayerMask.NameToLayer("Monster"));
            for (int i = 0; i < cSphere.Length; i++)
                print(cSphere[i].gameObject.name + "被击中");
        }
    }
}
```

## API 速查

- Physics.OverlapBox （方法 盒状范围检测 返回所有触碰到的碰撞器脚本）
- Physics.OverlapBoxNonAlloc （方法 盒状范围检测 返回所有触碰到的碰撞器数量）
- Physics.OverlapSphere （方法 球状范围检测 返回所有触碰到的碰撞器脚本）
- Physics.OverlapSphereNonAlloc（方法 球状范围检测 返回所有触碰到的碰撞器数量）
- Physics.OverlapCapsule（方法 胶囊范围检测 返回所有触碰到的碰撞器脚本）
- Physics.OverlapCapsuleNonAlloc（方法 胶囊范围检测 返回所有触碰到的碰撞器数量）
- LayerMask.NameToLayer （方法 层级名转为层级编号）
