## 重要文件夹

### Assets 工程文件夹

**注意：**
- 得到的是 /Assets 资源文件夹的目录
- 该方式获取到的路径，一般情况下只在编辑模式下使用
- 我们不会在实际发布游戏后还使用该路径
- 游戏发布过后该路径就不存在了

```csharp
Application.dataPath
```

---

### Resources 资源文件夹

**注意:**
- 一般不获取路径，只能通过 `Resources` 相关API进行加载
- 如果硬要获取，可以用工程路径拼接
- 该文件夹需要自己创建

**作用：**
- 需要通过 `Resources` 相关API动态加载的资源需要放在其中
- 该文件夹下所有文件都会被打包出去
- 打包时unity会对其压缩加密
- 该文件夹打包后只读只能通过 `Resources` 相关API加载

```csharp
Application.dataPath + "/Resources" //没有任何意义
```

---

### StreamingAssets 流动资源文件夹

**注意：**
- 需要自己创建
- 不要通过拼接取得到路径，因为不同平台的 `streamingAssets` 文件夹可能不一样

**作用：流文件夹**
- 打包出去不会被压缩加密，可以任由我们摆布
- 移动平台只读，PC平台可读可写
- 可以放入一些需要自定义动态加载的初始资源

```csharp
Application.streamingAssetsPath
```

---

### persistentDataPath 持久资源文件夹

**注意：**
- 不需要自己创建
- 在不同平台路径不一样

**作用：固定数据文件夹**
- 所有平台都可读可写
- 一般用于放置动态下载或者动态创建的文件，游戏中创建或者获取的文件都放在其中

```csharp
Application.persistentDataPath;
```

---

## 非重要文件夹

---

### Plugins 插件文件夹

**注意：**
- 一般不获取
- 需要自己创建

**作用：**
- 不同平台的插件相关文件放在其中，比如Ios和Android平台

---

### Editor 编辑器文件夹

**注意：**
- 需要自己创建

**作用：**
- 开发unity编辑器时，编辑器相关脚本放在该文件夹中
- 该文件夹中内容不会被打包出去

---

### Standard Assets 默认资源文件夹

**注意：**
- 一般不获取
- 需要自己创建

**作用：**
- 一般unity自带资源都放在这个文件夹下
- 代码和资源优先被编译

---

## 总结

| **文件夹名称**              | **主要作用**                      | 存储文件    | **打包方式**          | **只读 / 可读写** | **是否手动创建** | **其他注意事项**                        |
| ---------------------- | ----------------------------- | ------- | ----------------- | ------------ | ---------- | --------------------------------- |
| **Assets**             | 工程根目录，存放所有资源和代码               | 包含以下    | 不直接打包，仅打包被引用的资源   | 开发期可读写       | 系统默认       | 路径仅在 Unity 编辑器模式下有效               |
| **Resources**          | 存放需通过 `Resources` API 动态加载的资源 | 动态加载资源  | 压缩、加密并打包进主程序      | 发布后只读        | 是          | 只能通过 `Resources.Load` 访问，不建议放大型资源 |
| **StreamingAssets**    | 存放不需压缩加密的原始资源文件               | 二进制/流数据 | **原封不动**打包，不压缩加密  | 移动端只读，PC端可读写 | 是          | 各平台路径不同，建议用 `UnityWebRequest` 读取  |
| **persistentDataPath** | 存放运行时动态创建、下载或修改的文件            | 持久化数据   | **不打包**（由程序运行时产生） | **所有平台均可读写** | 否          | 适合存放游戏存档、下载的更新包、热更资源              |
| **Plugins**            | 存放不同平台的第三方库或插件                | 跨平台插件/库 | 根据平台目标编译进包体       | 只读           | 是          | 存放原生插件（如 .dll, .so, .a, .jar）     |
| **Editor**             | 存放编辑器扩展脚本                     | 编辑器扩展脚本 | **不打包**（仅留在本地工程）  | 只读           | 是          | 脚本必须放在此文件夹或其子文件夹下才能调用 Editor API  |
| **Standard Assets**    | 存放 Unity 自带资源或通用库             | 官方默认资源  | 优先编译，作为独立程序集      | 只读           | 是          | 放在此处的脚本会先于普通脚本编译                  |
