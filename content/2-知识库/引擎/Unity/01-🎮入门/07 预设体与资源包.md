## 预设体（Prefab / 预制体）

**定义：** 预先设置好的物体，可以重复实例化使用。

**创建：** 把 Hierarchy 中的对象拖到 Project 的 Assets 中，即可创建预设体（变成蓝色）。

**修改方式：**

| 修改方式                     | 备注            | 图示                    |
| ------------------------ | ------------- | --------------------- |
| 直接拖拽覆盖，或者在 Overrides 中覆盖 | 左边取消 <br>右边覆盖 | [[预设体 Overrides.png]] |
| 取消预设体                    |               | [[取消预设体 Unpack.png]]  |
| 打开预设体                    | 直接双击预设体也可以    | [[预设体 Open.png]]      |
| 定位预设体                    | Inspector面板也有 | [[预设体 Select.png]]    |

## 资源包导入导出

**导出资源包：**
- Assets → Export Package...
- 选择需要导出的文件 → Export

**导入资源包：**
- Assets → Import Package → Custom Package...
- 选择 .unitypackage 文件
