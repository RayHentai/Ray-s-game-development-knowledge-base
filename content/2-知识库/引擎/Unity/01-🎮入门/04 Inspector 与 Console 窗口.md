## Inspector（检查窗口）

| [[Inspector（检查窗口）.png]] |
| ----------------------- |
|                         |

**作用：** 查看和编辑场景中游戏对象关联的组件信息（Transform、C#脚本等）。

**窗口内容：**

| [[Inspector 窗口详情.png]] |
| ----------------------------------- |
|                                     |

- 红色部分：游戏对象的基本设置（名称、激活/失活、Tag、Layer）
- 蓝色部分：关联的 C# 脚本
- 绿色部分：脚本的公共成员变量（可在 Inspector 中直接编辑）
- 显示标签 / 激活、失活
- 类型标记（Tag）
- 层级（Layer）
- 位置信息：可以直接拖拽修改

## Console（控制台窗口）

| 入口  | [[Console 入口.png]] |
| --- | ------------------ |

**作用：** 查看调试信息，报错、警告、测试打印都可以显示在其中。

**打开方式：** Ctrl+Shift+C

**打印方式：**
```csharp
// 方式1：Debug 类
Debug.Log("普通信息");
Debug.LogError("错误信息");
Debug.LogWarning("警告信息");

// 方式2：print（MonoBehavior 中可用）
print("信息");
```

**窗口功能：**

| 名称                 | 功能      |
| ------------------ | ------- |
| Collapse           | 折叠信息    |
| clean on play      | 运行时清除信息 |
| Clear on Build<br> | 构建时清除   |
| Clear on Recompile | 报错时清除   |
| Error Pause        | 报错时暂停   |
