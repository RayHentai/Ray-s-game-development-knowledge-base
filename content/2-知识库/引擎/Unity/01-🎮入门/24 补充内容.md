## 其他 API

```csharp
// 检测游戏是否在运行状态（编辑器中使用）
Application.isPlaying

// 退出游戏
Application.Quit();

Cursor.visible = bool;//隐藏鼠标
// 锁定鼠标在窗口内 可以通过Esc摆脱编辑模式下的锁定
Cursor.lockState = CursorLockMode.Confined; // 限制在窗口内
Cursor.lockState = CursorLockMode.Locked; // 限制在屏幕的中心点 并隐藏
Cursor.lockState = CursorLockMode.None; // 不限制

//设置鼠标图片
//参数一 光标图片
//参数二 偏移位置 相对图片左上角   
//参数三 平台支持的光标模式（软件或硬件）
Texture2D tex;
Cursor.SetCursor(tex, Vector2.zero, CursorMode.Auto)
```

## 注意事项

- **在运行状态下的脚本关联编辑不会被记录**（运行结束后会恢复）
- **运行时失活状态下的对象不会执行**任何生命周期函数
- Unity 里的 `Object` 不是万物之父 `object`，而是命名空间在 `UnityEngine` 中的 `Object` 类，也继承了万物之父；C# 中的 `Object` 命名空间是在 `System` 中

## 模型资源的导入

1. 将模型文件（.fbx/.obj 等）拖入 Assets 文件夹
2. Unity 自动导入并生成相应资源
3. 在 Inspector 中可以设置导入选项（缩放、材质等）
4. 面朝模型坐标系的Z轴，缩放大小单位要注意
