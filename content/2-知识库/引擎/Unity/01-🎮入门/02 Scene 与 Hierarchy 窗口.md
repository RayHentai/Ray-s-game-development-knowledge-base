## Hierarchy（层级窗口）


| [[Hierarchy（层级窗口）.png]] |
| ----------------------- |
|                         |

**作用：**
- 创建或拖拽各种游戏对象
- 显示的是**一个场景中的所有对象**

**窗口操作：**
- 创建对象（窗口右键也可以创建）

**窗口功能：**

| 名称          | 作用         | 图示                   |
| ----------- | ---------- | -------------------- |
| 右键Hierarchy | 新建对象相关     | [[右键Hierarchy.png]]  |
| +号          | 同上         | [[Hierarchy +号.png]] |
| 选中、隐藏       | 无法被选中和隐藏对象 | [[选中、隐藏.png]]        |

**快捷键：**

| 快捷键 | 功能 |
|--------|------|
| F2 | 对象改名 |
| Ctrl+C | 复制 |
| Ctrl+V | 粘贴 |
| Ctrl+D | 克隆一个 |
| Delete | 删除 |

## Scene（场景窗口）

| [[Scene（场景窗口）.png]] |
| ------------------- |
|                     |

**作用：**
- 查看所有游戏对象
- 设置所有游戏对象

**窗口功能：**

| 名称        | 作用              | 图示                   |
| --------- | --------------- | -------------------- |
| 渲染模式      | 选择不同的渲染模式       | [[渲染模式（普通、线框等）.png]] |
| 2D视图      | 切换、关闭2D视图       | [[2D视图.png]]         |
| 光源、音效、特效  | 开启、关闭在Scenen中显示 | [[光源、音效、特效显示.png]]   |
| 隐藏、显示标记对象 | 隐藏、显示标记对象       | [[Scene显示、隐藏.png]]   |
| 辅助线       | 显示、关闭辅助线        | [[辅助线.png]]          |
| 摄像机       | 修改摄像机参数         | [[Scene摄像机视口.png]]   |
| 辅助功能      | 控制场景上提示图标等      | [[Scene辅助功能.png]]    |
| 场景轴向      | 切换透视、正交模式       | [[场景轴向.png]]         |
| 工具栏       | 基础编辑工具          | [[工具栏快捷键.png]]       |
| 旋转时轴的相对方向 | 世界坐标、本地坐标       | [[坐标轴方向.png]]        |
| 步长        | 修改编辑步长          | [[步长.png]]           |

**坐标轴：**
- 以屏幕为坐标轴
- 垂直是 Z 正方向
- 平行向右是 X 正方向
- 平行向上是 Y 正方向

**Scene 窗口快捷键：**

| [[工具栏快捷键.png]] | [[鼠标快捷键.png]] |     |
| -------------- | ------------- | --- |

| 快捷键 | 功能 |
|--------|------|
| 鼠标右键+WASD | 飞行模式漫游场景 |
| F | 聚焦到选中对象 |
| Alt+鼠标左键 | 绕选中点旋转视角 |
| 鼠标中键 | 平移视角 |
| 滚轮 | 缩放视角 |
