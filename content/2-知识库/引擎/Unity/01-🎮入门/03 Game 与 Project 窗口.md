| [[Game（游戏窗口）.png]] |

|                    |

**窗口功能：**

| 名称               | 作用                                                                                     | 图示                                  |

## Project（工程窗口）


| [[Project.png]] |
| --------------- |
|                 |

**作用：**
- 显示 Assets 文件夹中的所有内容
- 管理资源脚本文件

**默认内容：**
- Scenes：场景文件夹，默认会有一个空场景
- Packages：官方拓展包

**窗口功能：**

| 名称          | 作用             | 图示                  |
| ----------- | -------------- | ------------------- |
| 右键Project窗口 | 创建资源文件         |                     |
| +           | 同上             |                     |
| 查找          | 按类型查找<br>按标签查找 | [[Project查找.png]]   |
| 窗口设置        | 修改显示布局         | [[Project窗口设置.png]] |

## 练习

> 1. 让之前你创建的汽车能够在 Game 窗口中被完整的看到

```csharp
操作步骤：
1. 选中摄像机（Main Camera）
2. 调整摄像机的 Position 和 Rotation，让汽车完整出现在视野中
3. 也可以先选中汽车，按 F 聚焦，然后通过 GameObject → Align View To Selected 对齐摄像机
```
