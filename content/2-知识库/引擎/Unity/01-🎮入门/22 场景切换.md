## 场景切换

> **注意：** 直接写代码切换场景可能会报错，需要先在 Build Settings 中添加场景。

```csharp
using UnityEngine.SceneManagement;

// 切换到指定场景（按名字）
SceneManager.LoadScene("场景名");

// 切换到指定场景（按序号）
SceneManager.LoadScene(0);

// 重新加载当前场景
SceneManager.LoadScene(SceneManager.GetActiveScene().name);

// 异步加载场景
SceneManager.LoadSceneAsync("场景名");
```

---

## 添加场景到 Build Settings

1. File → Build Settings
2. 将需要的场景文件拖到 Scenes In Build 列表中
3. 场景顺序号就是 LoadScene 使用的序号

## DontDestroyOnLoad

```csharp
// 切换场景时不销毁该对象（用于音乐管理器、数据管理器等）
DontDestroyOnLoad(this.gameObject);
```
