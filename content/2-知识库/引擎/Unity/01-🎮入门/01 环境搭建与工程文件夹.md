## 工程文件夹结构

| 文件夹 | 说明 |
|--------|------|
| **Assets** | 工程资源文件夹（美术资源、脚本文件）—— **我们工作的地方** |
| Library | 库文件夹（unity 自动生成管理） |
| Logs | 日志文件夹（unity 自动生成管理） |
| obj | 编译产生的中间文件（unity 自动生成管理） |
| Packages | 包配置信息（unity 自动生成管理） |
| ProjectSettings | 工程设置信息（unity 自动生成管理） |

> **注意：** 只有 Assets 和 ProjectSettings 文件夹需要纳入版本管理，其他文件夹都是自动生成的。
