## 一、游戏概览

### 电梯简介（Elevator Pitch）
> 你是一名星际工程师，独自降落在一颗颗未知星球上。用双手采集、用机器自动化、用科技树升级装备，在波涛汹涌的怪物攻势下建立属于你的星际工业帝国——然后飞向下一颗星球，重新开始。

### 核心三支柱
| 支柱 | 体验目标 | 参考 |
|------|---------|------|
| **探索** | 每颗星球都有新鲜感，驱动玩家向前 | Minecraft 整合包任务链 |
| **建造** | 自动化生产线带来"机器运转"的成就感 | Factorio、Shapez |
| **生存** | 战斗带来紧张感，掉落物驱动成长 | Vampire Survivors |

### 目标受众
- 喜欢放置/自动化游戏但想要更多行动感的玩家
- 独立游戏爱好者，接受简约像素风
- 求职作品集：展示系统设计与完整工程能力

## 二、核心游戏循环

```
降落星球
    ↓
采集资源（点击 + 自动化采集器）
    ↙             ↘
自动战斗           建造自动化
（怪物刷新·掉落）  （采集器·传送带·工厂）
    ↘             ↙
     消耗资源升级科技树
          ↓
      完成任务阶段
          ↓
      解锁新星球
          ↓
       （循环）
```

### 单局节奏（单颗星球游玩约 20–40 分钟）
1. **0–5 min**：手动点击采集，第一台采集器建造完成
2. **5–15 min**：自动化初步建立，怪物开始刷新，第一次死亡/战斗爽感
3. **15–30 min**：科技树前半段解锁，生产线初具规模
4. **30–40 min**：完成任务链最终目标，解锁新星球传送门

## 三、系统设计

### 3.1 资源系统
> 详见 [[资源系统]]

**资源分三层：**

| 层级 | 资源 | 获取方式 | 用途 |
|------|------|---------|------|
| 基础 | 矿石、结晶、金属 | 点击 / 采集器 | 建造基础设施 |
| 稀有 | 星核碎片、能量晶体 | 怪物掉落 | 解锁科技树 |
| 精华 | 星际元素（每星球唯一） | 任务奖励 | 解锁新星球 |

**设计原则：** 基础资源永远充足（不让玩家干等），稀有资源制造短缺感，精华资源是"毕业门票"。

---

### 3.2 自动化系统
> 详见 [[自动化系统]]

**MVP 范围内只实现三种建筑：**

| 建筑 | 功能 | 建造成本 |
|------|------|---------|
| 采集器 Lv.1 | 每秒 +1 基础资源 | 矿石 ×10 |
| 采集器 Lv.2 | 每秒 +5 基础资源 | 矿石 ×50 + 结晶 ×10 |
| 能量转化炉 | 将基础资源转化为稀有资源 | 金属 ×30 + 星核碎片 ×5 |

> **v2.0 扩展：** 传送带、分拣机、多级工厂（参考 Shapez）

---

### 3.3 战斗系统（类吸血鬼幸存者）
> 详见 [[战斗系统]]

**核心机制：**
- 玩家用 WASD 移动，武器**全自动**攻击最近的敌人
- 每击杀 N 只怪物 → 弹出升级选项（从3个被动词条中选1个）
- 波次机制：每隔 60 秒，怪物强度和数量提升

**MVP 词条池（6个）：**

| 词条 | 效果 |
|------|------|
| 快速射击 | 攻击速度 +25% |
| 穿透弹 | 子弹穿透 1 个额外敌人 |
| 范围爆炸 | 击杀时触发小范围爆炸 |
| 磁力场 | 掉落物自动吸附（范围 +50%） |
| 护盾再生 | 每 10 秒恢复 1 点护盾 |
| 超载模式 | 每隔 15 秒，下一发子弹伤害 ×3 |

**掉落表（MVP）：**
- 普通怪：80% 矿石 / 20% 结晶
- 精英怪（每 5 波出现一次）：100% 星核碎片

> **v2.0 路线：** 替换为俯视角主动攻击战斗系统

---

### 3.4 科技树
> 详见 [[科技树]]

**结构：** 3条主线，每条 4 个节点，共 12 个节点（MVP）

```
采集线：采集器Lv2 → 多矿采集 → 自动精炼 → 星核提取
战斗线：武器强化 → 护盾系统 → 词条槽+1 → 轨道炮
建造线：建筑上限+3 → 传送带基础 → 能量网络 → 星球核心激活
```

解锁条件：消耗指定数量的 **稀有资源**（星核碎片 + 能量晶体），每个节点解锁后立即生效，无需手动装备。

---

### 3.5 任务系统
> 详见 [[任务系统]]

**设计参考：** Minecraft 整合包（如 All the Mods 系列）的书本式任务面板

**结构：** 任务分"章节"，每章对应一个星球阶段

**第一星球任务链示例（共 8 个任务）：**

- [ ] **Ch.1-1** 初次降落：采集矿石 ×20 → 奖励：采集器图纸
- [ ] **Ch.1-2** 第一台机器：建造采集器 ×1 → 奖励：矿石 ×50
- [ ] **Ch.1-3** 来者不善：击败第一只敌人 → 奖励：能量晶体 ×3
- [ ] **Ch.1-4** 初窥科技：解锁第一个科技树节点 → 奖励：稀有矿石 ×10
- [ ] **Ch.1-5** 工厂雏形：同时运行 3 台采集器 → 奖励：建筑上限+1
- [ ] **Ch.1-6** 精英首战：击败精英怪 ×1 → 奖励：星核碎片 ×5
- [ ] **Ch.1-7** 星球精华：获得第一份星际元素 → 奖励：星图碎片
- [ ] **Ch.1-8** 启航：激活星球核心，解锁星图 → 解锁第二星球

## 五、美术方向

**风格定义：** 极简几何像素（参考 Shapez 的扁平工业感）

**具体执行：**
- 角色：16×16 或 24×24 像素精灵
- 建筑：简单几何图形 + 少量动画（旋转、闪光）
- 星球背景：纯色或简单渐变 + 少量粒子效果
- UI：扁平风格，以图标为主，数字字体选用等宽字体

**美术资源来源策略（按优先级）：**
1. Kenney.nl 免费素材包（直接可用）
2. Bing Image Creator / DALL·E 生成概念参考图
3. Unity Particle System 制作特效
4. 必要时用纯色几何形代替插画（完成 > 完美）

## 六、技术架构概览

**架构模式：** 事件驱动 + 单例管理器

**核心 Manager 清单：**

| Manager | 职责 |
|---------|------|
| `ResourceManager` | 全局资源数据增删查 |
| `BuildingManager` | 建筑放置、运行、升级 |
| `EnemySpawner` | 波次控制、怪物生成 |
| `CombatManager` | 玩家战斗、词条系统 |
| `TechTreeManager` | 科技树状态与解锁 |
| `QuestManager` | 任务状态追踪与奖励发放 |
| `PlanetManager` | 星球切换、存档 |
| `UIManager` | 所有 UI 面板的显示控制 |
